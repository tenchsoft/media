# Track Lock Toggle

## 한 줄 정의
사용자가 트랙을 잠가 편집을 방지할 수 있는 버튼이다.

## 시각적 계약
- 위치: 트랙 헤더 (표시 토글 옆).
- 아이콘: 자물쇠 열림(잠금 해제) / 자물쇠 닫힘(잠금) 아이콘.
- 상태: 잠금 해제(기본) / 잠금(노란색 자물쇠).
- 잠금 시 트랙의 클립이 줄무늬 오버레이로 표시된다.

## 인터랙션
1. 자물쇠 아이콘을 클릭한다.
2. 트랙이 잠기고 모든 편집이 차단된다.
3. 다시 클릭하면 잠금이 해제된다.

## 성공 조건
- [ ] 토글 시 잠금/해제가 즉시 전환된다.
- [ ] 잠긴 트랙의 클립은 드래그, 트리밍, 삭제가 불가하다.
- [ ] 잠긴 트랙은 시각적으로 구분된다.

## 범위 외
- 트랙 표시/숨김은 track-visibility-toggle 참조.
