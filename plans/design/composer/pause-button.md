# Pause Button

## 한 줄 정의
사용자가 재생 중인 타임라인을 일시 정지할 수 있는 버튼이다.

## 시각적 계약
- 위치: 재생 컨트롤 영역 (재생 버튼 위치).
- 아이콘: 일시정지 두 줄 아이콘.
- 재생 중에만 활성 상태로 표시된다.
- 단축키: Space (재생 버튼과 토글).

## 인터랙션
1. 재생 중 버튼을 클릭한다 (또는 Space).
2. 재생이 일시 정지된다.
3. 재생 헤드가 현재 위치에 머문다.

## 성공 조건
- [ ] 클릭 시 재생이 즉시 정지된다.
- [ ] 재생 헤드 위치가 유지된다.

## 범위 외
- 정지(처음으로 돌아가기)는 stop-button 참조.
