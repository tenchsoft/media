# Top Bar Undo Button

## 한 줄 정의
사용자가 마지막 편집 작업을 취소할 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음.
- 아이콘: 왼쪽 화살표(Undo) 아이콘.
- 상태: 기본 / 호버 / 누름 / 비활성화(실행 취소할 작업이 없을 때 반투명).
- 단축키: Ctrl+Z.

## 인터랙션
1. 버튼을 클릭한다 (또는 Ctrl+Z).
2. 마지막 편집 작업이 취소된다.
3. 타임라인과 프리뷰가 이전 상태로 복원된다.

## 성공 조건
- [ ] 클릭 시 마지막 작업이 즉시 취소된다.
- [ ] 취소할 작업이 없으면 버튼이 비활성화된다.
- [ ] 여러 번 클릭하면 연속적으로 작업이 취소된다.

## 범위 외
- 실행 취소 히스토리 패널은 별도 기능이다.
