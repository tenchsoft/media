# Out Point Set

## 한 줄 정의
사용자가 클립의 재생 끝점(아웃 포인트)을 설정할 수 있는 기능이다.

## 시각적 계약
- 단축키: O.
- 설정 시: 프리뷰에 아웃 포인트 마커가 표시된다.
- 타임라인에 끝 표시가 나타난다.

## 인터랙션
1. 재생 헤드를 원하는 끝 위치로 이동한다.
2. O 키를 누른다.
3. 아웃 포인트가 설정된다.

## 성공 조건
- [ ] O 키로 아웃 포인트가 설정된다.
- [ ] 설정 위치가 시각적으로 표시된다.
- [ ] 인~아웃 포인트 사이 구간이 하이라이트된다.

## 범위 외
- 인 포인트는 in-point-set 참조.
