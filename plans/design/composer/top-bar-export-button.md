# Top Bar Export Button

## 한 줄 정의
사용자가 편집한 프로젝트를 비디오 파일로 내보낼 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음 오른쪽 영역.
- 아이콘: 업로드 화살표 또는 공유 아이콘.
- 라벨: "내보내기" 텍스트.
- 상태: 기본 / 호버 / 누름 / 비활성화(프로젝트가 비어 있을 때).
- 클릭 시 내보내기 설정 패널을 연다.

## 인터랙션
1. 버튼을 클릭한다.
2. 내보내기 설정 패널이 열린다 (형식, 해상도, 비트레이트 선택).
3. 설정 후 "내보내기"를 클릭하면 렌더링이 시작된다.

## 성공 조건
- [ ] 클릭 시 내보내기 설정 패널이 열린다.
- [ ] 프로젝트가 비어 있으면 버튼이 비활성화된다.

## 범위 외
- 내보내기 설정 상세는 export-format-selection 등 별도 spec 참조.
