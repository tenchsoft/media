# Effect Property Slider

## 한 줄 정의
사용자가 적용된 효과의 속성값을 슬라이더로 조절할 수 있는 컨트롤이다.

## 시각적 계약
- 위치: 효과 속성 패널.
- 형태: 속성별 수평 슬라이더 + 숫자 입력 필드.
- 범위: 속성에 따른 최소/최대값.
- 기본값: 더블 클릭으로 리셋.

## 인터랙션
1. 클립에 적용된 효과를 선택한다.
2. 속성 슬라이더를 드래그하여 값을 조절한다.
3. 프리뷰가 실시간으로 업데이트된다.

## 성공 조건
- [ ] 슬라이더로 속성값이 조절된다.
- [ ] 프리뷰에 실시간 반영된다.
- [ ] 더블 클릭으로 기본값 복원이 가능하다.

## 범위 외
- 키프레임 기반 속성 변경은 keyframe-add 참조.
