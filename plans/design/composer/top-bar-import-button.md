# Top Bar Import Button

## 한 줄 정의
사용자가 미디어 파일(비디오, 오디오, 이미지)을 프로젝트로 가져올 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음 왼쪽 영역.
- 아이콘: 폴더 열기 또는 다운로드 화살표 아이콘.
- 라벨: "가져오기" 텍스트.
- 상태: 기본 / 호버(배경 강조) / 누름(배경 어둡게) / 비활성화(반투명).
- 클릭 시 시스템 파일 열기 대화상자를 표시한다.

## 인터랙션
1. 버튼을 클릭한다.
2. 시스템 파일 대화상자가 열린다 (비디오, 오디오, 이미지 필터).
3. 파일을 선택하고 확인을 누른다.
4. 선택한 파일이 미디어 빈에 추가된다.

## 성공 조건
- [ ] 클릭 시 시스템 파일 대화상자가 열린다.
- [ ] 지원 형식만 필터링하여 표시한다.
- [ ] 여러 파일을 동시에 선택할 수 있다.
- [ ] 가져온 파일이 미디어 빈에 즉시 나타난다.

## 범위 외
- 파일 가져오기 진행률 표시는 별도 기능이다.
- 드래그 앤 드롭으로 파일 가져오기는 별도 기능이다.
