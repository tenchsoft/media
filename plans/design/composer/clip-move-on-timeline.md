# Clip Move on Timeline

## 한 줄 정의
사용자가 타임라인에서 클립을 드래그하여 위치를 이동할 수 있는 기능이다.

## 시각적 계약
- 클립을 클릭하면 선택 테두리가 표시된다.
- 드래그 중: 클립이 반투명으로 커서를 따라간다.
- 이동 위치에 수직 안내선이 표시된다.
- 스냅 시 인접 클립에 자동 정렬된다.

## 인터랙션
1. 클립을 클릭하여 선택한다.
2. 클릭한 채 드래그하여 새 위치로 이동한다.
3. 마우스를 놓아 배치한다.
4. 같은 트랙 내 또는 다른 트랙으로 이동할 수 있다.

## 성공 조건
- [ ] 드래그로 클립 위치가 이동된다.
- [ ] 다른 트랙으로 이동이 가능하다.
- [ ] 스냅이 활성화되면 인접 클립에 정렬된다.
- [ ] 잠긴 트랙의 클립은 이동할 수 없다.

## 범위 외
- 클립 트리밍은 clip-trim-handle 참조.
