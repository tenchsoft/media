# Selection Tool

## 한 줄 정의
사용자가 클립을 선택하고 이동할 수 있는 기본 도구이다.

## 시각적 계약
- 위치: 도구 모음 첫 번째.
- 아이콘: 화살표 아이콘.
- 활성 시: 기본 커서.
- 단축키: V.

## 인터랙션
1. 선택 도구를 활성화한다 (V 키 또는 아이콘 클릭).
2. 클립을 클릭하여 선택한다.
3. 선택된 클립을 드래그하여 이동한다.
4. Shift+클릭으로 여러 클립을 선택한다.

## 성공 조건
- [ ] 클릭으로 클립이 선택된다.
- [ ] 드래그로 클립이 이동된다.
- [ ] Shift+클릭으로 다중 선택이 가능하다.

## 범위 외
- 면도날 도구는 razor-tool 참조.
