# Clip Nest

## 한 줄 정의
사용자가 선택한 클립들을 중첩 시퀀스로 그룹화할 수 있는 기능이다.

## 시각적 계약
- 우클릭 메뉴: "중첩" 항목.
- 중첩 시: 여러 클립이 하나의 중첩 클립으로 통합된다.
- 중첩 클립: 더블 클릭하여 내부 편집 가능.

## 인터랙션
1. 여러 클립을 선택한다.
2. 우클릭 > "중첩"을 선택한다.
3. 클립들이 하나의 중첩 클립으로 통합된다.
4. 더블 클릭하여 내부를 편집할 수 있다.

## 성공 조건
- [ ] 선택한 클립이 중첩 클립으로 통합된다.
- [ ] 더블 클릭으로 내부 편집이 가능하다.
- [ ] 중첩 해제가 가능하다.

## 범위 외
- 클립 그룹화는 clip-group 참조.
