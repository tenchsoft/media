# Media Bin Search

## 한 줄 정의
사용자가 미디어 빈에서 파일 이름으로 검색할 수 있는 검색 필드이다.

## 시각적 계약
- 위치: 미디어 빈 상단.
- 형태: 텍스트 입력 필드 + 돋보기 아이콘.
- 플레이스홀더: "파일 검색...".
- 입력 시 실시간 필터링.

## 인터랙션
1. 검색 필드를 클릭한다.
2. 파일 이름의 일부를 입력한다.
3. 목록이 실시간으로 필터링된다.
4. X 버튼으로 검색을 지운다.

## 성공 조건
- [ ] 입력 시 목록이 실시간으로 필터링된다.
- [ ] 대소문자 구분 없이 검색된다.
- [ ] 검색어 지우면 전체 목록이 복원된다.

## 범위 외
- 파일 정렬은 media-bin-sort 참조.
