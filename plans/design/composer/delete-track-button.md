# Delete Track Button

## 한 줄 정의
사용자가 선택한 트랙을 타임라인에서 제거할 수 있는 버튼이다.

## 시각적 계약
- 위치: 트랙 헤더 영역 (선택된 트랙에만 표시).
- 아이콘: 휴지통 또는 "X" 아이콘.
- 라벨: "트랙 삭제" 툴팁.
- 상태: 기본 / 호버(빨간색 강조) / 누름.

## 인터랙션
1. 트랙 헤더를 클릭하여 트랙을 선택한다.
2. 삭제 버튼을 클릭한다.
3. 트랙에 클립이 있으면 확인 대화상자가 나타난다.
4. 확인하면 트랙과 모든 클립이 제거된다.

## 성공 조건
- [ ] 클릭 시 선택된 트랙이 제거된다.
- [ ] 클립이 있는 트랙은 삭제 전 확인을 요청한다.
- [ ] 빈 트랙은 즉시 삭제된다.

## 범위 외
- 클립만 삭제하고 트랙은 유지하는 기능은 clip-delete 참조.
