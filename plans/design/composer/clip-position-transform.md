# Clip Position Transform

## 한 줄 정의
사용자가 클립의 화면 내 위치를 이동할 수 있는 기능이다.

## 시각적 계약
- 위치: 클립 속성 패널 > 변형 섹션.
- 컨트롤: X, Y 좌표 입력 필드.
- 프리뷰에서 직접 드래그하여 이동.
- 키프레임 지원.

## 인터랙션
1. 클립을 선택한다.
2. X, Y 값을 입력하거나 프리뷰에서 드래그한다.
3. 클립이 지정한 위치로 이동한다.

## 성공 조건
- [ ] 좌표 입력으로 정밀 이동이 가능하다.
- [ ] 프리뷰에서 드래그로 직관적 이동이 가능하다.
- [ ] 키프레임으로 움직임 애니메이션이 가능하다.

## 범위 외
- 클립 스케일은 clip-scale-transform 참조.
