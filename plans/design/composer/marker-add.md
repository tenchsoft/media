# Marker Add

## 한 줄 정의
사용자가 타임라인에 마커를 추가하여 중요 위치를 표시할 수 있는 기능이다.

## 시각적 계약
- 단축키: M.
- 마커: 타임라인 상단 시간 눈금 위에 삼각형 마커.
- 색상: 기본 노란색 (변경 가능).
- 라벨: 마커에 이름을 붙일 수 있다.

## 인터랙션
1. 재생 헤드를 마커를 추가할 위치로 이동한다.
2. M 키를 누르거나 마커 추가 버튼을 클릭한다.
3. 마커가 추가된다.
4. 더블 클릭하여 이름을 편집할 수 있다.

## 성공 조건
- [ ] M 키로 마커가 즉시 추가된다.
- [ ] 마커가 타임라인에 시각적으로 표시된다.
- [ ] 마커에 이름을 붙일 수 있다.

## 범위 외
- 마커 삭제는 marker-delete 참조.
