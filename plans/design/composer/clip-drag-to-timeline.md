# Clip Drag to Timeline

## 한 줄 정의
사용자가 미디어 빈에서 파일을 드래그하여 타임라인에 클립으로 배치할 수 있는 기능이다.

## 시각적 계약
- 드래그 중: 파일 썸네일이 반투명으로 커서를 따라간다.
- 타임라인 위: 삽입 위치에 수직 안내선이 표시된다.
- 스냅 활성화 시: 인접 클립/재생 헤드에 스냅된다.
- 드롭 불가 영역: 빨간색 안내선으로 표시된다.

## 인터랙션
1. 미디어 빈에서 파일을 클릭한 채 드래그한다.
2. 타임라인의 원하는 위치로 이동한다.
3. 마우스를 놓아 클립을 배치한다.
4. 클립이 해당 트랙에 추가된다.

## 성공 조건
- [ ] 드래그 앤 드롭으로 클립이 타임라인에 배치된다.
- [ ] 삽입 위치가 시각적으로 안내된다.
- [ ] 비디오 파일은 비디오 트랙에, 오디오 파일은 오디오 트랙에 배치된다.
- [ ] 스냅이 활성화되어 있으면 인접 클립에 자동 정렬된다.

## 범위 외
- 클립 이동은 clip-move-on-timeline 참조.
