# Preview Fullscreen

## 한 줄 정의
사용자가 프리뷰를 전체 화면으로 표시할 수 있는 버튼이다.

## 시각적 계약
- 위치: 프리뷰 패널 우측 하단.
- 아이콘: 전체 화면 확장 아이콘.
- 단축키: F11 또는 Ctrl+Enter.

## 인터랙션
1. 버튼을 클릭한다.
2. 프리뷰가 전체 화면으로 전환된다.
3. Esc 키로 원래 크기로 복귀한다.

## 성공 조건
- [ ] 클릭 시 프리뷰가 전체 화면으로 전환된다.
- [ ] Esc로 복귀할 수 있다.
- [ ] 전체 화면에서도 재생 컨트롤이 사용 가능하다.

## 범위 외
- 프리뷰 확대는 preview-zoom 참조.
