# Razor Tool

## 한 줄 정의
사용자가 클릭 위치에서 클립을 정밀하게 분할할 수 있는 도구이다.

## 시각적 계약
- 위치: 도구 모음.
- 아이콘: 면도날 아이콘.
- 활성 시: 커서가 가위 모양으로 변경된다.
- 단축키: C.

## 인터랙션
1. 면도날 도구를 선택한다 (C 키 또는 아이콘 클릭).
2. 분할할 위치를 클릭한다.
3. 클릭한 위치에서 클립이 두 개로 분할된다.

## 성공 조건
- [ ] 클릭 위치에서 클립이 정확히 분할된다.
- [ ] 클릭한 트랙의 클립만 분할된다.
- [ ] 분할 후 두 클립은 독립적으로 편집 가능하다.

## 범위 외
- 재생 헤드 기준 분할은 clip-split-at-playhead 참조.
