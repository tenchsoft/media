# Text Tool Color

## 한 줄 정의
사용자가 텍스트 클립의 색상을 선택할 수 있는 색상 선택기이다.

## 시각적 계약
- 위치: 텍스트 편집 패널.
- 형태: 색상 스와치 버튼.
- 클릭 시 색상 선택기 대화상자가 열린다.

## 인터랙션
1. 색상 스와치를 클릭한다.
2. 색상 선택기에서 원하는 색상을 선택한다.
3. 텍스트 색상이 변경된다.

## 성공 조건
- [ ] 클릭 시 색상 선택기가 열린다.
- [ ] 선택한 색상이 텍스트에 즉시 반영된다.

## 범위 외
- 텍스트 위치는 text-tool-position 참조.
