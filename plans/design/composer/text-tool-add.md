# Text Tool Add

## 한 줄 정의
사용자가 타임라인에 텍스트 클립을 추가할 수 있는 기능이다.

## 시각적 계약
- 위치: 도구 모음 "텍스트" 버튼 또는 메뉴 > "텍스트 추가".
- 클릭 시 타임라인에 기본 텍스트 클립이 생성된다.
- 프리뷰에 "텍스트"라는 기본 텍스트가 표시된다.
- 텍스트 편집 패널이 열린다.

## 인터랙션
1. 텍스트 도구를 선택한다.
2. 타임라인에 텍스트 클립이 추가된다.
3. 텍스트 내용, 폰트, 크기, 색상을 편집한다.

## 성공 조건
- [ ] 클릭 시 텍스트 클립이 타임라인에 추가된다.
- [ ] 프리뷰에 텍스트가 표시된다.
- [ ] 텍스트 편집 패널이 열린다.

## 범위 외
- 텍스트 속성 상세는 text-tool-font, text-tool-size 등 참조.
