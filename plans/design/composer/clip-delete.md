# Clip Delete

## 한 줄 정의
사용자가 선택한 클립을 타임라인에서 제거할 수 있는 기능이다.

## 시각적 계약
- 선택된 클립에 Delete 키를 누르거나 우클릭 메뉴에서 "삭제"를 선택.
- 삭제 후 빈 공간이 남는다 (리플 삭제가 아닌 경우).

## 인터랙션
1. 클립을 클릭하여 선택한다.
2. Delete 키를 누르거나 우클릭 > "삭제"를 선택한다.
3. 클립이 타임라인에서 제거된다.

## 성공 조건
- [ ] 선택된 클립이 즉시 제거된다.
- [ ] 삭제 후 빈 공간이 유지된다.
- [ ] 잠긴 트랙의 클립은 삭제할 수 없다.

## 범위 외
- 리플 삭제는 clip-ripple-delete 참조.
