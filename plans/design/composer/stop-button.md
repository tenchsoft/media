# Stop Button

## 한 줄 정의
사용자가 재생을 정지하고 재생 헤드를 시작 위치로 되돌릴 수 있는 버튼이다.

## 시각적 계약
- 위치: 재생 컨트롤 영역.
- 아이콘: 정지 사각형 아이콘.
- 상태: 기본 / 호버 / 누름.

## 인터랙션
1. 재생 중 또는 일시정지 상태에서 버튼을 클릭한다.
2. 재생이 정지된다.
3. 재생 헤드가 타임라인 시작 위치로 이동한다.

## 성공 조건
- [ ] 클릭 시 재생이 정지된다.
- [ ] 재생 헤드가 시작 위치로 이동한다.

## 범위 외
- 일시정지는 pause-button 참조.
