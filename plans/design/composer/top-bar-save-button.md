# Top Bar Save Button

## 한 줄 정의
사용자가 현재 프로젝트를 저장할 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음.
- 아이콘: 플로피 디스크 또는 저장 아이콘.
- 라벨: "저장" 텍스트.
- 상태: 기본 / 호버 / 누름 / 비활성화(저장할 변경사항이 없을 때).
- 단축키: Ctrl+S.

## 인터랙션
1. 버튼을 클릭한다 (또는 Ctrl+S).
2. 처음 저장이면 "다른 이름으로 저장" 대화상자가 열린다.
3. 기존 프로젝트면 즉시 저장된다.
4. 저장 완료 시 짧은 토스트 메시지가 표시된다.

## 성공 조건
- [ ] 클릭 시 프로젝트가 저장된다.
- [ ] 처음 저장 시 "다른 이름으로 저장" 대화상자가 열린다.
- [ ] 저장 후 제목 표시줄에 저장 상태가 반영된다.

## 범위 외
- 자동 저장 기능은 별도 기능이다.
