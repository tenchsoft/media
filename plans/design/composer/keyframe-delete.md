# Keyframe Delete

## 한 줄 정의
사용자가 기존 키프레임을 삭제할 수 있는 기능이다.

## 시각적 계약
- 위치: 키프레임 마커 우클릭 메뉴.
- 메뉴 항목: "키프레임 삭제".
- 단축키: Delete (키프레임 선택 상태에서).

## 인터랙션
1. 타임라인에서 키프레임 마커를 선택한다.
2. Delete 키를 누르거나 우클릭 > "삭제"를 선택한다.
3. 키프레임이 제거되고 보간이 재계산된다.

## 성공 조건
- [ ] 선택한 키프레임이 즉시 삭제된다.
- [ ] 나머지 키프레임 간 보간이 자동 재계산된다.

## 범위 외
- 키프레임 추가는 keyframe-add 참조.
