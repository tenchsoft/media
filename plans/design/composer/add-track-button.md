# Add Track Button

## 한 줄 정의
사용자가 타임라인에 새 트랙(비디오 또는 오디오)을 추가할 수 있는 버튼이다.

## 시각적 계약
- 위치: 타임라인 패널 좌측 트랙 헤더 영역 하단.
- 아이콘: "+" 플러스 아이콘.
- 라벨: "트랙 추가" 툴팁.
- 상태: 기본 / 호버 / 누름.
- 클릭 시 드롭다운 메뉴가 열린다: "비디오 트랙", "오디오 트랙".

## 인터랙션
1. 버튼을 클릭한다.
2. 드롭다운에서 "비디오 트랙" 또는 "오디오 트랙"을 선택한다.
3. 새 빈 트랙이 타임라인에 추가된다.

## 성공 조건
- [ ] 클릭 시 트랙 유형 선택 드롭다운이 열린다.
- [ ] 선택한 유형의 빈 트랙이 타임라인 하단에 추가된다.
- [ ] 새 트랙에 기본 이름이 자동 할당된다.

## 범위 외
- 트랙 삭제는 delete-track-button 참조.
