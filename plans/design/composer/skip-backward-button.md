# Skip Backward Button

## 한 줄 정의
사용자가 재생 헤드를 지정된 시간(예: 5초) 뒤로 건너뛸 수 있는 버튼이다.

## 시각적 계약
- 위치: 재생 컨트롤 영역 (재생 버튼 좌측).
- 아이콘: 왼쪽 이중 화살표 아이콘.
- 라벨: "뒤로 건너뛰기" 툴팁.
- 단축키: 왼쪽 화살표 키.

## 인터랙션
1. 버튼을 클릭한다 (또는 왼쪽 화살표 키).
2. 재생 헤드가 5초 뒤로 이동한다.
3. 프리뷰가 해당 프레임을 표시한다.

## 성공 조건
- [ ] 클릭 시 재생 헤드가 5초 뒤로 이동한다.
- [ ] 타임라인 시작을 초과하지 않는다.

## 범위 외
- 건너뛰기 시간 설정은 별도 기능이다.
