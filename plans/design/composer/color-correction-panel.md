# Color Correction Panel

## 한 줄 정의
사용자가 클립의 색상을 보정할 수 있는 패널이다.

## 시각적 계약
- 위치: 우측 패널 (색상 보정 탭).
- 슬라이더: 밝기, 대비, 채도, 색온도, 틴트, 하이라이트, 그림자.
- 프리뷰에 실시간 반영.
- 키프레임 지원.
- 리셋 버튼.

## 인터랙션
1. 클립을 선택한다.
2. 색상 보정 패널을 연다.
3. 각 슬라이더를 조절하여 색상을 보정한다.
4. 프리뷰에서 결과를 확인한다.

## 성공 조건
- [ ] 각 색상 속성이 슬라이더로 조절된다.
- [ ] 프리뷰에 실시간 반영된다.
- [ ] 리셋 버튼으로 기본값 복원이 가능하다.
- [ ] 키프레임으로 시간에 따른 변화가 가능하다.

## 범위 외
- 효과 적용은 effect-drag-to-clip 참조.
