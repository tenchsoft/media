# Export Format Selection

## 한 줄 정의
사용자가 내보낼 비디오의 파일 형식을 선택할 수 있는 드롭다운이다.

## 시각적 계약
- 위치: 내보내기 설정 패널 상단.
- 형태: 드롭다운 선택.
- 옵션: MP4, MOV, AVI, WebM, GIF 등.
- 선택 시 해당 형식의 기본 설정이 자동 적용된다.

## 인터랙션
1. 드롭다운을 클릭한다.
2. 원하는 형식을 선택한다.
3. 코덱 및 품질 설정이 자동 업데이트된다.

## 성공 조건
- [ ] 지원 형식 목록이 표시된다.
- [ ] 선택 시 코덱 옵션이 형식에 맞게 업데이트된다.

## 범위 외
- 해상도 설정은 export-resolution 참조.
