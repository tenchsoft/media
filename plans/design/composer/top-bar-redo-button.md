# Top Bar Redo Button

## 한 줄 정의
사용자가 취소했던 편집 작업을 다시 실행할 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음 (Undo 버튼 옆).
- 아이콘: 오른쪽 화살표(Redo) 아이콘.
- 상태: 기본 / 호버 / 누름 / 비활성화(다시 실행할 작업이 없을 때 반투명).
- 단축키: Ctrl+Shift+Z.

## 인터랙션
1. 버튼을 클릭한다 (또는 Ctrl+Shift+Z).
2. 마지막으로 취소한 작업이 다시 실행된다.
3. 타임라인과 프리뷰가 업데이트된다.

## 성공 조건
- [ ] 클릭 시 취소했던 작업이 즉시 복원된다.
- [ ] 복원할 작업이 없으면 버튼이 비활성화된다.

## 범위 외
- 실행 취소/다시 실행 히스토리는 별도 기능이다.
