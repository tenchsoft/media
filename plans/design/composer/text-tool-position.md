# Text Tool Position

## 한 줄 정의
사용자가 텍스트 클립의 화면 내 위치를 조절할 수 있는 기능이다.

## 시각적 계약
- 프리뷰에서 텍스트를 직접 드래그하여 이동.
- 또는 속성 패널에서 X, Y 좌표 입력.
- 가이드라인이 중앙 정렬을 지원한다.

## 인터랙션
1. 프리뷰에서 텍스트를 클릭하여 선택한다.
2. 드래그하여 원하는 위치로 이동한다.
3. 또는 속성 패널에서 좌표를 직접 입력한다.

## 성공 조건
- [ ] 드래그로 텍스트 위치가 이동된다.
- [ ] 좌표 입력으로 정밀 배치가 가능하다.
- [ ] 중앙 정렬 가이드라인이 표시된다.

## 범위 외
- 텍스트 크기는 text-tool-size 참조.
