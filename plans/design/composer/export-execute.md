# Export Execute

## 한 줄 정의
사용자가 내보내기 설정을 확정하고 렌더링을 시작할 수 있는 버튼이다.

## 시각적 계약
- 위치: 내보내기 설정 패널 하단.
- 라벨: "내보내기" 버튼.
- 상태: 기본(파란색) / 렌더링 중(진행률 바로 전환) / 완료(초록색).
- 렌더링 중: 진행률 퍼센트와 예상 남은 시간 표시.

## 인터랙션
1. 내보내기 설정을 확인한다.
2. "내보내기" 버튼을 클릭한다.
3. 렌더링이 시작되고 진행률이 표시된다.
4. 완료 시 알림이 표시된다.

## 성공 조건
- [ ] 클릭 시 렌더링이 시작된다.
- [ ] 진행률이 실시간으로 표시된다.
- [ ] 완료 시 파일이 지정된 위치에 저장된다.
- [ ] 렌더링 중 취소가 가능하다.

## 범위 외
- 내보내기 설정은 export-format-selection 등 참조.
