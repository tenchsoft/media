# Keyframe Add

## 한 줄 정의
사용자가 클립 속성에 키프레임을 추가하여 시간에 따른 변화를 정의할 수 있는 기능이다.

## 시각적 계약
- 위치: 효과 속성 패널의 각 속성 옆.
- 아이콘: 다이아몬드(키프레임) 토글 버튼.
- 단축키: 타임라인에서 Ctrl+클릭.
- 키프레임이 추가되면 타임라인에 다이아몬드 마커가 표시된다.

## 인터랙션
1. 클립을 선택하고 속성 패널을 연다.
2. 재생 헤드를 원하는 위치로 이동한다.
3. 속성 옆의 키프레임 버튼을 클릭한다.
4. 속성값을 조절한다.
5. 키프레임이 추가된다.

## 성공 조건
- [ ] 클릭 시 현재 위치에 키프레임이 추가된다.
- [ ] 키프레임이 타임라인에 시각적으로 표시된다.
- [ ] 두 키프레임 사이의 값이 보간된다.

## 범위 외
- 키프레임 삭제는 keyframe-delete 참조.
