# Track Volume Slider

## 한 줄 정의
사용자가 개별 트랙의 오디오 볼륨을 조절할 수 있는 슬라이더이다.

## 시각적 계약
- 위치: 트랙 헤더 (오디오 트랙만).
- 형태: 수평 슬라이더.
- 범위: -∞ dB ~ +12 dB (기본값 0 dB).
- 라벨: 현재 dB 값 표시.
- 상태: 기본 / 호버 / 드래그 중.

## 인터랙션
1. 슬라이더 핸들을 드래그하여 볼륨을 조절한다.
2. 더블 클릭하면 기본값(0 dB)으로 리셋된다.
3. 프리뷰 재생 시 실시간으로 볼륨이 반영된다.

## 성공 조건
- [ ] 드래그 시 볼륨이 실시간으로 조절된다.
- [ ] dB 값이 슬라이더 옆에 표시된다.
- [ ] 더블 클릭 시 0 dB로 리셋된다.

## 범위 외
- 키프레임 기반 볼륨 조절은 audio-volume-keyframe 참조.
