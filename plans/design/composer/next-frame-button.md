# Next Frame Button

## 한 줄 정의
사용자가 한 프레임 앞으로 이동할 수 있는 버튼이다.

## 시각적 계약
- 위치: 재생 컨트롤 영역 (앞으로 건너뛰기 버튼 옆).
- 아이콘: 오른쪽 화살표 + 프레임 아이콘.
- 단축키: 오른쪽 화살표 키 (Shift).

## 인터랙션
1. 버튼을 클릭한다.
2. 재생 헤드가 정확히 한 프레임 앞으로 이동한다.
3. 프리뷰가 해당 프레임을 표시한다.

## 성공 조건
- [ ] 클릭 시 정확히 한 프레임 앞으로 이동한다.
- [ ] 정밀 편집에 사용할 수 있다.

## 범위 외
- 이전 프레임은 previous-frame-button 참조.
