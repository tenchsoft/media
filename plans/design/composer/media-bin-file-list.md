# Media Bin File List

## 한 줄 정의
사용자가 프로젝트에 가져온 미디어 파일 목록을 확인할 수 있는 패널이다.

## 시각적 계약
- 위치: 좌측 패널 (미디어 빈 탭).
- 형태: 파일 목록 (썸네일 + 파일명 + 길이).
- 각 행: 썸네일, 파일명, 재생 시간, 파일 크기.
- 정렬: 이름, 날짜, 유형, 길이 기준.
- 더블 클릭 시 프리뷰에서 재생.

## 인터랙션
1. 미디어 빈 탭을 선택한다.
2. 파일 목록이 표시된다.
3. 파일을 더블 클릭하여 프리뷰에서 확인한다.
4. 파일을 드래그하여 타임라인에 배치한다.

## 성공 조건
- [ ] 프로젝트의 모든 미디어 파일이 목록에 표시된다.
- [ ] 썸네일, 파일명, 길이가 표시된다.
- [ ] 정렬 및 필터링이 가능하다.

## 범위 외
