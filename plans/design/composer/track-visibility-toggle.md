# Track Visibility Toggle

## 한 줄 정의
사용자가 트랙의 표시 여부를 토글할 수 있는 버튼이다.

## 시각적 계약
- 위치: 트랙 헤더 좌측.
- 아이콘: 눈 모양 아이콘 (표시) / 눈 가림 아이콘 (숨김).
- 상태: 활성(파란색) / 비활성(회색).
- 비활성 시 트랙의 클립이 반투명하게 표시된다.

## 인터랙션
1. 눈 아이콘을 클릭한다.
2. 트랙이 숨겨지고 프리뷰에 반영되지 않는다.
3. 다시 클릭하면 트랙이 다시 표시된다.

## 성공 조건
- [ ] 토글 시 트랙 표시/숨김이 즉시 전환된다.
- [ ] 숨겨진 트랙의 클립은 프리뷰에 렌더링되지 않는다.
- [ ] 숨겨진 트랙은 타임라인에서 반투명하게 표시된다.

## 범위 외
- 트랙 잠금은 track-lock-toggle 참조.
