# Top Bar Save As Button

## 한 줄 정의
사용자가 프로젝트를 새 이름이나 위치에 저장할 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 도구 모음 (저장 버튼 옆) 또는 파일 메뉴 내.
- 아이콘: 저장과 폴더가 결합된 아이콘.
- 라벨: "다른 이름으로 저장" 텍스트.
- 단축키: Ctrl+Shift+S.

## 인터랙션
1. 버튼을 클릭한다 (또는 Ctrl+Shift+S).
2. "다른 이름으로 저장" 대화상자가 열린다.
3. 파일 이름과 위치를 지정하고 저장한다.

## 성공 조건
- [ ] 클릭 시 "다른 이름으로 저장" 대화상자가 열린다.
- [ ] 새 이름/위치로 저장된다.

## 범위 외
- 프로젝트 템플릿 저장은 별도 기능이다.
