# Settings Open

## 한 줄 정의
사용자가 Composer 설정 패널을 열 수 있는 버튼이다.

## 시각적 계약
- 위치: 상단 메뉴 > "설정" 또는 톱니바퀴 아이콘.
- 클릭 시 설정 패널이 열린다.
- 카테고리: 일반, 재생, 렌더링, 단축키.

## 인터랙션
1. 설정 버튼을 클릭한다.
2. 설정 패널이 열린다.
3. 원하는 카테고리의 설정을 변경한다.

## 성공 조건
- [ ] 클릭 시 설정 패널이 열린다.
- [ ] 각 카테고리별 설정이 표시된다.

## 범위 외
- 개별 설정 항목은 별도 spec 참조.
