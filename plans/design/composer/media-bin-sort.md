# Media Bin Sort

## 한 줄 정의
사용자가 미디어 빈의 파일 목록을 정렬 기준에 따라 정리할 수 있는 기능이다.

## 시각적 계약
- 위치: 미디어 빈 상단 정렬 드롭다운.
- 옵션: 이름, 날짜, 유형, 길이.
- 오름차순/내림차순 토글.

## 인터랙션
1. 정렬 드롭다운을 클릭한다.
2. 정렬 기준을 선택한다.
3. 파일 목록이 즉시 재정렬된다.

## 성공 조건
- [ ] 선택한 기준으로 파일이 정렬된다.
- [ ] 오름차순/내림차순 전환이 가능하다.

## 범위 외
- 파일 검색은 media-bin-search 참조.
