# Text Tool Size

## 한 줄 정의
사용자가 텍스트 클립의 글꼴 크기를 조절할 수 있는 컨트롤이다.

## 시각적 계약
- 위치: 텍스트 편집 패널 (글꼴 선택 옆).
- 형태: 숫자 입력 필드 + 증감 버튼.
- 범위: 8pt ~ 200pt.

## 인터랙션
1. 크기 값을 직접 입력하거나 증감 버튼을 클릭한다.
2. 프리뷰에 실시간 반영된다.

## 성공 조건
- [ ] 크기 변경이 프리뷰에 즉시 반영된다.
- [ ] 8pt ~ 200pt 범위 내에서 조절된다.

## 범위 외
- 글꼴 선택은 text-tool-font 참조.
