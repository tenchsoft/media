# Effect Remove

## 한 줄 정의
사용자가 클립에서 적용된 효과를 제거할 수 있는 기능이다.

## 시각적 계약
- 위치: 효과 속성 패널의 효과 이름 옆.
- 아이콘: 휴지통 또는 "X" 아이콘.
- 우클릭 메뉴: "효과 제거" 항목.

## 인터랙션
1. 클립을 선택하고 효과 속성 패널을 연다.
2. 제거할 효과의 삭제 버튼을 클릭한다.
3. 효과가 즉시 제거된다.

## 성공 조건
- [ ] 클릭 시 효과가 즉시 제거된다.
- [ ] 프리뷰가 원본 상태로 복원된다.

## 범위 외
- 효과 적용은 effect-drag-to-clip 참조.
