# Window Fullscreen

## 한 줄 정의
전체 화면 모드로 전환하는 기능이다.

## 진입점
- F11 키.
- 하단 도구 모음 전체 화면 버튼.
- 더블 클릭(설정에서 활성화 시).

## 시각 요소
- 전체 화면: 상단 바, 하단 도구 모음 자동 숨김.
- 마우스 상단/하단 이동 시 오버레이 표시.

## 인터랙션
- 전환: 즉시 전체 화면.
- Esc: 창 모드 복귀.

## 성공 조건
- [ ] 전체 화면으로 전환된다.
- [ ] UI 오버레이가 마우스 위치에 따라 나타난다.
- [ ] Esc로 창 모드로 복귀한다.
