# Resize Dialog

## 한 줄 정의
이미지 크기를 변경하는 다이얼로그이다.

## 진입점
- 하단 도구 모음 > 크기 조정 버튼.
- 메뉴 > 이미지 > 크기 조정.

## 시각 요소
- 다이얼로그: 너비/높이 입력 필드.
- 비율 유지 잠금 아이콘.
- 단위 선택: 픽셀/퍼센트.
- 미리보기.

## 인터랙션
- 값 입력 > 적용 또는 취소.

## 성공 조건
- [ ] 너비/높이를 개별 또는 비율 유지로 변경 가능하다.
- [ ] 적용 시 이미지가 새 크기로 리샘플링된다.

## 실패/취소
- 0 이하 값 입력 시 에러.
- 취소 시 원래 크기 유지.

## 범위 외
- 캔버스 크기 조정.
