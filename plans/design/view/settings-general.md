# Settings General

## 한 줄 정의
일반 설정(언어, 테마, 시작 동작 등)을 구성하는 패널이다.

## 진입점
- 하단 도구 모음 > 설정 버튼.
- 메뉴 > 설정.

## 시각 요소
- 설정 다이얼로그 > 일반 탭.
- 언어 선택 드롭다운.
- 테마 선택 (라이트/다크/시스템).
- 시작 시 마지막 파일 열기 토글.

## 인터랙션
- 값 변경: 즉시 적용 또는 적용 버튼.

## 성공 조건
- [ ] 언어 변경 시 UI가 즉시 전환된다.
- [ ] 테마 변경이 즉시 반영된다.
- [ ] 시작 동작 설정이 저장된다.

## 범위 외
- 이미지 관련 설정.
