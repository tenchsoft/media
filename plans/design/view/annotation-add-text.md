# Annotation Add Text

## 한 줄 정의
텍스트 주석을 추가하는 도구이다.

## 진입점
- 주석 도구 모음의 "텍스트" 버튼.

## 시각 요소
- "T" 아이콘 버튼.
- 활성: 강조 색상.
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 텍스트 모드 활성화.
- 캔버스 클릭: 텍스트 입력 상자 표시.
- 타입: 텍스트 입력.
- Enter: 텍스트 확정.

## 성공 조건
- [ ] 클릭 위치에 텍스트 입력 상자가 나타난다.
- [ ] 텍스트가 설정된 색상과 크기로 표시된다.

## 범위 외
- 텍스트 렌더링.
