# Slideshow Start

## 한 줄 정의
슬라이드쇼를 시작하는 기능이다.

## 진입점
- 하단 도구 모음 > 슬라이드쇼 버튼.
- 메뉴 > 보기 > 슬라이드쇼.
- 단축키: F5.

## 시각 요소
- 버튼 아이콘: 재생 + 이미지.
- 전체 화면 모드로 전환.
- 하단 컨트롤 오버레이: 일시정지, 속도, 이전/다음, 종료.

## 인터랙션
- 클릭: 전체 화면 슬라이드쇼 시작.
- Esc: 슬라이드쇼 종료.

## 성공 조건
- [ ] 전체 화면으로 슬라이드쇼가 시작된다.
- [ ] 설정된 간격으로 자동 전환된다.
- [ ] 일시정지/재생이 가능하다.
- [ ] Esc로 종료된다.

## 범위 외
- 슬라이드쇼 설정 변경.
