# Annotation Draw Freehand

## 한 줄 정의
자유 곡선으로 주석을 그리는 도구이다.

## 진입점
- 주석 도구 모음의 "자유 그리기" 버튼.

## 시각 요소
- 연필 아이콘 버튼.
- 활성: 강조 색상.
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 자유 그리기 모드 활성화.
- 드래그: 캔버스에 곡선 그리기.
- 놓기: 그리기 완료.

## 성공 조건
- [ ] 드래그 시 마우스 경로를 따라 선이 그려진다.
- [ ] 선 색상과 두께가 설정값에 맞는다.

## 범위 외
- 펜 입력 보정 로직.
