# Annotation Clear All

## 한 줄 정의
모든 주석을 한 번에 삭제하는 버튼이다.

## 진입점
- 주석 도구 모음의 "전체 삭제" 버튼.

## 시각 요소
- 휴지통 아이콘 버튼.
- 비활성화: 주석이 없으면 회색.
- 호버: 경고 색상 배경.

## 인터랙션
- 클릭: 확인 다이얼로그 표시.
- 확인: 모든 주석 삭제.

## 성공 조건
- [ ] 확인 후 모든 주석이 삭제된다.
- [ ] 원본 이미지는 영향을 받지 않는다.

## 실패/취소 흐름
- 취소: 아무 일도 일어나지 않는다.

## 범위 외
- 주석 데이터 관리.
