# Info Panel File

## 한 줄 정의
이미지 파일의 기본 정보를 표시하는 패널이다.

## 진입점
- 하단 도구 모음 > 정보 버튼.
- 메뉴 > 정보.

## 시각 요소
- 사이드 패널.
- 파일명, 경로, 크기, 해상도, 포맷, 비트 심도, DPI.

## 인터랙션
- 읽기 전용.

## 성공 조건
- [ ] 파일명과 전체 경로가 표시된다.
- [ ] 파일 크기가 읽기 쉬운 단위로 표시된다.
- [ ] 픽셀 해상도와 DPI가 표시된다.
- [ ] 이미지 포맷과 비트 심도가 표시된다.

## 범위 외
- 파일 시스템 작업.
