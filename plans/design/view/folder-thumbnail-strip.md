# Folder Thumbnail Strip

## 한 줄 정의
현재 폴더의 이미지를 하단 썸네일 스트립으로 보여주는 기능이다.

## 진입점
- 폴더 열기 시 자동 표시.

## 시각 요소
- 하단 가로 스크롤 스트립.
- 현재 이미지 하이라이트 테두리.
- 썸네일 크기: 약 64x64px.

## 인터랙션
- 클릭: 해당 이미지로 전환.
- 가로 스크롤/드래그: 썸네일 탐색.

## 성공 조건
- [ ] 폴더 내 모든 이미지가 썸네일로 표시된다.
- [ ] 현재 보는 이미지가 하이라이트된다.
- [ ] 클릭 시 해당 이미지로 전환된다.

## 범위 외
- 썸네일 생성 로직.
