# Bottom Toolbar Delete

## 한 줄 정의
현재 이미지 파일을 삭제하는 하단 도구 모음 버튼이다.

## 진입점
- 하단 도구 모음의 "삭제" 버튼.

## 시각 요소
- 휴지통 아이콘 + "삭제" 레이블.
- 비활성화: 파일이 열려 있지 않으면 회색.
- 호버: 경고 색상 배경.

## 인터랙션
- 클릭: 삭제 확인 다이얼로그 표시.
- 확인: 파일 삭제.

## 성공 조건
- [ ] 확인 후 파일이 휴지통으로 이동한다.
- [ ] 다음 이미지가 자동으로 표시된다.

## 실패/취소 흐름
- 취소: 아무 일도 일어나지 않는다.

## 범위 외
- 파일 시스템 작업.
