# Builtin Subtitle Track Row

## 한 줄 정의
내장 자막 트랙을 선택하는 자막 메뉴의 행 버튼이다.

## 진입점
- 자막 선택 메뉴의 각 자막 트랙 행.

## 시각 요소
- 트랙 언어/이름 텍스트 (예: "Korean", "English").
- 선택됨: 체크 마크 표시.
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 해당 자막 트랙 활성화.

## 성공 조건
- [ ] 클릭 시 해당 자막이 화면에 표시된다.
- [ ] 이전 자막이 해제되고 새 자막으로 전환된다.

## 범위 외
- 자막 타이밍 렌더링.
