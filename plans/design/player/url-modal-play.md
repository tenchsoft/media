# URL Modal Play

## 한 줄 정의
입력한 URL의 미디어를 재생하는 버튼이다.

## 진입점
- URL 모달의 "재생" 버튼.

## 시각 요소
- 주요 색상 버튼, "재생" 레이블.
- 비활성화: URL이 비� 경우 회색.
- 호버: 배경 진하게.

## 인터랙션
- 클릭: URL 미디어 재생 시작.

## 성공 조건
- [ ] 모달이 닫히고 URL 미디어 재생이 시작된다.
- [ ] 유효하지 않은 URL인 경우 에러 메시지가 표시된다.

## 실패/취소 흐름
- URL이 접근 불가: 에러 토스트 표시, 모달 유지.

## 범위 외
- 네트워크 스트리밍 로직.
