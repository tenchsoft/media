# Repeat Mode Button

## 한 줄 정의
반복 재생 모드를 전환하는 하단 컨트롤 바의 버튼이다.

## 진입점
- 하단 컨트롤 바의 반복 버튼.

## 시각 요소
- 반복 없음: 반복 아이콘 (회색).
- 전체 반복: 반복 아이콘 (강조 색상).
- 1곡 반복: 반복 아이콘 + "1" 표시 (강조 색상).
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 반복 없음 → 전체 반복 → 1곡 반복 순환.

## 성공 조건
- [ ] 클릭할 때마다 반복 모드가 순차적으로 전환된다.
- [ ] 현재 모드가 아이콘으로 구분된다.

## 범위 외
- 반복 재생 엔진 동작.
