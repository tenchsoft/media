# AB Loop Button

## 한 줄 정의
재생 구간을 A-B 반복으로 설정하는 하단 컨트롤 바의 버튼이다.

## 진입점
- 하단 컨트롤 바의 A-B 루프 버튼.

## 시각 요소
- 기본: 루프 아이콘.
- 활성: 강조 색상으로 표시.
- 호버: 배경 하이라이트.

## 상태
- 비활성: 클릭 시 A 포인트 설정.
- A 설정됨: 클릭 시 B 포인트 설정, 루프 시작.
- 루프 중: 클릭 시 루프 해제.

## 인터랙션
- 클릭: A/B 포인트 설정 또는 루프 해제.

## 성공 조건
- [ ] 첫 클릭으로 A 포인트가 현재 위치에 설정된다.
- [ ] 두 번째 클릭으로 B 포인트가 설정되고 A-B 구간 반복이 시작된다.
- [ ] 세 번째 클릭으로 루프가 해제된다.

## 범위 외
- AB 루프 재생 강제 동작.
