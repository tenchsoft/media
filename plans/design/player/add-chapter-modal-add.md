# Add Chapter Modal Add

## 한 줄 정의
새 챕터를 추가하는 모달의 확인 버튼이다.

## 진입점
- 챕터 추가 모달의 "추가" 버튼.

## 시각 요소
- 주요 색상 버튼, "추가" 레이블.
- 비활성화: 제목이 비어 있으면 회색.
- 호버: 배경 진하게.

## 인터랙션
- 클릭: 입력한 제목과 현재 재생 위치로 챕터 추가.

## 성공 조건
- [ ] 모달이 닫히고 챕터 목록에 새 항목이 추가된다.
- [ ] 타임라인에 챕터 마커가 표시된다.

## 실패/취소 흐름
- 제목 비어 있음: 버튼 비활성화.

## 범위 외
- 챕터 저장 로직.
