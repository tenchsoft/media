# Aspect Mode Button

## 한 줄 정의
비디오 화면 비율 모드를 전환하는 하단 컨트롤 바의 버튼이다.

## 진입점
- 하단 컨트롤 바의 화면 비율 버튼.

## 시각 요소
- 기본: 화면 비율 아이콘.
- 활성: 현재 모드 텍스트 표시 (예: "16:9", "4:3", "원본").
- 호버: 배경 하이라이트.

## 상태
- 원본 / 16:9 / 4:3 / 맞춤 모드 간 순환.

## 인터랙션
- 클릭: 다음 화면 비율 모드로 전환.

## 성공 조건
- [ ] 클릭할 때마다 화면 비율이 순차적으로 전환된다.
- [ ] 현재 모드가 버튼에 표시된다.

## 범위 외
- 화면 비율 렌더링 동작.
