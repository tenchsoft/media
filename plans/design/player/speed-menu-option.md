# Speed Menu Option

## 한 줄 정의
재생 속도를 선택하는 팝업 메뉴의 항목이다.

## 진입점
- 속도 메뉴 토글 버튼 클릭 → 속도 옵션 행.

## 시각 요소
- 속도 텍스트 (예: "0.5x", "1.0x", "1.5x", "2.0x").
- 현재 선택: 체크 마크.
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 해당 속도로 재생 속도 변경.

## 성공 조건
- [ ] 클릭 시 재생 속도가 즉시 변경된다.
- [ ] 메뉴가 닫히고 속도 버튼에 새 속도가 표시된다.

## 범위 외
- 속도 변경 재생 엔진.
