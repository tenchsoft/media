# Context Menu Aspect Item

## 한 줄 정의
우클릭 메뉴에서 화면 비율을 설정하는 메뉴 항목이다.

## 진입점
- 비디오 영역 우클릭 → 화면 비율 하위 메뉴.

## 시각 요소
- 메뉴 항목 텍스트: "화면 비율".
- 하위 메뉴: 원본, 16:9, 4:3, 맞춤.
- 현재 선택: 체크 마크.
- 호버: 배경 하이라이트.

## 인터랙션
- 클릭: 해당 화면 비율 적용.

## 성공 조건
- [ ] 클릭 시 화면 비율이 즉시 변경된다.
- [ ] 현재 설정에 체크 마크가 표시된다.

## 범위 외
- 화면 비율 렌더링 동작.
