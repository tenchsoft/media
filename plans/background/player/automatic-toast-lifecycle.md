# Toast Lifecycle

## 한 줄 정의
토스트 메시지의 표시, 지속, 자동 소멸을 관리하는 런타임 동작이다.

## 트리거
- 시스템 이벤트(스크린샷 저장, 단축키 안내 등) 발생 시 자동 동작.

## 런타임 계약
- 토스트를 화면 하단에 표시한다.
- 3초 후 자동으로 사라진다.
- 동시에 여러 토스트가 발생하면 스택으로 쌓는다.
- 사용자가 토스트를 클릭하면 즉시 닫는다.

## 성공 조건
- [ ] 토스트가 지정 시간 후 자동으로 사라진다.
- [ ] 여러 토스트가 겹치지 않고 순서대로 표시된다.

## 범위 외
- 토스트 내용 생성.
