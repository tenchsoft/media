# Buffering Progress Bar

## 한 줄 정의
네트워크 스트리밍 또는 대용량 파일 재생 시 버퍼링 진행 상태를 자동으로 표시하는 동작이다.

## 트리거
- 미디어 데이터가 충분히 로드되지 않아 재생이 버퍼링을 기다릴 때 자동으로 표시된다.

## 런타임 계약
- 시크바 위에 버퍼링된 구간을 다른 색상으로 표시한다.
- 버퍼링 중일 때 로딩 스피너가 표시된다.
- 충분히 버퍼링되면 스피너가 사라지고 재생이 재개된다.

## 성공 조건
- [ ] 버퍼링 시 진행 상태가 시각적으로 표시된다.
- [ ] 버퍼링 완료 후 자동으로 재생이 재개된다.

## 범위 외
- 재생 진행 표시는 automatic-playback-position-progress 참조.
