# AB Loop Playback Enforcement

## 한 줄 정의
사용자가 설정한 A-B 구간을 반복 재생하는 런타임 동작이다.

## 트리거
- A-B 루프 버튼이 활성화되어 있을 때 자동으로 동작한다.

## 런타임 계약
- 재생 헤드가 B 지점에 도달하면 자동으로 A 지점으로 되돌아간다.
- 루프 해제 시 정상 재생으로 복귀한다.
- A/B 지점은 시간 코드로 정밀하게 관리된다.
- 오디오와 비디오가 동기화되어 루프된다.

## 성공 조건
- [ ] B 지점 도달 시 A 지점으로 즉시 되돌아간다.
- [ ] 루프 중 끊김 없이 부드럽게 반복된다.
- [ ] 루프 해제 시 현재 위치에서 계속 재생된다.

## 실패/예외
- A/B 지점이 설정되지 않은 경우 루프가 동작하지 않는다.
- 매우 짧은 구간(1프레임 미만)의 경우 루프가 무시된다.

## 범위 외
- A-B 루프 설정 UI는 bottom-ab-loop-button 참조.
