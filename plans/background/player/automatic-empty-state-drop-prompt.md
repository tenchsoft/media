# Empty State Drop Prompt

## 한 줄 정의
파일이 열려 있지 않을 때 드래그 앤 드롭 안내를 자동으로 표시하는 동작이다.

## 트리거
- 플레이어에 파일이 열려 있지 않을 때 자동 표시.

## 런타임 계약
- 중앙에 "파일을 드래그하여 열기" 안내 텍스트와 아이콘을 표시한다.
- 파일이 열리면 안내가 사라진다.
- 파일을 드래그하여 플레이어 위로 가져오면 드롭존이 하이라이트된다.

## 성공 조건
- [ ] 빈 상태에서 안내가 자동 표시된다.
- [ ] 파일 열기 후 안내가 사라진다.
- [ ] 드래그 시 드롭존이 시각적으로 표시된다.

## 범위 외
- 파일 열기 대화상자는 별도 기능이다.
