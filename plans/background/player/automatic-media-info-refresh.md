# Media Info Refresh

## 한 줄 정의
미디어 파일이 열리거나 변경될 때 파일 메타데이터를 자동으로 갱신하는 런타임 동작이다.

## 트리거
- 새 미디어 파일이 열리면 자동 동작.
- 현재 파일의 메타데이터가 변경되면 자동 동작.

## 런타임 계약
- 파일의 해상도, 코덱, 비트레이트, 재생 시간 등 메타데이터를 파싱한다.
- 파싱 결과를 정보 패널에 반영한다.
- 파싱 실패 시 에러 메시지를 표시하지 않고 기본값을 표시한다.

## 성공 조건
- [ ] 파일 열기 직후 메타데이터가 정보 패널에 표시된다.
- [ ] 지원하지 않는 파일도 기본 정보(파일명, 크기)는 표시된다.

## 범위 외
- 정보 패널 UI 렌더링.
