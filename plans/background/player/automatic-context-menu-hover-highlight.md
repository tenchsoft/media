# Context Menu Hover Highlight

## 한 줄 정의
컨텍스트 메뉴에서 마우스가 올라간 항목을 자동으로 강조하는 동작이다.

## 트리거
- 마우스가 컨텍스트 메뉴 항목 위를 이동할 때 자동 동작.

## 런타임 계약
- 마우스가 있는 항목에 호버 배경색을 적용한다.
- 마우스가 벗어나면 즉시 강조가 해제된다.
- 키보드 탐색 시에도 해당 항목이 강조된다.

## 성공 조건
- [ ] 마우스 호버 시 항목이 즉시 강조된다.
- [ ] 마우스 이동에 따라 부드럽게 강조가 이동한다.

## 범위 외
- 컨텍스트 메뉴 항목 선택은 각 항목별 spec 참조.
