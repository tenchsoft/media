# Click Region Refresh

## 한 줄 정의
재생 상태 변화에 따라 클릭 가능 영역이 자동으로 업데이트되는 동작이다.

## 트리거
- 재생/일시정지 전환, 전체화면 전환, 컨트롤 표시/숨김 시 자동 동작.

## 런타임 계약
- 컨트롤이 숨겨지면 비디오 영역 클릭이 재생/일시정지로 동작한다.
- 컨트롤이 표시되면 각 버튼의 클릭 영역이 활성화된다.
- 전체화면 전환 시 클릭 영역이 새 해상도에 맞게 재계산된다.

## 성공 조건
- [ ] 상태 변화 시 클릭 영역이 즉시 업데이트된다.
- [ ] 잘못된 클릭 영역으로 인한 오동작이 없다.

## 범위 외
- 컨트롤 표시/숨김은 별도 기능이다.
