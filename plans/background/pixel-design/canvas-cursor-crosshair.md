# 캔버스 커서 십자선

## 런타임 계약

### 트리거
- 브러시, 지우개, 연필 등 드로잉 도구가 활성 상태일 때 자동 표시

### 동작
- 마우스 커서 위치에 도구 크기를 나타내는 원형/십자선 커서 표시
- 도구 크기 변경 시 커서 크기 실시간 업데이트
- 캔버스 영역 밖에서는 기본 시스템 커서로 복원

### 성공 조건
- 커서가 도구 크기를 정확히 반영함
- 마우스 이동에 지연 없이 따라옴

### 실패 처리
- 렌더링 오류 시 기본 시스템 커서로 폴백

### 제한
- 매우 큰 브러시 크기에서는 커서가 캔버스 경계를 넘어설 수 있음
