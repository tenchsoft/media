# 필터 미리보기 렌더

## 런타임 계약

### 트리거
- 사용자가 필터 효과를 선택하거나 슬라이더를 조정할 때 자동 실행

### 동작
- 현재 캔버스 이미지에 선택된 필터를 실시간으로 적용하여 미리보기 표시
- 슬라이더 값 변경 시마다 미리보기 갱신
- 미리보기 해상도는 성능을 위해 캔버스 전체 해상도보다 낮을 수 있음

### 성공 조건
- 필터 적용 결과가 즉시 미리보기에 반영됨
- 슬라이더 조정 중 끊김 없이 부드럽게 갱신됨

### 실패 처리
- 필터 연산 실패 시 이전 상태 유지
- 메모리 부족 시 저해상도 미리보기로 전환

### 제한
- 미리보기는 적용 버튼을 누르기 전까지 실제 이미지에 반영되지 않음
