# Audio Waveform Display

## 한 줄 정의
오디오 트랙의 파형을 시각적으로 표시하여 사용자가 소리의 크기와 패턴을 직관적으로 파악할 수 있게 한다.

## 트리거
- 오디오 또는 비디오 클립이 타임라인에 배치될 때 자동으로 렌더링된다.
- 오디오 파일이 미디어 빈에서 선택될 때 미리보기에 표시된다.

## 런타임 계약
- 파형은 오디오 샘플 데이터를 기반으로 피크 값을 계산하여 렌더링한다.
- 줌 레벨에 따라 파형 해상도가 자동 조정된다 (축소 시 피크 요약, 확대 시 상세 파형).
- 재생 헤드 위치에 해당하는 파형 영역이 하이라이트된다.
- 파형 색상은 트랙 색상과 일치한다.
- 오디오가 없는 클립은 파형을 표시하지 않는다.

## 성공 조건
- [ ] 오디오 클립의 파형이 타임라인에 정확하게 렌더링된다.
- [ ] 파형이 오디오의 실제 볼륨 레벨을 반영한다 (조용한 구간은 낮게, 큰 구간은 높게).
- [ ] 줌 인/아웃 시 파형 해상도가 부드럽게 업데이트된다.
- [ ] 재생 중 현재 위치가 시각적으로 표시된다.

## 실패/예외
- 오디오 파일이 손상된 경우 빈 파형을 표시하고 사용자에게 알린다.
- 매우 긴 오디오 파일의 경우 초기 로딩 시간을 최소화하기 위해 점진적 렌더링한다.

## 범위 외
- 파형 편집 (자르기, 복사 등)은 별도 기능이다.
- 오디오 분석 (비트 감지 등)은 별도 기능이다.
