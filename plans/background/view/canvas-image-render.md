# Canvas Image Render

## 한 줄 정의
이미지 파일을 캔버스에 렌더링하는 백그라운드 프로세스이다.

## 트리거 조건
- 파일 열기 완료 시.
- 이미지 변환(회전, 반전, 리사이즈) 적용 시.
- 포맷 변환 완료 시.

## 런타임 계약
- 이미지 디코딩은 백그라운드에서 수행된다.
- 대용량 이미지는 점진적 렌더링된다.
- 렌더링 중 취소 요청 시 즉시 중단한다.

## 성공 조건
- [ ] 이미지가 캔버스 중앙에 올바른 비율로 표시된다.
- [ ] 렌더링 완료 후 줌/팬이 가능하다.

## 실패/취소 흐름
- 손상된 파일: 에러 메시지 표시.
- 메모리 부족: 저해상도 미리보기로 대체.

## 범위 외
- 이미지 편집 기능.
