# Window Resize Reflow

## 한 줄 정의
윈도우 크기 변경 시 레이아웃을 자동으로 재배치하는 백그라운드 프로세스이다.

## 트리거 조건
- 윈도우 크기 변경 시.
- 전체화면 전환/해제 시.

## 런타임 계약
- 캔버스는 이미지 비율을 유지하며 윈도우에 맞춘다.
- 도구 모음은 윈도우 너비에 맞게 재배치된다.
- 사이드 패널은 최소 너비 이하에서 자동 숨김.

## 성공 조건
- [ ] 윈도우 크기 변경 후 모든 UI 요소가 올바르게 배치된다.
- [ ] 이미지가 항상 보이고 중앙에 위치한다.

## 실패/취소 흐름
- 최소 크기 이하: 윈도우 크기 제한.

## 범위 외
- 사용자 수동 레이아웃 변경.
