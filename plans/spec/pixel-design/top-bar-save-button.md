# Spec: top-bar-save-button

## 한 줄 정의
사용자가 상단 바의 저장 버튼을 클릭하면 현재 이미지가 저장된다.

## 진입점
- 상단 바: "Save" 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Save" 버튼을 클릭한다.
2. 현재 이미지가 저장된다.
3. "저장됨" 토스트가 나타난다.

## 성공 조건
- [ ] 현재 이미지가 저장된다.
- [ ] "저장됨" 토스트가 나타난다.

## 실패 / 취소 흐름
- 이미지가 열려 있지 않으면 → 아무 동작도 발생하지 않는다.
- 저장 실패 → 오류 토스트가 나타난다.

## 경계 / 예외
- 새 이미지(저장되지 않은) → 다른 이름으로 저장 대화상자가 열린다.

## 범위 외
- 다른 이름으로 저장 동작 (별도 spec).
- 내보내기 동작.
