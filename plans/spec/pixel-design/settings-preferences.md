# Spec: settings-preferences

## 한 줄 정의
사용자가 설정을 열면 환경설정 대화상자가 표시된다.

## 진입점
- 상단 바: 설정 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 설정 버튼을 클릭한다.
2. 환경설정 대화상자가 표시된다.
3. 각종 설정 항목을 변경할 수 있다.
4. 닫기 버튼을 클릭하면 대화상자가 닫힌다.

## 성공 조건
- [ ] 클릭 시 환경설정 대화상자가 표시된다.
- [ ] 설정 항목을 변경할 수 있다.

## 실패 / 취소 흐름
- 닫기 버튼 → 대화상자가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 설정 항목 동작 (별도 spec).
- 내보내기 동작.
