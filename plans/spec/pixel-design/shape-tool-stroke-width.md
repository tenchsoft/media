# Spec: shape-tool-stroke-width

## 한 줄 정의
사용자가 도형의 선 굵기를 변경하면 도형 테두리 두께가 변경된다.

## 진입점
- 속성 패널: 선 굵기 입력 필드

## 사용자 흐름
1. 사용자가 도형 도구를 선택한다.
2. 속성 패널에서 선 굵기를 입력한다.
3. 도형 테두리 두께가 변경된다.

## 성공 조건
- [ ] 선 굵기를 입력할 수 있다.
- [ ] 입력한 굵기가 도형 테두리에 적용된다.

## 실패 / 취소 흐름
- 잘못된 값 → 이전 값으로 복원된다.

## 경계 / 예외
- 0 → 테두리가 표시되지 않는다.

## 범위 외
- 선 색상 동작 (별도 spec).
- 채우기 색상 동작.
