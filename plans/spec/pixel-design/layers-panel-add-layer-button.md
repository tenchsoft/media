# Spec: layers-panel-add-layer-button

## 한 줄 정의
사용자가 레이어 패널에서 레이어 추가 버튼을 클릭하면 새 레이어가 생성된다.

## 진입점
- 레이어 패널: "+" 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널의 "+" 버튼을 클릭한다.
2. 현재 선택된 레이어 위에 새 빈 레이어가 생성된다.
3. 새 레이어가 선택된다.

## 성공 조건
- [ ] 새 빈 레이어가 생성된다.
- [ ] 새 레이어가 선택된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 레이어 삭제 동작 (별도 spec).
- 레이어 복제 동작.
