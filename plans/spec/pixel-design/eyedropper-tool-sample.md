# Spec: eyedropper-tool-sample

## 한 줄 정의
사용자가 스포이드 도구로 클릭하면 해당 위치의 색상이 전경색으로 설정된다.

## 진입점
- 도구 모음: 스포이드 도구 → 캔버스 클릭

## 사용자 흐름
1. 사용자가 도구 모음에서 스포이드 도구를 선택한다.
2. 캔버스에서 원하는 위치를 클릭한다.
3. 해당 위치의 색상이 전경색으로 설정된다.
4. 전경색 견본에 반영된다.

## 성공 조건
- [ ] 클릭 시 해당 위치의 색상이 전경색으로 설정된다.
- [ ] 전경색 견본에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외

- 전경색/배경색 교환 동작.
