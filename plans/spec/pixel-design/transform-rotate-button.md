# Spec: transform-rotate-button

## 한 줄 정의
사용자가 회전 버튼을 클릭하면 이미지가 90도 회전한다.

## 진입점
- 변형 패널: 회전 버튼 클릭

## 사용자 흐름
1. 사용자가 변형 패널에서 회전 버튼을 클릭한다.
2. 이미지가 90도 시계 방향으로 회전한다.
3. 캔버스에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 이미지가 90도 회전한다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자유 회전 동작 (별도 spec).
- 뒤집기 동작.
