# Spec: brush-tool-smoothing

## 한 줄 정의
사용자가 브러시 평활화를 조정하면 스트로크가 부드러워진다.

## 진입점
- 속성 패널: 평활화 슬라이더

## 사용자 흐름
1. 사용자가 브러시 도구를 선택한다.
2. 속성 패널에서 평활화 슬라이더를 드래그한다.
3. 스트로크가 더 부드러워진다.

## 성공 조건
- [ ] 슬라이더 드래그 시 스트로크 평활화가 실시간으로 변경된다.
- [ ] 0% → 평활화 없음 (원시 입력).
- [ ] 높은 값 → 스트로크가 더 부드러워짐.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 높은 평활화 → 입력 지연이 발생할 수 있다.

## 범위 외
- 브러시 유량 동작 (별도 spec).
- 브러시 경도 동작.
