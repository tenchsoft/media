# Spec: adjust-opacity-slider-control

## 한 줄 정의
사용자가 불투명도 조정 슬라이더를 드래그하면 이미지 전체 투명도가 변경된다.

## 진입점
- 조정 패널: 불투명도 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 불투명도 슬라이더를 드래그한다.
2. 이미지 전체 투명도가 실시간으로 변경된다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 이미지 투명도가 실시간으로 변경된다.
- [ ] 슬라이더를 놓으면 값이 확정된다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 불투명도 값으로 복원된다.

## 경계 / 예외
- 0% → 이미지가 완전 투명해진다.

## 범위 외
- 레이어 불투명도 동작 (별도 spec).
- 브러시 불투명도 동작.
