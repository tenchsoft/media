# Spec: text-tool-font-family

## 한 줄 정의
사용자가 텍스트 도구에서 글꼴을 선택하면 텍스트에 해당 글꼴이 적용된다.

## 진입점
- 속성 패널: 글꼴 드롭다운

## 사용자 흐름
1. 사용자가 텍스트 도구를 선택한다.
2. 속성 패널에서 글꼴 드롭다운을 클릭한다.
3. 원하는 글꼴을 선택한다.
4. 새로 입력하거나 선택한 텍스트에 해당 글꼴이 적용된다.

## 성공 조건
- [ ] 드롭다운에서 글꼴을 선택할 수 있다.
- [ ] 선택한 글꼴이 텍스트에 적용된다.

## 실패 / 취소 흐름
- 바깥 클릭 → 드롭다운이 닫히고 이전 글꼴이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 글꼴 크기 동작 (별도 spec).
- 텍스트 정렬 동작.
