# Spec: canvas-brush-stroke-control

## 한 줄 정의
사용자가 브러시 도구로 캔버스에서 드래그하면 브러시 스트로크가 그려진다.

## 진입점
- 캔버스: 브러시 도구로 드래그

## 사용자 흐름
1. 사용자가 브러시 도구를 선택한다.
2. 캔버스에서 마우스를 누르고 드래그한다.
3. 전경색으로 브러시 스트로크가 실시간으로 그려진다.
4. 마우스를 놓으면 스트로크가 완성된다.

## 성공 조건
- [ ] 드래그 중 브러시 스트로크가 실시간으로 그려진다.
- [ ] 전경색으로 그려진다.
- [ ] 마우스를 놓으면 스트로크가 완성된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 빠른 드래그 → 스트로크가 부드럽게 보간된다.
- 태블릿 펜 → 필압이 반영된다.

## 범위 외
- 브러시 크기/경도 조정 동작 (별도 spec).
- 지우개 동작.
