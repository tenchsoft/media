# Spec: canvas-crop-drag-control

## 한 줄 정의
사용자가 자르기 도구로 캔버스에서 드래그하면 자르기 영역이 설정된다.

## 진입점
- 캔버스: 자르기 도구로 드래그

## 사용자 흐름
1. 사용자가 자르기 도구를 선택한다.
2. 캔버스에서 드래그하여 자르기 영역을 설정한다.
3. 영역 모서리를 드래그하여 조정할 수 있다.
4. Enter를 누르면 영역 외부가 잘린다.

## 성공 조건
- [ ] 드래그로 자르기 영역이 설정된다.
- [ ] 모서리 드래그로 영역 조정이 가능하다.
- [ ] Enter 시 영역 외부가 잘린다.

## 실패 / 취소 흐름
- Escape → 자르기가 취소된다.

## 경계 / 예외
- 영역이 너무 작으면 → 자르기가 적용되지 않는다.

## 범위 외
- 캔버스 크기 변경 동작.
- 선택 도구 동작 (별도 spec).
