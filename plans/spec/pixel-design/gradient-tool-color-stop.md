# Spec: gradient-tool-color-stop

## 한 줄 정의
사용자가 그라데이션 편집기에서 색상 정지점을 추가하면 그라데이션에 새 색상이 추가된다.

## 진입점
- 속성 패널: 그라데이션 편집기 바

## 사용자 흐름
1. 사용자가 그라데이션 편집기 바 아래 빈 영역을 클릭한다.
2. 새 색상 정지점이 추가된다.
3. 색상 정지점을 더블 클릭하여 색상을 변경한다.
4. 드래그하여 위치를 조정한다.

## 성공 조건
- [ ] 클릭 시 새 색상 정지점이 추가된다.
- [ ] 더블 클릭으로 색상을 변경할 수 있다.
- [ ] 드래그로 위치를 조정할 수 있다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최소 2개의 색상 정지점이 필요하다.

## 범위 외
- 그라데이션 적용 동작 (별도 spec).
- 브러시 도구 동작.
