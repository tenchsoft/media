# Spec: color-picker-apply-button

## 한 줄 정의
사용자가 색상 선택기에서 확인 버튼을 클릭하면 선택한 색상이 적용된다.

## 진입점
- 색상 선택기: "OK" 버튼 클릭

## 사용자 흐름
1. 사용자가 색상 선택기에서 색상을 선택한다.
2. "OK" 버튼을 클릭한다.
3. 선택한 색상이 전경색(또는 배경색)으로 적용된다.
4. 색상 선택기가 닫힌다.

## 성공 조건
- [ ] 선택한 색상이 적용된다.
- [ ] 색상 선택기가 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 취소 동작 (별도 spec).
- 색상 슬라이더 동작.
