# Spec: canvas-guide-drag

## 한 줄 정의
사용자가 눈금자에서 드래그하면 캔버스에 가이드 라인이 생성된다.

## 진입점
- 캔버스: 눈금자에서 드래그

## 사용자 흐름
1. 사용자가 상단 또는 좌측 눈금자를 클릭한 채 드래그한다.
2. 캔버스에 수평 또는 수직 가이드 라인이 나타난다.
3. 원하는 위치에 놓으면 가이드가 고정된다.

## 성공 조건
- [ ] 눈금자에서 드래그 시 가이드 라인이 나타난다.
- [ ] 원하는 위치에 고정할 수 있다.

## 실패 / 취소 흐름
- 캔버스 밖으로 드래그 → 가이드가 생성되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 가이드 삭제 동작 (별세 spec).
- 가이드 잠금 동작.
