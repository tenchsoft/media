# Spec: adjust-highlights-slider-control

## 한 줄 정의
사용자가 하이라이트 조정 슬라이더를 드래그하면 밝은 영역이 밝아지거나 어두워진다.

## 진입점
- 조정 패널: 하이라이트 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 하이라이트 슬라이더를 드래그한다.
2. 밝은 영역이 실시간으로 변경된다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 밝은 영역이 실시간으로 변경된다.
- [ ] 어두운 영역은 영향을 받지 않는다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 하이라이트 값으로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 그림자 조정 동작 (별도 spec).
- 노출 조정 동작.
