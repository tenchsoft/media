# Spec: top-bar-redo-button

## 한 줄 정의
사용자가 상단 바의 재실행 버튼을 클릭하면 마지막으로 취소한 편집 동작이 다시 적용된다.

## 진입점
- 상단 바: "Redo" 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Redo" 버튼을 클릭한다.
2. 마지막으로 취소한 편집 동작이 다시 적용된다.
3. 캔버스가 재적용된 상태로 갱신된다.

## 성공 조건
- [ ] 마지막으로 취소한 동작이 다시 적용된다.
- [ ] 캔버스가 재적용된 상태로 갱신된다.

## 실패 / 취소 흐름
- 재적용할 동작이 없으면 → 버튼이 비활성화되어 있다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- Undo 동작 (별도 spec).
- Ctrl+Shift+Z 단축키 동작.
