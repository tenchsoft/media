# Spec: canvas-select-drag-control

## 한 줄 정의
사용자가 선택 도구로 캔버스에서 드래그하면 사각형 선택 영역이 만들어진다.

## 진입점
- 캔버스: 선택 도구로 드래그

## 사용자 흐름
1. 사용자가 선택 도구를 선택한다.
2. 캔버스에서 마우스를 누르고 드래그한다.
3. 사각형 선택 영역이 실시간으로 표시된다.
4. 마우스를 놓으면 선택 영역이 확정된다.

## 성공 조건
- [ ] 드래그 중 사각형 선택 영역이 실시간으로 표시된다.
- [ ] 마우스를 놓으면 선택 영역이 확정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- Shift 누른 상태 → 정사각형 선택 영역이 만들어진다.
- 드래그 없이 클릭 → 선택 영역이 해제된다.

## 범위 외
- 선택 영역 편집 동작.
- 브러시 동작 (별도 spec).
