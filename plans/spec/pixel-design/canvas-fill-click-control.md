# Spec: canvas-fill-click-control

## 한 줄 정의
사용자가 채우기 도구로 캔버스에서 클릭하면 전경색으로 영역이 채워진다.

## 진입점
- 캔버스: 채우기 도구로 클릭

## 사용자 흐름
1. 사용자가 채우기 도구를 선택한다.
2. 캔버스에서 클릭한다.
3. 클릭한 위치와 인접한 같은 색상 영역이 전경색으로 채워진다.

## 성공 조건
- [ ] 클릭 시 인접한 같은 색상 영역이 전경색으로 채워진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 선택 영역이 있으면 → 선택 영역 내부만 채워진다.
- 허용도 조정 → 비슷한 색상도 함께 채워진다.

## 범위 외
- 전경색 선택 동작 (별도 spec).
- 그라디언트 동작.
