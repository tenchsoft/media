# Spec: layer-row-lock-toggle

## 한 줄 정의
사용자가 레이어 패널에서 잠금 아이콘을 클릭하면 레이어가 잠기거나 잠금 해제된다.

## 진입점
- 레이어 패널: 잠금 아이콘 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어의 잠금 아이콘을 클릭한다.
2. 레이어가 잠긴다 (편집 불가).
3. 다시 클릭하면 잠금이 해제된다.

## 성공 조건
- [ ] 클릭 시 레이어가 잠긴다.
- [ ] 잠긴 레이어는 편집할 수 없다.
- [ ] 다시 클릭 시 잠금이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 잠긴 레이어에 편집 시도 → "잠긴 레이어입니다" 토스트가 나타난다.

## 범위 외
- 레이어 가시성 동작 (별도 spec).
- 레이어 선택 동작.
