# Spec: adjust-sharpen-slider-control

## 한 줄 정의
사용자가 선명하게 조정 슬라이더를 드래그하면 이미지가 선명해진다.

## 진입점
- 조정 패널: 선명하게 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 선명하게 슬라이더를 드래그한다.
2. 이미지가 실시간으로 선명해진다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 이미지가 실시간으로 선명해진다.
- [ ] 슬라이더를 놓으면 값이 확정된다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 선명하게 값으로 복원된다.

## 경계 / 예외
- 최대값 → 이미지에 노이즈가 발생할 수 있다.

## 범위 외
- 블러 조정 동작 (별도 spec).
- 브러시 도구 동작.
