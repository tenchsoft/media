# Spec: selection-rectangle-drag

## 한 줄 정의
사용자가 선택 도구로 드래그하면 사각형 영역이 선택된다.

## 진입점
- 캔버스: 선택 도구 활성화 후 드래그

## 사용자 흐름
1. 사용자가 도구 모음에서 선택 도구를 선택한다.
2. 캔버스에서 드래그한다.
3. 사각형 선택 영역이 표시된다.
4. 드래그를 놓으면 선택 영역이 확정된다.

## 성공 조건
- [ ] 드래그 시 사각형 선택 영역이 실시간으로 표시된다.
- [ ] 드래그를 놓으면 선택 영역이 확정된다.

## 실패 / 취소 흐름
- Esc → 선택이 해제된다.

## 경계 / 예외
- Shift 드래그 → 정사각형으로 제한된다.

## 범위 외
- 올가미 선택 동작 (별도 spec).
- 선택 영역 이동 동작.
