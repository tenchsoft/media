# Spec: canvas-text-placement-control

## 한 줄 정의
사용자가 텍스트 도구로 캔버스에서 클릭하면 텍스트 입력이 시작된다.

## 진입점
- 캔버스: 텍스트 도구로 클릭

## 사용자 흐름
1. 사용자가 텍스트 도구를 선택한다.
2. 캔버스에서 텍스트를 배치할 위치를 클릭한다.
3. 텍스트 입력 필드가 나타난다.
4. 텍스트를 입력한다.
5. Enter 또는 다른 도구 선택 시 텍스트가 확정된다.

## 성공 조건
- [ ] 클릭 시 텍스트 입력 필드가 나타난다.
- [ ] 텍스트가 캔버스에 표시된다.
- [ ] 확정 시 텍스트 레이어가 생성된다.

## 실패 / 취소 흐름
- Escape → 텍스트 입력이 취소된다.

## 경계 / 예외
- 빈 텍스트 → 텍스트 레이어가 생성되지 않는다.

## 범위 외
- 폰트/크기 설정 동작.
- 브러시 동작 (별도 spec).
