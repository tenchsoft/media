# Spec: gradient-tool-opacity-stop

## 한 줄 정의
사용자가 그라데이션 편집기에서 투명도 정지점을 조정하면 해당 위치의 투명도가 변경된다.

## 진입점
- 속성 패널: 그라데이션 편집기 투명도 바

## 사용자 흐름
1. 사용자가 그라데이션 편집기 위쪽 투명도 바에서 정지점을 드래그한다.
2. 해당 위치의 투명도가 변경된다.
3. 그라데이션에 실시간으로 반영된다.

## 성공 조건
- [ ] 드래그 시 해당 위치의 투명도가 변경된다.
- [ ] 그라데이션에 실시간으로 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 색상 정지점 동작 (별도 spec).
- 그라데이션 적용 동작.
