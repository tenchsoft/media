# Spec: canvas-gradient-drag-control

## 한 줄 정의
사용자가 그라디언트 도구로 캔버스에서 드래그하면 그라디언트가 생성된다.

## 진입점
- 캔버스: 그라디언트 도구로 드래그

## 사용자 흐름
1. 사용자가 그라디언트 도구를 선택한다.
2. 캔버스에서 드래그 방향과 길이를 설정한다.
3. 전경색에서 배경색으로 그라디언트가 생성된다.

## 성공 조건
- [ ] 드래그 방향으로 그라디언트가 생성된다.
- [ ] 전경색에서 배경색으로 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 선택 영역이 있으면 → 선택 영역 내부에만 그라디언트가 적용된다.

## 범위 외
- 채우기 동작 (별도 spec).
- 전경색/배경색 선택 동작.
