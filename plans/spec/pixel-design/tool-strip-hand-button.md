# Spec: tool-strip-hand-button

## 한 줄 정의
사용자가 도구 스트립에서 손 버튼을 클릭하면 화면 이동 도구가 활성화된다.

## 진입점
- 도구 스트립: "Hand" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 손 버튼을 클릭한다.
2. 화면 이동 도구가 활성화된다.
3. 캔버스에서 드래그하면 화면이 이동한다.

## 성공 조건
- [ ] 화면 이동 도구가 활성화된다.
- [ ] 드래그 시 화면이 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 스페이스바를 누른 상태에서 드래그 → 다른 도구에서도 화면 이동이 가능하다.

## 범위 외
- 줌 동작 (별도 spec).
- 이동 도구 동작.
