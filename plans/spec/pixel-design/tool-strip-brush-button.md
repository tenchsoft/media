# Spec: tool-strip-brush-button

## 한 줄 정의
사용자가 도구 스트립에서 브러시 버튼을 클릭하면 브러시 도구가 활성화된다.

## 진입점
- 도구 스트립: "Brush" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 브러시 버튼을 클릭한다.
2. 브러시 도구가 활성화된다.
3. 캔버스에서 드래그하면 브러시 스트로크가 그려진다.

## 성공 조건
- [ ] 브러시 도구가 활성화된다.
- [ ] 캔버스에서 드래그 시 브러시 스트로크가 그려진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 브러시 도구 → 아무 변화가 없다.

## 범위 외
- 브러시 크기/경도 조정 동작 (별도 spec).
- 다른 도구 동작.
