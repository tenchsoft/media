# Spec: properties-panel-brush-hardness-control

## 한 줄 정의
사용자가 속성 패널에서 브러시 경도 슬라이더를 조정하면 브러시 가장자리가 변경된다.

## 진입점
- 속성 패널: 브러시 경도 슬라이더

## 사용자 흐름
1. 사용자가 속성 패널의 브러시 경도 슬라이더를 드래그한다.
2. 브러시 가장자리가 실시간으로 변경된다.
3. 0% → 완전 부드러운 가장자리, 100% → 날카로운 가장자리.

## 성공 조건
- [ ] 슬라이더 드래그 시 브러시 가장자리가 실시간으로 변경된다.
- [ ] 0%에서 부드럽고 100%에서 날카롭게 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 브러시 크기 동작 (별도 spec).
- 브러시 불투명도 동작.
