# Spec: background-color-swatch-button

## 한 줄 정의
사용자가 배경색 스와치를 클릭하면 색상 선택기가 열린다.

## 진입점
- 도구 스트립: 배경색 스와치 클릭

## 사용자 흐름
1. 사용자가 배경색 스와치를 클릭한다.
2. 색상 선택기가 열린다.
3. 색상을 선택하고 확인한다.
4. 배경색이 변경된다.

## 성공 조건
- [ ] 클릭 시 색상 선택기가 열린다.
- [ ] 색상 선택 시 배경색이 변경된다.

## 실패 / 취소 흐름
- 색상 선택기에서 취소 → 배경색이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전경색 스와치 동작 (별도 spec).
- 스포이드 동작.
