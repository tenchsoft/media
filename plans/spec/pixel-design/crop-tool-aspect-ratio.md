# Spec: crop-tool-aspect-ratio

## 한 줄 정의
사용자가 자르기 도구에서 비율을 설정하면 자르기 영역이 해당 비율로 제한된다.

## 진입점
- 속성 패널: 비율 드롭다운

## 사용자 흐름
1. 사용자가 자르기 도구를 선택한다.
2. 속성 패널에서 비율 드롭다운을 클릭한다.
3. 원하는 비율을 선택한다 (자유, 1:1, 4:3, 16:9 등).
4. 자르기 영역이 해당 비율로 제한된다.

## 성공 조건
- [ ] 비율 선택 시 자르기 영역이 해당 비율로 제한된다.
- [ ] 드래그해도 비율이 유지된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 자유 비율 → 제한 없이 자유롭게 조정된다.

## 범위 외
- 자르기 적용 동작 (별도 spec).
- 캔버스 크기 변경 동작.
