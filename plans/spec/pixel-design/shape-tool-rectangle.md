# Spec: shape-tool-rectangle

## 한 줄 정의
사용자가 도형 도구에서 사각형을 선택하고 드래그하면 사각형이 그려진다.

## 진입점
- 도구 모음: 도형 도구 → 사각형

## 사용자 흐름
1. 사용자가 도구 모음에서 도형 도구를 선택한다.
2. 속성 패널에서 사각형을 선택한다.
3. 캔버스에서 드래그한다.
4. 사각형이 그려진다.

## 성공 조건
- [ ] 드래그 시 사각형이 실시간으로 표시된다.
- [ ] 드래그를 놓으면 사각형이 확정된다.

## 실패 / 취소 흐름
- Esc → 그리기가 취소된다.

## 경계 / 예외
- Shift 드래그 → 정사각형으로 제한된다.

## 범위 외
- 원형 도구 동작 (별도 spec).
- 선 도구 동작.
