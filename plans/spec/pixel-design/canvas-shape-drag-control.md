# Spec: canvas-shape-drag-control

## 한 줄 정의
사용자가 도형 도구로 캔버스에서 드래그하면 도형이 그려진다.

## 진입점
- 캔버스: 도형 도구로 드래그

## 사용자 흐름
1. 사용자가 도형 도구를 선택한다.
2. 캔버스에서 마우스를 누르고 드래그한다.
3. 도형이 실시간으로 표시된다.
4. 마우스를 놓으면 도형이 확정된다.

## 성공 조건
- [ ] 드래그 중 도형이 실시간으로 표시된다.
- [ ] 마우스를 놓으면 도형이 확정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- Shift 누른 상태 → 정비율 도형이 그려진다.

## 범위 외
- 도형 종류 선택 동작.
- 브러시 동작 (별도 spec).
