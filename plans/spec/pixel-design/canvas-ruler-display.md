# Spec: canvas-ruler-display

## 한 줄 정의
캔버스 상단과 좌측에 눈금자가 표시되어 위치를 확인할 수 있다.

## 진입점
- 캔버스: 눈금자 표시 (자동)

## 사용자 흐름
1. 사용자가 캔버스를 연다.
2. 상단과 좌측에 눈금자가 표시된다.
3. 마우스 커서 위치에 따라 눈금자에 가이드라인이 표시된다.

## 성공 조건
- [ ] 상단과 좌측에 눈금자가 표시된다.
- [ ] 마우스 커서 위치에 가이드라인이 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 확대/축소 시 눈금자 단위가 조정된다.

## 범위 외
- 가이드 생성 동작 (별도 spec).
- 그리드 표시 동작.
