# Spec: tool-strip-text-button

## 한 줄 정의
사용자가 도구 스트립에서 텍스트 버튼을 클릭하면 텍스트 도구가 활성화된다.

## 진입점
- 도구 스트립: "Text" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 텍스트 버튼을 클릭한다.
2. 텍스트 도구가 활성화된다.
3. 캔버스에서 클릭하면 텍스트 입력 필드가 나타난다.
4. 텍스트를 입력한다.

## 성공 조건
- [ ] 텍스트 도구가 활성화된다.
- [ ] 클릭 시 텍스트 입력 필드가 나타난다.
- [ ] 텍스트가 캔버스에 표시된다.

## 실패 / 취소 흐름
- Escape → 텍스트 입력이 취소된다.

## 경계 / 예외
- 빈 텍스트 → 텍스트 레이어가 생성되지 않는다.

## 범위 외
- 폰트/크기 설정 동작.
- 브러시 도구 동작 (별도 spec).
