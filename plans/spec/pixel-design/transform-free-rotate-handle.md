# Spec: transform-free-rotate-handle

## 한 줄 정의
사용자가 회전 핸들을 드래그하면 이미지가 자유 각도로 회전한다.

## 진입점
- 변형 패널: 회전 핸들 드래그

## 사용자 흐름
1. 사용자가 변형 모드에서 회전 핸들을 잡는다.
2. 드래그하여 원하는 각도로 회전한다.
3. 핸들을 놓으면 회전 각도가 확정된다.

## 성공 조건
- [ ] 드래그 시 이미지가 자유 각도로 회전한다.
- [ ] 핸들을 놓으면 각도가 확정된다.

## 실패 / 취소 흐름
- Esc → 회전이 취소되고 원래 각도로 복원된다.

## 경계 / 예외
- Shift 드래그 → 15도 단위로 스냅된다.

## 범위 외
- 90도 회전 동작 (별도 spec).
- 뒤집기 동작.
