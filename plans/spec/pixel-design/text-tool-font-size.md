# Spec: text-tool-font-size

## 한 줄 정의
사용자가 텍스트 도구에서 글꼴 크기를 변경하면 텍스트 크기가 변경된다.

## 진입점
- 속성 패널: 글꼴 크기 입력 필드

## 사용자 흐름
1. 사용자가 텍스트 도구를 선택한다.
2. 속성 패널에서 글꼴 크기 입력 필드를 클릭한다.
3. 새 크기를 입력하거나 드롭다운에서 선택한다.
4. 텍스트 크기가 변경된다.

## 성공 조건
- [ ] 글꼴 크기를 입력할 수 있다.
- [ ] 입력한 크기가 텍스트에 적용된다.

## 실패 / 취소 흐름
- 잘못된 값 → 이전 값으로 복원된다.

## 경계 / 예외
- 최소 크기 제한이 있다.
- 최대 크기 제한이 있다.

## 범위 외
- 글꼴 선택 동작 (별도 spec).
- 텍스트 색상 동작.
