# Spec: settings-auto-save

## 한 줄 정의
사용자가 자동 저장 설정을 켜면 일정 간격으로 작업이 자동 저장된다.

## 진입점
- 설정: 일반 탭 → 자동 저장 토글

## 사용자 흐름
1. 사용자가 설정에서 일반 탭을 선택한다.
2. 자동 저장 토글을 켠다.
3. 저장 간격을 설정한다.
4. 설정 간격마다 작업이 자동 저장된다.

## 성공 조건
- [ ] 토글 켜기 → 설정 간격마다 자동 저장된다.
- [ ] 토글 끄기 → 자동 저장이 비활성화된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 수동 저장 동작 (별도 spec).
- 내보내기 동작.
