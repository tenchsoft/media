# Spec: layer-row-select-control

## 한 줄 정의
사용자가 레이어 패널에서 레이어 행을 클릭하면 해당 레이어가 선택된다.

## 진입점
- 레이어 패널: 레이어 행 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어 행을 클릭한다.
2. 해당 레이어가 선택된다.
3. 선택된 레이어가 시각적으로 하이라이트된다.
4. 편집 동작이 해당 레이어에 적용된다.

## 성공 조건
- [ ] 해당 레이어가 선택된다.
- [ ] 시각적으로 하이라이트된다.
- [ ] 편집 동작이 해당 레이어에 적용된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 선택된 레이어 → 아무 변화가 없다.

## 범위 외
- 레이어 가시성 동작 (별도 spec).
- 레이어 잠금 동작.
