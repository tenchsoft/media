# Spec: layer-row-visibility-toggle

## 한 줄 정의
사용자가 레이어 패널에서 가시성 아이콘을 클릭하면 레이어가 보이거나 숨겨진다.

## 진입점
- 레이어 패널: 가시성 아이콘 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어의 눈 아이콘을 클릭한다.
2. 레이어가 숨겨진다 (캔버스에 표시되지 않음).
3. 다시 클릭하면 레이어가 다시 보인다.

## 성공 조건
- [ ] 클릭 시 레이어가 숨겨진다.
- [ ] 다시 클릭 시 레이어가 다시 보인다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외

- 레이어 불투명도 동작.
