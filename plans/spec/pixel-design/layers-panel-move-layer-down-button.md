# Spec: layers-panel-move-layer-down-button

## 한 줄 정의
사용자가 레이어 패널에서 아래로 이동 버튼을 클릭하면 선택된 레이어가 한 단계 아래로 이동한다.

## 진입점
- 레이어 패널: 아래로 이동 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어를 선택한다.
2. 아래로 이동 버튼을 클릭한다.
3. 선택된 레이어가 한 단계 아래로 이동한다.
4. 캔버스에 즉시 반영된다.

## 성공 조건
- [ ] 선택된 레이어가 한 단계 아래로 이동한다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 최하위 레이어 → 아무 동작도 발생하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 위로 이동 동작 (별도 spec).
- 레이어 삭제 동작.
