# Spec: canvas-snap-to-guide

## 한 줄 정의
사용자가 개체를 드래그할 때 가이드 라인에 스냅된다.

## 진입점
- 캔버스: 개체 드래그 (스냅 활성화 시)

## 사용자 흐름
1. 사용자가 보기 메뉴에서 스냅을 활성화한다.
2. 개체를 드래그한다.
3. 가이드 라인 근처에서 개체가 가이드에 스냅된다.

## 성공 조건
- [ ] 가이드 근처에서 개체가 스냅된다.
- [ ] 스냅 시 시각적 피드백이 표시된다.

## 실패 / 취소 흐름
- 스냅 비활성화 → 개체가 자유롭게 이동한다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 그리드 스냅 동작 (별도 spec).
- 가이드 생성 동작.
