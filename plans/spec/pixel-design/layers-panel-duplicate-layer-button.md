# Spec: layers-panel-duplicate-layer-button

## 한 줄 정의
사용자가 레이어 패널에서 레이어 복제 버튼을 클릭하면 선택된 레이어가 복제된다.

## 진입점
- 레이어 패널: 복제 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어를 선택한다.
2. 복제 버튼을 클릭한다.
3. 선택된 레이어의 복제본이 위에 생성된다.
4. 복제본이 선택된다.

## 성공 조건
- [ ] 선택된 레이어의 복제본이 생성된다.
- [ ] 복제본이 선택된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 레이어 추가 동작 (별도 spec).
- 레이어 삭제 동작.
