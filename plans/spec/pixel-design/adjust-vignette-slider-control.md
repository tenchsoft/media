# Spec: adjust-vignette-slider-control

## 한 줄 정의
사용자가 비네트 조정 슬라이더를 드래그하면 이미지 가장자리가 어두워진다.

## 진입점
- 조정 패널: 비네트 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 비네트 슬라이더를 드래그한다.
2. 이미지 가장자리가 실시간으로 어두워진다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 가장자리가 실시간으로 어두워진다.
- [ ] 중앙은 영향을 받지 않는다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 비네트 값으로 복원된다.

## 경계 / 예외
- 최소값 → 비네트가 적용되지 않는다.

## 범위 외
- 블러 조정 동작 (별도 spec).
- 대비 조정 동작.
