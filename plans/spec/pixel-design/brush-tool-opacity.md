# Spec: brush-tool-opacity

## 한 줄 정의
사용자가 브러시 불투명도를 조정하면 브러시 스트로크의 투명도가 변경된다.

## 진입점
- 속성 패널: 불투명도 슬라이더

## 사용자 흐름
1. 사용자가 브러시 도구를 선택한다.
2. 속성 패널에서 불투명도 슬라이더를 드래그한다.
3. 브러시 스트로크의 투명도가 변경된다.
4. 캔버스에 반영된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 브러시 불투명도가 실시간으로 변경된다.
- [ ] 100% → 완전 불투명.
- [ ] 낮은 값 → 반투명 스트로크.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 브러시 크기 동작 (별도 spec).
- 브러시 경도 동작.
