# Spec: export-quality-slider

## 한 줄 정의
사용자가 내보내기 품질 슬라이더를 조정하면 압축 품질이 설정된다.

## 진입점
- 내보내기 대화상자: 품질 슬라이더

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 품질 슬라이더를 드래그한다.
2. 품질 수치가 실시간으로 표시된다.
3. 파일 크기 예상치가 함께 표시된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 품질 수치가 실시간으로 표시된다.
- [ ] 파일 크기 예상치가 함께 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최소값 → 최저 품질, 최소 파일 크기.
- 최대값 → 최고 품질, 최대 파일 크기.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 형식 선택 동작.
