# Spec: gradient-tool-linear

## 한 줄 정의
사용자가 그라데이션 도구에서 선형을 선택하고 드래그하면 선형 그라데이션이 적용된다.

## 진입점
- 도구 모음: 그라데이션 도구 → 선형

## 사용자 흐름
1. 사용자가 도구 모음에서 그라데이션 도구를 선택한다.
2. 속성 패널에서 선형을 선택한다.
3. 캔버스에서 드래그하여 방향과 길이를 설정한다.
4. 선형 그라데이션이 적용된다.

## 성공 조건
- [ ] 드래그 방향에 따라 선형 그라데이션이 적용된다.
- [ ] 드래그 길이에 따라 그라데이션 범위가 결정된다.

## 실패 / 취소 흐름
- Esc → 그라데이션이 취소된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 방사형 그라데이션 동작 (별도 spec).
- 브러시 도구 동작.
