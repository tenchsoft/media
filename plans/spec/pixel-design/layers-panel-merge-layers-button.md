# Spec: layers-panel-merge-layers-button

## 한 줄 정의
사용자가 레이어 패널에서 병합 버튼을 클릭하면 선택된 레이어들이 하나로 합쳐진다.

## 진입점
- 레이어 패널: 병합 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 여러 레이어를 선택한다 (Ctrl+클릭).
2. 병합 버튼을 클릭한다.
3. 선택된 레이어들이 하나의 레이어로 합쳐진다.
4. 캔버스에 즉시 반영된다.

## 성공 조건
- [ ] 선택된 레이어들이 하나로 합쳐진다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 레이어가 1개만 선택됨 → 병합이 실행되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 레이어 평탄화 동작 (별도 spec).
- 레이어 삭제 동작.
