# Spec: color-picker-saturation-value-control

## 한 줄 정의
사용자가 색상 선택기에서 채도/명도 영역을 클릭하면 해당 색상이 선택된다.

## 진입점
- 색상 선택기: 채도/명도 영역 클릭/드래그

## 사용자 흐름
1. 사용자가 색상 선택기를 연다.
2. 채도/명도 영역에서 원하는 위치를 클릭한다.
3. 해당 색상이 선택된다.
4. 드래그하면 실시간으로 색상이 변경된다.

## 성공 조건
- [ ] 클릭/드래그 시 해당 색상이 선택된다.
- [ ] 실시간으로 색상이 변경된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 색상 슬라이더 동작 (별도 spec).
- 색상 선택기 확인/취소 동작.
