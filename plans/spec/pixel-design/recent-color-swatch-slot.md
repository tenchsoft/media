# Spec: recent-color-swatch-slot

## 한 줄 정의
사용자가 최근 사용 색상 스와치를 클릭하면 해당 색상이 전경색으로 설정된다.

## 진입점
- 색상 영역: 최근 사용 색상 스와치 클릭

## 사용자 흐름
1. 사용자가 최근 사용한 색상 스와치 중 하나를 클릭한다.
2. 해당 색상이 전경색으로 설정된다.
3. 전경색 스와치가 갱신된다.

## 성공 조건
- [ ] 클릭 시 해당 색상이 전경색으로 설정된다.
- [ ] 전경색 스와치가 갱신된다.

## 실패 / 취소 흐름
- 빈 슬롯 → 아무 동작도 발생하지 않는다.

## 경계 / 예외
- 최근 사용한 색상이 6개 초과 → 가장 오래된 것이 교체된다.

## 범위 외
- 색상 선택기 동작 (별도 spec).
- 전경색 스와치 동작.
