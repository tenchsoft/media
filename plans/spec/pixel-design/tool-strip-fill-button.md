# Spec: tool-strip-fill-button

## 한 줄 정의
사용자가 도구 스트립에서 채우기 버튼을 클릭하면 채우기 도구가 활성화된다.

## 진입점
- 도구 스트립: "Fill" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 채우기 버튼을 클릭한다.
2. 채우기 도구가 활성화된다.
3. 캔버스에서 클릭하면 전경색으로 해당 영역이 채워진다.

## 성공 조건
- [ ] 채우기 도구가 활성화된다.
- [ ] 클릭 시 전경색으로 영역이 채워진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 선택 영역이 있으면 → 선택 영역 내부만 채워진다.

## 범위 외
- 전경색 선택 동작 (별도 spec).
- 그라디언트 도구 동작.
