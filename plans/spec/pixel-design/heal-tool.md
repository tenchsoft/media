# Spec: heal-tool

## 한 줄 정의
사용자가 복원 도구로 드래그하면 주변 영역과 자연스럽게 blending되어 복원된다.

## 진입점
- 도구 모음: 복원 도구

## 사용자 흐름
1. 사용자가 도구 모음에서 복원 도구를 선택한다.
2. Alt+클릭으로 원본 영역을 설정한다.
3. 복원할 위치에서 드래그한다.
4. 주변 영역과 자연스럽게 blending되어 복원된다.

## 성공 조건
- [ ] Alt+클릭으로 원본 영역을 설정할 수 있다.
- [ ] 드래그 시 주변 영역과 자연스럽게 blending된다.

## 실패 / 취소 흐름
- 원본 미설정 상태에서 드래그 → "Alt+클릭으로 원본을 설정하세요" 토스트가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 복제 도장 도구 동작 (별도 spec).
- 브러시 도구 동작.
