# Spec: adjust-blur-slider-control

## 한 줄 정의
사용자가 블러 조정 슬라이더를 드래그하면 이미지에 블러 효과가 적용된다.

## 진입점
- 조정 패널: 블러 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 블러 슬라이더를 드래그한다.
2. 이미지에 블러 효과가 실시간으로 적용된다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 블러 효과가 실시간으로 적용된다.
- [ ] 슬라이더를 놓으면 값이 확정된다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 블러 값으로 복원된다.

## 경계 / 예외
- 최소값 → 블러가 적용되지 않는다.
- 최대값 → 이미지가 극도로 흐려진다.

## 범위 외
- 선명하게 조정 동작 (별도 spec).
- 브러시 도구 동작.
