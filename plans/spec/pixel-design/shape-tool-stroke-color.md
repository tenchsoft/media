# Spec: shape-tool-stroke-color

## 한 줄 정의
사용자가 도형의 선 색상을 변경하면 도형 테두리에 해당 색상이 적용된다.

## 진입점
- 속성 패널: 선 색상 버튼

## 사용자 흐름
1. 사용자가 도형 도구를 선택한다.
2. 속성 패널에서 선 색상 버튼을 클릭한다.
3. 색상 선택기에서 색상을 선택한다.
4. 도형 테두리에 해당 색상이 적용된다.

## 성공 조건
- [ ] 색상 선택기에서 색상을 선택할 수 있다.
- [ ] 선택한 색상이 도형 테두리에 적용된다.

## 실패 / 취소 흐름
- 취소 → 이전 색상이 유지된다.

## 경계 / 예외
- 선 없음 선택 → 테두리가 표시되지 않는다.

## 범위 외
- 채우기 색상 동작 (별도 spec).
- 선 굵기 동작.
