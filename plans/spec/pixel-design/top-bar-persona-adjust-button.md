# Spec: top-bar-persona-adjust-button

## 한 줄 정의
사용자가 상단 바의 조정 페르소나 버튼을 클릭하면 이미지 조정 도구가 활성화된다.

## 진입점
- 상단 바: "Adjust" 페르소나 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Adjust" 버튼을 클릭한다.
2. 조정 도구 슬라이더가 표시된다.
3. 밝기, 대비, 채도 등 조정 항목이 나타난다.

## 성공 조건
- [ ] 조정 도구 슬라이더가 표시된다.
- [ ] 조정 항목이 나타난다.

## 실패 / 취소 흐름
- 이미지가 열려 있지 않으면 → 조정 항목이 비활성화되어 있다.

## 경계 / 예외
- 이미 조정 모드 → 아무 변화가 없다.

## 범위 외
- 개별 조정 슬라이더 동작 (별도 spec).
- Edit 페르소나 동작.
