# Spec: color-picker-cancel-button

## 한 줄 정의
사용자가 색상 선택기에서 취소 버튼을 클릭하면 변경사항이 폐기된다.

## 진입점
- 색상 선택기: "Cancel" 버튼 클릭

## 사용자 흐름
1. 사용자가 색상 선택기를 연다.
2. 색상을 변경하다가 "Cancel" 버튼을 클릭한다.
3. 색상 변경이 폐기된다.
4. 이전 색상이 유지된다.
5. 색상 선택기가 닫힌다.

## 성공 조건
- [ ] 색상 변경이 폐기된다.
- [ ] 이전 색상이 유지된다.
- [ ] 색상 선택기가 닫힌다.

## 실패 / 취소 흐름
