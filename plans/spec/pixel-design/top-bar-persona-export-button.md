# Spec: top-bar-persona-export-button

## 한 줄 정의
사용자가 상단 바의 내보내기 페르소나 버튼을 클릭하면 내보내기 패널이 활성화된다.

## 진입점
- 상단 바: "Export" 페르소나 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Export" 버튼을 클릭한다.
2. 내보내기 패널이 표시된다.
3. 형식, 품질, 크기 설정이 나타난다.

## 성공 조건
- [ ] 내보내기 패널이 표시된다.
- [ ] 형식, 품질, 크기 설정이 나타난다.

## 실패 / 취소 흐름
- 이미지가 열려 있지 않으면 → 아무 동작도 발생하지 않는다.

## 경계 / 예외
- 이미 내보내기 모드 → 아무 변화가 없다.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 저장 동작.
