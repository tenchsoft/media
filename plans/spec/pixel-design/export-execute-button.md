# Spec: export-execute-button

## 한 줄 정의
사용자가 내보내기 버튼을 클릭하면 설정된 형식과 품질로 파일이 저장된다.

## 진입점
- 내보내기 대화상자: 내보내기 버튼 클릭

## 사용자 흐름
1. 사용자가 내보내기 설정을 확인한다.
2. 내보내기 버튼을 클릭한다.
3. 파일이 지정된 위치에 저장된다.
4. 완료 토스트가 표시된다.

## 성공 조건
- [ ] 클릭 시 파일이 지정된 위치에 저장된다.
- [ ] 완료 토스트가 표시된다.

## 실패 / 취소 흐름
- 디스크 공간 부족 → 오류 토스트가 표시된다.
- 쓰기 권한 없음 → 오류 토스트가 표시된다.

## 경계 / 예외
- 대용량 파일 → 진행 바가 표시된다.

## 범위 외
- 내보내기 설정 동작 (별도 spec).
- 저장 동작.
