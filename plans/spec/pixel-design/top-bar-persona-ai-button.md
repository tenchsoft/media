# Spec: top-bar-persona-ai-button

## 한 줄 정의
사용자가 상단 바의 AI 페르소나 버튼을 클릭하면 AI 도구 모음이 활성화된다.

## 진입점
- 상단 바: "AI" 페르소나 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "AI" 버튼을 클릭한다.
2. AI 도구 스트립이 활성화된다.
3. AI 패널이 표시된다.

## 성공 조건
- [ ] AI 도구 스트립이 활성화된다.
- [ ] AI 패널이 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 AI 모드 → 아무 변화가 없다.

## 범위 외
- Edit 페르소나 동작 (별도 spec).
- AI 작업 실행 동작.
