# Spec: clone-stamp-tool

## 한 줄 정의
사용자가 복제 도장 도구로 드래그하면 원본 영역이 복제되어 적용된다.

## 진입점
- 도구 모음: 복제 도장 도구

## 사용자 흐름
1. 사용자가 도구 모음에서 복제 도장 도구를 선택한다.
2. Alt+클릭으로 복제 원본 영역을 설정한다.
3. 다른 위치에서 드래그한다.
4. 원본 영역이 복제되어 드래그 위치에 적용된다.

## 성공 조건
- [ ] Alt+클릭으로 원본 영역을 설정할 수 있다.
- [ ] 드래그 시 원본 영역이 복제되어 적용된다.
- [ ] 원본과 대상 간의 상대적 위치가 유지된다.

## 실패 / 취소 흐름
- 원본 미설정 상태에서 드래그 → "Alt+클릭으로 원본을 설정하세요" 토스트가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 브러시 도구 동작 (별도 spec).
- 지우개 도구 동작.
