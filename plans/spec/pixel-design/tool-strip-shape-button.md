# Spec: tool-strip-shape-button

## 한 줄 정의
사용자가 도구 스트립에서 도형 버튼을 클릭하면 도형 도구가 활성화된다.

## 진입점
- 도구 스트립: "Shape" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 도형 버튼을 클릭한다.
2. 도형 도구가 활성화된다.
3. 캔버스에서 드래그하면 도형이 그려진다.

## 성공 조건
- [ ] 도형 도구가 활성화된다.
- [ ] 드래그 시 도형이 그려진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- Shift 누른 상태로 드래그 → 정비율 도형이 그려진다.

## 범위 외
- 도형 종류 선택 동작.
- 브러시 도구 동작 (별도 spec).
