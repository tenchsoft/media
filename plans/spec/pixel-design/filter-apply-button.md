# Spec: filter-apply-button

## 한 줄 정의
사용자가 필터 적용 버튼을 클릭하면 선택된 필터가 이미지에 적용된다.

## 진입점
- 필터 패널: 적용 버튼 클릭

## 사용자 흐름
1. 사용자가 필터 목록에서 필터를 선택한다.
2. 미리보기에서 필터 효과를 확인한다.
3. 적용 버튼을 클릭한다.
4. 필터가 이미지에 적용된다.

## 성공 조건
- [ ] 적용 버튼 클릭 시 필터가 이미지에 적용된다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 취소 버튼 → 필터가 적용되지 않고 이전 상태로 복원된다.

## 경계 / 예외
- 필터가 선택되지 않음 → 적용 버튼이 비활성화된다.

## 범위 외
- 필터 미리보기 동작 (별도 spec).
- 조정 패널 동작.
