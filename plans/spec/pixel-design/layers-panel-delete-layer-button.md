# Spec: layers-panel-delete-layer-button

## 한 줄 정의
사용자가 레이어 패널에서 레이어 삭제 버튼을 클릭하면 선택된 레이어가 삭제된다.

## 진입점
- 레이어 패널: 삭제 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어를 선택한다.
2. 삭제 버튼을 클릭한다.
3. 선택된 레이어가 삭제된다.

## 성공 조건
- [ ] 선택된 레이어가 삭제된다.

## 실패 / 취소 흐름
- 레이어가 하나뿐이면 → 삭제되지 않는다.

## 경계 / 예외
- 마지막 레이어 → 삭제되지 않고 경고가 표시된다.

## 범위 외
- 레이어 추가 동작 (별도 spec).
- Undo 동작.
