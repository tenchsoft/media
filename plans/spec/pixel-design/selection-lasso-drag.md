# Spec: selection-lasso-drag

## 한 줄 정의
사용자가 올가미 도구로 드래그하면 자유 형태 영역이 선택된다.

## 진입점
- 캔버스: 올가미 도구 활성화 후 드래그

## 사용자 흐름
1. 사용자가 도구 모음에서 올가미 도구를 선택한다.
2. 캔버스에서 자유 형태로 드래그한다.
3. 드래그 경로를 따라 선택 영역이 표시된다.
4. 드래그를 놓으면 시작점과 끝점이 연결되어 선택 영역이 확정된다.

## 성공 조건
- [ ] 드래그 경로를 따라 선택 영역이 실시간으로 표시된다.
- [ ] 드래그를 놓으면 시작점과 끝점이 연결된다.

## 실패 / 취소 흐름
- Esc → 선택이 해제된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 사각형 선택 동작 (별도 spec).
- 마술봉 선택 동작.
