# Spec: filter-category-tab

## 한 줄 정의
사용자가 필터 카테고리 탭을 클릭하면 해당 카테고리의 필터 목록이 표시된다.

## 진입점
- 필터 패널: 카테고리 탭 클릭

## 사용자 흐름
1. 사용자가 필터 패널에서 카테고리 탭을 클릭한다.
2. 해당 카테고리의 필터 목록이 표시된다.
3. 이전 카테고리 목록은 숨겨진다.

## 성공 조건
- [ ] 클릭 시 해당 카테고리 필터 목록이 표시된다.
- [ ] 이전 카테고리 목록은 숨겨진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 선택된 탭 클릭 → 아무 변화가 없다.

## 범위 외
- 필터 적용 동작 (별도 spec).
- 필터 미리보기 동작.
