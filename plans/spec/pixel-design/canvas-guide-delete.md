# Spec: canvas-guide-delete

## 한 줄 정의
사용자가 가이드 라인을 캔버스 밖으로 드래그하면 가이드가 삭제된다.

## 진입점
- 캔버스: 가이드 라인을 캔버스 밖으로 드래그

## 사용자 흐름
1. 사용자가 기존 가이드 라인을 클릭한 채 드래그한다.
2. 캔버스 영역 밖으로 이동한다.
3. 놓으면 가이드가 삭제된다.

## 성공 조건
- [ ] 캔버스 밖으로 드래그 후 놓으면 가이드가 삭제된다.

## 실패 / 취소 흐름
- 캔버스 안에 놓으면 가이드가 새 위치로 이동한다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 가이드 생성 동작 (별도 spec).
- 가이드 잠금 동작.
