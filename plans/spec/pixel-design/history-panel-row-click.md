# Spec: history-panel-row-click

## 한 줄 정의
사용자가 히스토리 패널에서 특정 상태를 클릭하면 해당 시점으로 돌아간다.

## 진입점
- 히스토리 패널: 히스토리 행 클릭

## 사용자 흐름
1. 사용자가 히스토리 패널을 연다.
2. 목록에서 특정 편집 상태를 클릭한다.
3. 캔버스가 해당 시점의 상태로 복원된다.

## 성공 조건
- [ ] 클릭 시 해당 시점의 상태로 캔버스가 복원된다.
- [ ] 클릭한 이후의 히스토리 항목은 회색으로 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최신 상태 클릭 → 현재 상태가 유지된다.

## 범위 외
- 실행 취소 동작 (별도 spec).
- 다시 실행 동작.
