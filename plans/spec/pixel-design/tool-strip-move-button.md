# Spec: tool-strip-move-button

## 한 줄 정의
사용자가 도구 스트립에서 이동 버튼을 클릭하면 이동 도구가 활성화된다.

## 진입점
- 도구 스트립: "Move" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 이동 버튼을 클릭한다.
2. 이동 도구가 활성화된다.
3. 캔버스에서 레이어를 드래그하면 위치가 이동한다.

## 성공 조건
- [ ] 이동 도구가 활성화된다.
- [ ] 레이어 드래그 시 위치가 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 이동 도구 → 아무 변화가 없다.
- 배경 레이어 잠금 → 이동할 수 없다.

## 범위 외
- 레이어 패널 동작 (별도 spec).
- 선택 도구 동작.
