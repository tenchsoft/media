# Spec: adjust-shadows-slider-control

## 한 줄 정의
사용자가 그림자 조정 슬라이더를 드래그하면 어두운 영역이 밝아지거나 어두워진다.

## 진입점
- 조정 패널: 그림자 슬라이더

## 사용자 흐름
1. 사용자가 조정 패널에서 그림자 슬라이더를 드래그한다.
2. 어두운 영역이 실시간으로 변경된다.
3. 슬라이더를 놓으면 값이 적용된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 어두운 영역이 실시간으로 변경된다.
- [ ] 밝은 영역은 영향을 받지 않는다.

## 실패 / 취소 흐름
- 취소 버튼 → 이전 그림자 값으로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 하이라이트 조정 동작 (별도 spec).
- 밝기 조정 동작.
