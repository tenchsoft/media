# Spec: top-bar-open-button

## 한 줄 정의
사용자가 상단 바의 열기 버튼을 클릭하면 파일 열기 대화상자가 나타난다.

## 진입점
- 상단 바: "Open" 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Open" 버튼을 클릭한다.
2. 파일 열기 대화상자가 나타난다.
3. 이미지 파일을 선택하고 확인한다.
4. 이미지가 캔버스에 열린다.

## 성공 조건
- [ ] 파일 열기 대화상자가 나타난다.
- [ ] 이미지 파일 선택 시 캔버스에 열린다.

## 실패 / 취소 흐름
- 대화상자에서 취소 → 아무 동작도 발생하지 않는다.
- 지원하지 않는 형식 → "지원하지 않는 파일 형식입니다" 토스트가 나타난다.

## 경계 / 예외
- 큰 파일 → 로딩 인디케이터가 표시된다.

## 범위 외
- 파일 저장 동작 (별도 spec).
- 드래그 앤 드롭 열기 동작.
