# Spec: canvas-hand-pan-control

## 한 줄 정의
사용자가 손 도구로 캔버스에서 드래그하면 화면이 이동한다.

## 진입점
- 캔버스: 손 도구로 드래그

## 사용자 흐름
1. 사용자가 손 도구를 선택한다.
2. 캔버스에서 마우스를 누르고 드래그한다.
3. 화면이 드래그 방향으로 이동한다.

## 성공 조건
- [ ] 드래그 시 화면이 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 스페이스바를 누른 상태에서 드래그 → 다른 도구에서도 화면 이동이 가능하다.

## 범위 외
- 줌 동작 (별도 spec).
- 이동 도구 동작.
