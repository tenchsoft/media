# Spec: top-bar-undo-button

## 한 줄 정의
사용자가 상단 바의 되돌리기 버튼을 클릭하면 마지막 편집 동작이 취소된다.

## 진입점
- 상단 바: "Undo" 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Undo" 버튼을 클릭한다.
2. 마지막 편집 동작이 취소된다.
3. 캔버스가 이전 상태로 복원된다.

## 성공 조건
- [ ] 마지막 편집 동작이 취소된다.
- [ ] 캔버스가 이전 상태로 복원된다.

## 실패 / 취소 흐름
- 취소할 동작이 없으면 → 버튼이 비활성화되어 있다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- Redo 동작 (별도 spec).
- Ctrl+Z 단축키 동작.
