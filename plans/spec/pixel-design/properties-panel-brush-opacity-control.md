# Spec: properties-panel-brush-opacity-control

## 한 줄 정의
사용자가 속성 패널에서 브러시 불투명도 슬라이더를 조정하면 브러시 투명도가 변경된다.

## 진입점
- 속성 패널: 브러시 불투명도 슬라이더

## 사용자 흐름
1. 사용자가 속성 패널의 브러시 불투명도 슬라이더를 드래그한다.
2. 브러시 투명도가 실시간으로 변경된다.
3. 0% → 완전 투명, 100% → 완전 불투명.

## 성공 조건
- [ ] 슬라이더 드래그 시 브러시 투명도가 실시간으로 변경된다.
- [ ] 0%에서 투명하고 100%에서 불투명하게 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 브러시 크기 동작 (별도 spec).
- 브러시 경도 동작.
