# Spec: selection-deselect

## 한 줄 정의
사용자가 선택 해제 동작을 수행하면 현재 선택 영역이 해제된다.

## 진입점
- 캔버스: Esc 키 또는 선택 영역 외부 클릭

## 사용자 흐름
1. 사용자가 선택 영역이 있는 상태에서 Esc 키를 누르거나 선택 영역 외부를 클릭한다.
2. 선택 영역이 해제된다.

## 성공 조건
- [ ] Esc 키 또는 외부 클릭 시 선택 영역이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 선택 생성 동작 (별도 spec).
- 선택 영역 이동 동작.
