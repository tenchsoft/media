# Spec: top-bar-persona-edit-button

## 한 줄 정의
사용자가 상단 바의 편집 페르소나 버튼을 클릭하면 편집 도구 모음이 활성화된다.

## 진입점
- 상단 바: "Edit" 페르소나 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바의 "Edit" 버튼을 클릭한다.
2. 편집 도구 스트립이 활성화된다.
3. 편집 관련 패널이 표시된다.

## 성공 조건
- [ ] 편집 도구 스트립이 활성화된다.
- [ ] 편집 관련 패널이 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 편집 모드 → 아무 변화가 없다.

## 범위 외
- AI 페르소나 동작 (별도 spec).
- Adjust 페르소나 동작.
