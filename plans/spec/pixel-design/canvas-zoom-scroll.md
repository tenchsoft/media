# Spec: canvas-zoom-scroll

## 한 줄 정의
사용자가 마우스 휠을 스크롤하면 캔버스가 확대 또는 축소된다.

## 진입점
- 캔버스: 마우스 휠 스크롤

## 사용자 흐름
1. 사용자가 캔버스 위에서 마우스 휠을 위로 스크롤한다.
2. 캔버스가 확대된다.
3. 마우스 휠을 아래로 스크롤하면 캔버스가 축소된다.

## 성공 조건
- [ ] 위로 스크롤 시 캔버스가 확대된다.
- [ ] 아래로 스크롤 시 캔버스가 축소된다.
- [ ] 마우스 커서 위치를 중심으로 확대/축소된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최대 확대 → 더 이상 확대되지 않는다.
- 최소 축소 → 더 이상 축소되지 않는다.

## 범위 외
- 핸드 팬 동작 (별도 spec).
- 미니맵 동작.
