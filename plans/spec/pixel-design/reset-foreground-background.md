# Spec: reset-foreground-background

## 한 줄 정의
사용자가 기본색 재설정 버튼을 클릭하면 전경색이 검정, 배경색이 흰색으로 재설정된다.

## 진입점
- 도구 모음: 기본색 재설정 버튼

## 사용자 흐름
1. 사용자가 기본색 재설정 버튼을 클릭한다.
2. 전경색이 검정으로, 배경색이 흰색으로 재설정된다.

## 성공 조건
- [ ] 클릭 시 전경색이 검정, 배경색이 흰색으로 재설정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전경색/배경색 교환 동작 (별도 spec).
- 색상 선택기 동작.
