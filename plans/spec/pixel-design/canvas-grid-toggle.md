# Spec: canvas-grid-toggle

## 한 줄 정의
사용자가 그리드 토글을 클릭하면 캔버스에 격자가 표시되거나 숨겨진다.

## 진입점
- 보기 메뉴: 그리드 토글 클릭

## 사용자 흐름
1. 사용자가 보기 메뉴에서 그리드 토글을 클릭한다.
2. 캔버스에 격자가 표시된다.
3. 다시 클릭하면 격자가 숨겨진다.

## 성공 조건
- [ ] 클릭 시 캔버스에 격자가 표시된다.
- [ ] 다시 클릭 시 격자가 숨겨진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 확대/축소 시 격자 간격이 조정된다.

## 범위 외
- 가이드 라인 동작 (별도 spec).
- 눈금자 동작.
