# Spec: canvas-eraser-stroke-control

## 한 줄 정의
사용자가 지우개 도구로 캔버스에서 드래그하면 해당 영역이 지워진다.

## 진입점
- 캔버스: 지우개 도구로 드래그

## 사용자 흐름
1. 사용자가 지우개 도구를 선택한다.
2. 캔버스에서 마우스를 누르고 드래그한다.
3. 드래그한 영역이 지워진다.
4. 마우스를 놓으면 지우기가 완성된다.

## 성공 조건
- [ ] 드래그 중 영역이 실시간으로 지워진다.
- [ ] 마우스를 놓으면 지우기가 완성된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 배경 레이어 → 배경색으로 칠해진다.
- 투명 레이어 → 투명해진다.

## 범위 외
- 브러시 동작 (별도 spec).
- 레이어 동작.
