# Spec: canvas-cursor-crosshair

## 한 줄 정의
사용자가 그리기 도구를 선택하면 캔버스 위에 십자선 커서가 표시된다.

## 진입점
- 캔버스: 그리기 도구 선택 시 자동

## 사용자 흐름
1. 사용자가 브러시, 지우개 등 그리기 도구를 선택한다.
2. 캔버스 위에 십자선 커서가 표시된다.
3. 브러시 크기에 따라 커서 크기가 변경된다.

## 성공 조건
- [ ] 그리기 도구 선택 시 십자선 커서가 표시된다.
- [ ] 브러시 크기에 따라 커서 크기가 변경된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 캔버스 밖 → 일반 커서로 돌아간다.

## 범위 외
- 이동 도구 커서 동작 (별도 spec).
- 텍스트 도구 커서 동작.
