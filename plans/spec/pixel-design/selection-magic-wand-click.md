# Spec: selection-magic-wand-click

## 한 줄 정의
사용자가 마술봉 도구로 클릭하면 비슷한 색상 영역이 자동으로 선택된다.

## 진입점
- 캔버스: 마술봉 도구 활성화 후 클릭

## 사용자 흐름
1. 사용자가 도구 모음에서 마술봉 도구를 선택한다.
2. 캔버스에서 원하는 영역을 클릭한다.
3. 클릭한 위치와 비슷한 색상의 인접 영역이 자동으로 선택된다.

## 성공 조건
- [ ] 클릭 시 비슷한 색상의 인접 영역이 자동으로 선택된다.
- [ ] 허용도 설정에 따라 선택 범위가 조정된다.

## 실패 / 취소 흐름
- Esc → 선택이 해제된다.

## 경계 / 예외
- 허용도가 낮으면 매우 비슷한 색상만 선택된다.
- 허용도가 높으면 넓은 범위의 색상이 선택된다.

## 범위 외
- 사각형 선택 동작 (별도 spec).
- 올가미 선택 동작.
