# Spec: tool-strip-gradient-button

## 한 줄 정의
사용자가 도구 스트립에서 그라디언트 버튼을 클릭하면 그라디언트 도구가 활성화된다.

## 진입점
- 도구 스트립: "Gradient" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 그라디언트 버튼을 클릭한다.
2. 그라디언트 도구가 활성화된다.
3. 캔버스에서 드래그하면 전경색에서 배경색으로 그라디언트가 생성된다.

## 성공 조건
- [ ] 그라디언트 도구가 활성화된다.
- [ ] 드래그 시 그라디언트가 생성된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 선택 영역이 있으면 → 선택 영역 내부에만 그라디언트가 적용된다.

## 범위 외
- 채우기 도구 동작 (별도 spec).
- 전경색/배경색 선택 동작.
