# Spec: export-format-dropdown

## 한 줄 정의
사용자가 내보내기 형식 드롭다운에서 형식을 선택하면 해당 형식으로 내보내기가 설정된다.

## 진입점
- 내보내기 대화상자: 형식 드롭다운 클릭

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 형식 드롭다운을 클릭한다.
2. 지원 형식 목록이 표시된다 (PNG, JPEG, WebP, TIFF, BMP).
3. 원하는 형식을 선택한다.
4. 선택된 형식이 표시된다.

## 성공 조건
- [ ] 클릭 시 지원 형식 목록이 표시된다.
- [ ] 선택 시 해당 형식이 설정된다.

## 실패 / 취소 흐름
- 바깥 클릭 → 드롭다운이 닫히고 이전 선택이 유지된다.

## 경계 / 예외
- JPEG 선택 → 품질 슬라이더가 추가로 표시된다.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 저장 동작.
