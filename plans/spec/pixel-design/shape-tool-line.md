# Spec: shape-tool-line

## 한 줄 정의
사용자가 도형 도구에서 선을 선택하고 드래그하면 선이 그려진다.

## 진입점
- 도구 모음: 도형 도구 → 선

## 사용자 흐름
1. 사용자가 도구 모음에서 도형 도구를 선택한다.
2. 속성 패널에서 선을 선택한다.
3. 캔버스에서 드래그한다.
4. 선이 그려진다.

## 성공 조건
- [ ] 드래그 시 선이 실시간으로 표시된다.
- [ ] 드래그를 놓으면 선이 확정된다.

## 실패 / 취소 흐름
- Esc → 그리기가 취소된다.

## 경계 / 예외
- Shift 드래그 → 수평/수직/45도로 제한된다.

## 범위 외
- 사각형 도구 동작 (별도 spec).
- 타원 도구 동작.
