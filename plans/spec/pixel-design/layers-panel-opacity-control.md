# Spec: layers-panel-opacity-control

## 한 줄 정의
사용자가 레이어 패널에서 불투명도 슬라이더를 조정하면 레이어 투명도가 변경된다.

## 진입점
- 레이어 패널: 불투명도 슬라이더

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어를 선택한다.
2. 불투명도 슬라이더를 드래그한다.
3. 레이어 투명도가 실시간으로 변경된다.
4. 캔버스에 즉시 반영된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 레이어 투명도가 실시간으로 변경된다.
- [ ] 캔버스에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 0% → 레이어가 완전 투명해진다.

## 범위 외
- 레이어 가시성 동작 (별도 spec).
- 브러시 불투명도 동작.
