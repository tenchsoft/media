# Spec: canvas-move-layer-drag-control

## 한 줄 정의
사용자가 이동 도구로 캔버스에서 레이어를 드래그하면 레이어 위치가 이동한다.

## 진입점
- 캔버스: 이동 도구로 레이어 드래그

## 사용자 흐름
1. 사용자가 이동 도구를 선택한다.
2. 캔버스에서 레이어를 마우스로 누르고 드래그한다.
3. 레이어가 실시간으로 이동한다.
4. 마우스를 놓으면 위치가 확정된다.

## 성공 조건
- [ ] 드래그 중 레이어가 실시간으로 이동한다.
- [ ] 마우스를 놓으면 위치가 확정된다.

## 실패 / 취소 흐름
- 배경 레이어 잠금 → 이동할 수 없다.

## 경계 / 예외
- Shift 누른 상태 → 수평/수직으로만 이동한다.

## 범위 외
- 레이어 패널 동작 (별도 spec).
- 선택 도구 동작.
