# Spec: tool-strip-select-button

## 한 줄 정의
사용자가 도구 스트립에서 선택 버튼을 클릭하면 선택 도구가 활성화된다.

## 진입점
- 도구 스트립: "Select" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 선택 버튼을 클릭한다.
2. 선택 도구가 활성화된다.
3. 캔버스에서 드래그하면 사각형 선택 영역이 만들어진다.

## 성공 조건
- [ ] 선택 도구가 활성화된다.
- [ ] 드래그 시 사각형 선택 영역이 만들어진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 선택 도구 → 아무 변화가 없다.

## 범위 외
- 선택 영역 편집 동작.
- 브러시 도구 동작 (별도 spec).
