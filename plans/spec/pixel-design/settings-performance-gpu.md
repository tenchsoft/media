# Spec: settings-performance-gpu

## 한 줄 정의
사용자가 GPU 가속 설정을 켜면 캔버스 렌더링에 GPU가 사용된다.

## 진입점
- 설정: 성능 탭 → GPU 가속 토글

## 사용자 흐름
1. 사용자가 설정에서 성능 탭을 선택한다.
2. GPU 가속 토글을 켠다.
3. 재시작 후 GPU 가속이 적용된다.

## 성공 조건
- [ ] 토글 켜기 → 재시작 후 GPU 가속이 적용된다.
- [ ] 토글 끄기 → CPU 렌더링이 사용된다.

## 실패 / 취소 흐름
- GPU 미지원 → 토글이 비활성화된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 일반 설정 동작 (별도 spec).
- 캔버스 확대/축소 동작.
