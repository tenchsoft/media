# Spec: layers-panel-rename-layer

## 한 줄 정의
사용자가 레이어 패널에서 레이어 이름을 더블 클릭하면 이름을 편집할 수 있다.

## 진입점
- 레이어 패널: 레이어 이름 더블 클릭

## 사용자 흐름
1. 사용자가 레이어 패널에서 레이어 이름을 더블 클릭한다.
2. 인라인 텍스트 편집 모드가 활성화된다.
3. 새 이름을 입력한다.
4. Enter 키를 누르거나 바깥 영역을 클릭하면 이름이 저장된다.

## 성공 조건
- [ ] 더블 클릭 시 인라인 편집 모드가 활성화된다.
- [ ] Enter 또는 바깥 클릭 시 이름이 저장된다.

## 실패 / 취소 흐름
- Esc → 편집이 취소되고 이전 이름이 유지된다.

## 경계 / 예외
- 빈 이름 → 편집이 취소되고 이전 이름이 유지된다.

## 범위 외
- 레이어 삭제 동작 (별도 spec).
- 레이어 병합 동작.
