# Spec: selection-invert

## 한 줄 정의
사용자가 선택 반전 동작을 수행하면 선택된 영역과 선택되지 않은 영역이 서로 바뀐다.

## 진입점
- 선택 메뉴: 선택 반전 클릭

## 사용자 흐름
1. 사용자가 선택 영역이 있는 상태에서 선택 반전을 클릭한다.
2. 선택된 영역과 선택되지 않은 영역이 바뀐다.

## 성공 조건
- [ ] 클릭 시 선택 영역이 반전된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 선택 생성 동작 (별도 spec).
- 선택 해제 동작.
