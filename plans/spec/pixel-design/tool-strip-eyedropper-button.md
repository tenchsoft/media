# Spec: tool-strip-eyedropper-button

## 한 줄 정의
사용자가 도구 스트립에서 스포이드 버튼을 클릭하면 스포이드 도구가 활성화된다.

## 진입점
- 도구 스트립: "Eyedropper" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 스포이드 버튼을 클릭한다.
2. 스포이드 도구가 활성화된다.
3. 캔버스에서 클릭하면 해당 위치의 색상이 전경색으로 설정된다.

## 성공 조건
- [ ] 스포이드 도구가 활성화된다.
- [ ] 클릭 시 해당 위치의 색상이 전경색으로 설정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미지가 열려 있지 않으면 → 아무 동작도 발생하지 않는다.

## 범위 외
- 전경색 스와치 동작 (별도 spec).
- 색상 선택기 동작.
