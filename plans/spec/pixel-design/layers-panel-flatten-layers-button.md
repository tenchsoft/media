# Spec: layers-panel-flatten-layers-button

## 한 줄 정의
사용자가 레이어 패널에서 레이어 병합 버튼을 클릭하면 모든 레이어가 하나로 병합된다.

## 진입점
- 레이어 패널: 병합 버튼 클릭

## 사용자 흐름
1. 사용자가 레이어 패널의 병합 버튼을 클릭한다.
2. 확인 대화상자가 나타난다.
3. 확인 시 모든 레이어가 하나로 병합된다.

## 성공 조건
- [ ] 확인 시 모든 레이어가 하나로 병합된다.
- [ ] 병합 후 단일 레이어만 남는다.

## 실패 / 취소 흐름
- 확인 대화상자에서 취소 → 병합되지 않는다.

## 경계 / 예외
- 레이어가 하나뿐이면 → 아무 동작도 발생하지 않는다.

## 범위 외
- 레이어 추가 동작 (별도 spec).
- Undo 동작.
