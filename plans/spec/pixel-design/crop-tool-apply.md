# Spec: crop-tool-apply

## 한 줄 정의
사용자가 자르기 도구에서 영역을 선택하고 적용하면 이미지가 해당 영역으로 잘린다.

## 진입점
- 도구 모음: 자르기 도구 → 드래그 → 적용 버튼

## 사용자 흐름
1. 사용자가 도구 모음에서 자르기 도구를 선택한다.
2. 캔버스에서 드래그하여 자를 영역을 설정한다.
3. 영역을 조정한다 (핸들 드래그).
4. 적용 버튼을 클릭하거나 Enter 키를 누른다.
5. 이미지가 해당 영역으로 잘린다.

## 성공 조건
- [ ] 드래그 시 자르기 영역이 실시간으로 표시된다.
- [ ] 적용 시 이미지가 해당 영역으로 잘린다.
- [ ] 영역 외부가 어둡게 표시된다.

## 실패 / 취소 흐름
- Esc → 자르기가 취소된다.

## 경계 / 예외
- 영역이 너무 작음 → 경고가 표시된다.

## 범위 외
- 캔버스 크기 변경 동작 (별도 spec).
- 변형 동작.
