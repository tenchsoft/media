# Spec: text-tool-text-align

## 한 줄 정의
사용자가 텍스트 정렬 버튼을 클릭하면 텍스트가 해당 정렬로 표시된다.

## 진입점
- 속성 패널: 정렬 버튼 (좌/중앙/우/양쪽)

## 사용자 흐름
1. 사용자가 텍스트 도구를 선택한다.
2. 속성 패널에서 정렬 버튼을 클릭한다.
3. 텍스트가 해당 정렬로 표시된다.

## 성공 조건
- [ ] 클릭 시 텍스트가 해당 정렬로 표시된다.
- [ ] 선택된 정렬 버튼이 활성 상태로 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.
