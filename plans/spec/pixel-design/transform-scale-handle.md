# Spec: transform-scale-handle

## 한 줄 정의
사용자가 크기 조절 핸들을 드래그하면 이미지 크기가 변경된다.

## 진입점
- 변형 모드: 모서리 핸들 드래그

## 사용자 흐름
1. 사용자가 변형 모드에서 모서리 핸들을 잡는다.
2. 드래그하여 크기를 조절한다.
3. 핸들을 놓으면 크기가 확정된다.

## 성공 조건
- [ ] 드래그 시 이미지 크기가 실시간으로 변경된다.
- [ ] 핸들을 놓으면 크기가 확정된다.

## 실패 / 취소 흐름
- Esc → 크기 변경이 취소되고 원래 크기로 복원된다.

## 경계 / 예외
- Shift 드래그 → 비율이 유지된다.
- 최소 크기 제한이 있다.

## 범위 외
- 회전 동작 (별도 spec).
- 자르기 동작.
