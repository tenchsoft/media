# Spec: swap-foreground-background

## 한 줄 정의
사용자가 전경색/배경색 교환 버튼을 클릭하면 두 색상이 서로 바뀐다.

## 진입점
- 도구 모음: 전경색/배경색 교환 버튼

## 사용자 흐름
1. 사용자가 전경색/배경색 교환 버튼을 클릭한다.
2. 전경색과 배경색이 서로 바뀐다.

## 성공 조건
- [ ] 클릭 시 전경색과 배경색이 서로 바뀐다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전경색 선택 동작 (별도 spec).
- 배경색 선택 동작.
