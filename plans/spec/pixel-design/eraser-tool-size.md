# Spec: eraser-tool-size

## 한 줄 정의
사용자가 지우개 크기를 조정하면 지우개 지름이 변경된다.

## 진입점
- 속성 패널: 크기 슬라이더

## 사용자 흐름
1. 사용자가 지우개 도구를 선택한다.
2. 속성 패널에서 크기 슬라이더를 드래그한다.
3. 지우개 지름이 변경된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 지우개 크기가 실시간으로 변경된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최소 크기 → 1px.

## 범위 외
- 브러시 크기 동작 (별도 spec).
- 지우개 경도 동작.
