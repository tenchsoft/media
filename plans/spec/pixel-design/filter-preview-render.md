# Spec: filter-preview-render

## 한 줄 정의
사용자가 필터를 선택하면 미리보기에 필터 효과가 실시간으로 표시된다.

## 진입점
- 필터 패널: 필터 선택

## 사용자 흐름
1. 사용자가 필터 목록에서 필터를 가리킨다.
2. 미리보기에 필터 효과가 실시간으로 표시된다.
3. 다른 필터를 가리키면 미리보기가 전환된다.

## 성공 조건
- [ ] 필터 가리킬 때 미리보기에 필터 효과가 표시된다.
- [ ] 다른 필터로 이동 시 미리보기가 전환된다.

## 실패 / 취소 흐름
- 필터를 벗어남 → 원본 이미지로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 필터 적용 동작 (별도 spec).
- 조정 패널 동작.
