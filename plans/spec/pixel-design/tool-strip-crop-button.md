# Spec: tool-strip-crop-button

## 한 줄 정의
사용자가 도구 스트립에서 자르기 버튼을 클릭하면 자르기 도구가 활성화된다.

## 진입점
- 도구 스트립: "Crop" 버튼 클릭

## 사용자 흐름
1. 사용자가 도구 스트립에서 자르기 버튼을 클릭한다.
2. 자르기 도구가 활성화된다.
3. 캔버스에서 드래그하면 자르기 영역이 표시된다.
4. Enter를 누르면 영역 외부가 잘린다.

## 성공 조건
- [ ] 자르기 도구가 활성화된다.
- [ ] 드래그 시 자르기 영역이 표시된다.
- [ ] Enter 시 영역 외부가 잘린다.

## 실패 / 취소 흐름
- Escape → 자르기가 취소된다.

## 경계 / 예외
- 영역이 너무 작으면 → 자르기가 적용되지 않는다.

## 범위 외
- 캔버스 크기 변경 동작.
- 브러시 도구 동작 (별도 spec).
