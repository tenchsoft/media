# Spec: export-resize-width-height

## 한 줄 정의
사용자가 내보내기 크기 입력 필드에서 너비와 높이를 변경하면 내보내기 해상도가 설정된다.

## 진입점
- 내보내기 대화상자: 너비/높이 입력 필드

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 너비 또는 높이 입력 필드를 클릭한다.
2. 새 값을 입력한다.
3. Enter 키를 누르거나 바깥을 클릭하면 값이 적용된다.

## 성공 조건
- [ ] 너비 변경 시 비율이 유지되면 높이도 자동 조정된다.
- [ ] 값이 유효하면 즉시 반영된다.

## 실패 / 취소 흐름
- 잘못된 값 → 이전 값으로 복원된다.

## 경계 / 예외
- 비율 잠금이 켜져 있으면 너비/높이가 비율에 맞게 자동 조정된다.
- 비율 잠금이 꺼져 있으면 독립적으로 조정된다.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 품질 슬라이더 동작.
