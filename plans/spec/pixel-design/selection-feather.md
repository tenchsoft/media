# Spec: selection-feather

## 한 줄 정의
사용자가 선택 영역에 feather 값을 적용하면 선택 영역 가장자리가 부드러워진다.

## 진입점
- 선택 메뉴: Feather 설정

## 사용자 흐름
1. 사용자가 선택 영역이 있는 상태에서 Feather 값을 입력한다.
2. 선택 영역 가장자리가 부드러워진다.
3. 이후 편집 동작이 부드러운 가장자리로 적용된다.

## 성공 조건
- [ ] Feather 값 적용 시 선택 영역 가장자리가 부드러워진다.
- [ ] 편집 동작이 부드러운 가장자리로 적용된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 0 → feather가 적용되지 않는다.

## 범위 외
- 선택 생성 동작 (별도 spec).
- 선택 반전 동작.
