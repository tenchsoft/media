# Spec: brush-tool-size

## 한 줄 정의
사용자가 브러시 크기를 조정하면 브러시 지름이 변경된다.

## 진입점
- 속성 패널: 크기 슬라이더 또는 [ / ] 단축키

## 사용자 흐름
1. 사용자가 브러시 도구를 선택한다.
2. 속성 패널에서 크기 슬라이더를 드래그하거나 [ / ] 키를 누른다.
3. 브러시 지름이 변경된다.
4. 캔버스 위에 브러시 크기 미리보기가 표시된다.

## 성공 조건
- [ ] 슬라이더 또는 단축키로 브러시 크기가 변경된다.
- [ ] 캔버스 위에 브러시 크기 미리보기가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최소 크기 → 1px.
- 최대 크기 제한이 있다.

## 범위 외
- 브러시 경도 동작 (별도 spec).
- 브러시 불투명도 동작.
