# Spec: bottom-toolbar-info

## 한 줄 정의
사용자가 정보 버튼을 누르면 이미지 메타데이터가 표시된다.

## 진입점
- 하단 도구 모음: 정보 버튼 클릭

## 사용자 흐름
1. 사용자가 정보 버튼을 클릭한다.
2. 이미지 정보 패널이 열린다.
3. 파일 크기, 해상도, 포맷, EXIF 데이터 등이 표시된다.

## 성공 조건
- [ ] 클릭 시 이미지 정보 패널이 열린다.
- [ ] 파일 메타데이터가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- EXIF 데이터가 없으면 기본 정보만 표시된다.

## 범위 외
- EXIF 편집 동작.
