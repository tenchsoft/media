# Spec: window-resize-reflow

## 한 줄 정의
창 크기를 변경하면 이미지가 새 창 크기에 맞게 자동 조정된다.

## 진입점
- 자동: 창 크기 변경 시

## 사용자 흐름
1. 사용자가 창 크기를 조정한다.
2. 이미지가 새 창 크기에 맞게 자동 조정된다.
3. 화면에 맞춤 모드가 유지된다.

## 성공 조건
- [ ] 창 크기 변경 시 이미지가 자동 조정된다.
- [ ] 비율이 유지된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 원본 크기 모드에서는 조정되지 않는다.

## 범위 외
- 확대/축소 동작 (별도 spec).
