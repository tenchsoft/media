# Spec: bottom-toolbar-rotate-right

## 한 줄 정의
사용자가 우회전 버튼을 누르면 이미지가 시계 방향으로 90도 회전한다.

## 진입점
- 하단 도구 모음: 우회전 버튼 클릭

## 사용자 흐름
1. 사용자가 우회전 버튼을 클릭한다.
2. 이미지가 시계 방향으로 90도 회전하여 표시된다.

## 성공 조건
- [ ] 클릭 시 이미지가 시계 방향으로 90도 회전한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 좌회전 동작 (별도 spec).
- EXIF 회전 동작.
