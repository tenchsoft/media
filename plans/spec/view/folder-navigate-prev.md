# Spec: folder-navigate-prev

## 한 줄 정의
사용자가 이전 이미지 버튼을 누르면 같은 폴더의 이전 이미지가 표시된다.

## 진입점
- 탐색 버튼: 이전 버튼 / 왼쪽 화살표 키

## 사용자 흐름
1. 사용자가 이전 버튼을 클릭한다.
2. 같은 폴더의 이전 이미지가 표시된다.

## 성공 조건
- [ ] 클릭 시 이전 이미지가 표시된다.
- [ ] 왼쪽 화살표 키로도 이동할 수 있다.

## 실패 / 취소 흐름
- 첫 번째 이미지 → 마지막 이미지로 이동한다.

## 경계 / 예외
- 폴더에 이미지가 1장뿐이면 → 버튼이 비활성화된다.

## 범위 외
- 다음 이미지 동작 (별도 spec).
