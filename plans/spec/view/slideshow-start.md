# Spec: slideshow-start

## 한 줄 정의
사용자가 슬라이드쇼 버튼을 누르면 현재 폴더의 이미지가 자동으로 전환된다.

## 진입점
- 하단 도구 모음: 슬라이드쇼 버튼

## 사용자 흐름
1. 사용자가 슬라이드쇼 버튼을 클릭한다.
2. 슬라이드쇼가 시작된다.
3. 설정된 간격으로 이미지가 자동 전환된다.
4. 클릭 또는 Esc로 슬라이드쇼를 종료한다.

## 성공 조건
- [ ] 클릭 시 슬라이드쇼가 시작된다.
- [ ] 이미지가 자동으로 전환된다.
- [ ] Esc로 종료할 수 있다.

## 실패 / 취소 흐름
- 폴더에 이미지가 1장뿐이면 → 슬라이드쇼가 시작되지 않는다.

## 경계 / 예외
- 마지막 이미지에서 처음으로 돌아간다.

## 범위 외
- 슬라이드쇼 속도 설정 동작.
