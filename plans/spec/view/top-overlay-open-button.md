# Spec: top-overlay-open-button

## 한 줄 정의
빈 상태에서 파일 열기 버튼을 누르면 파일 선택 대화상자가 나타난다.

## 진입점
- 빈 상태 오버레이: 파일 열기 버튼

## 사용자 흐름
1. 이미지가 열려 있지 않은 빈 상태에서 파일 열기 버튼이 표시된다.
2. 사용자가 버튼을 클릭한다.
3. 파일 선택 대화상자가 나타난다.
4. 이미지를 선택하여 연다.

## 성공 조건
- [ ] 빈 상태에서 파일 열기 버튼이 표시된다.
- [ ] 클릭 시 파일 선택 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 빈 상태가 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 열기 동작 (별도 spec).
