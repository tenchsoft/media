# Spec: bottom-toolbar-open-file

## 한 줄 정의
사용자가 파일 열기 버튼을 누르면 파일 선택 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 파일 열기 버튼 클릭

## 사용자 흐름
1. 사용자가 파일 열기 버튼을 클릭한다.
2. 파일 선택 대화상자가 나타난다.
3. 이미지 파일을 선택한다.
4. 선택한 이미지가 표시된다.

## 성공 조건
- [ ] 클릭 시 파일 선택 대화상자가 표시된다.
- [ ] 이미지 선택 시 이미지가 표시된다.

## 실패 / 취소 흐름
- 파일 선택 취소 → 아무 일도 일어나지 않는다.

## 경계 / 예외
- 지원하지 않는 형식 → 오류 메시지가 표시된다.

## 범위 외
- 드래그 앤 드롭 동작 (별도 spec).
