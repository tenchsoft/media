# Spec: info-panel-exif

## 한 줄 정의
정보 패널에서 이미지의 EXIF 데이터를 확인할 수 있다.

## 진입점
- 정보 패널: EXIF 탭

## 사용자 흐름
1. 사용자가 정보 패널을 연다.
2. EXIF 탭을 선택한다.
3. 카메라 모델, 촬영 날짜, 렌즈 정보, 노출 설정 등이 표시된다.

## 성공 조건
- [ ] EXIF 탭에서 메타데이터가 표시된다.
- [ ] 카메라, 촬영 설정, GPS 등 항목이 포함된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- EXIF 데이터가 없으면 "정보 없음"이 표시된다.

## 범위 외
- EXIF 편집 동작.
