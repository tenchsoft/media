# Spec: context-menu-properties

## 한 줄 정의
우클릭 메뉴에서 속성을 선택하면 파일 속성 대화상자가 나타난다.

## 진입점
- 이미지 영역: 우클릭 → "속성" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "속성"을 선택한다.
4. 파일 속성 대화상자가 나타난다.
5. 파일 크기, 해상도, 생성/수정 날짜 등이 표시된다.

## 성공 조건
- [ ] "속성" 선택 시 파일 속성 대화상자가 나타난다.
- [ ] 파일 정보가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.
