# Spec: annotation-save

## 한 줄 정의
주석 편집 완료 후 저장하면 주석이 이미지에 합성되어 저장된다.

## 진입점
- 주석 모드: 저장 버튼

## 사용자 흐름
1. 사용자가 주석을 그린다.
2. 저장 버튼을 누른다.
3. 주석이 이미지에 합성된다.
4. 파일로 저장된다.

## 성공 조건
- [ ] 주석이 이미지에 합성된다.
- [ ] 파일로 저장된다.

## 실패 / 취소 흐름
- 저장 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 그리기 동작 (별도 spec).
