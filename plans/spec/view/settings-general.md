# Spec: settings-general

## 한 줄 정의
설정 대화상자에서 뷰어 일반 동작을 변경할 수 있다.

## 진입점
- 설정 대화상자: 일반 탭

## 사용자 흐름
1. 사용자가 설정을 연다.
2. 일반 탭을 선택한다.
3. 시작 시 마지막 파일 열기, 언어, 테마 등을 변경한다.
4. 확인 버튼을 누른다.
5. 설정이 저장된다.

## 성공 조건
- [ ] 일반 설정 항목이 표시된다.
- [ ] 변경한 설정이 저장된다.

## 실패 / 취소 흐름
- 취소 → 설정이 유지된다.

## 경계 / 예외
- 일부 설정은 재시작 후 적용된다.

## 범위 외
- 개별 설정 항목 동작.
