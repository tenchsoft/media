# Spec: annotation-draw-rectangle

## 한 줄 정의
사용자가 사각형 도구를 선택하면 이미지 위에 사각형을 그릴 수 있다.

## 진입점
- 주석 도구 모음: 사각형 도구 클릭

## 사용자 흐름
1. 사용자가 사각형 도구를 선택한다.
2. 이미지 위에서 드래그하여 사각형 영역을 지정한다.
3. 사각형이 그려진다.

## 성공 조건
- [ ] 드래그 시 사각형이 그려진다.
- [ ] 선택한 색상과 두께가 적용된다.

## 실패 / 취소 흐름
- Ctrl+Z → 마지막 주석이 취소된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자유 그리기 동작 (별도 spec).
- 주석 텍스트 동작.
