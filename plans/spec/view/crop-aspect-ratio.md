# Spec: crop-aspect-ratio

## 한 줄 정의
사용자가 자르기 비율을 선택하면 지정된 비율로 자르기 영역이 제한된다.

## 진입점
- 자르기 모드: 비율 드롭다운

## 사용자 흐름
1. 사용자가 자르기 모드에서 비율 드롭다운을 연다.
2. 원하는 비율(예: 16:9, 4:3, 1:1, 자유)을 선택한다.
3. 자르기 영역이 선택한 비율로 제한된다.
4. 드래그하여 위치를 조정한다.

## 성공 조건
- [ ] 비율 선택 시 자르기 영역이 해당 비율로 제한된다.
- [ ] "자유" 선택 시 비율 제한이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
