# Spec: bottom-toolbar-crop

## 한 줄 정의
사용자가 자르기 버튼을 누르면 이미지 자르기 모드가 활성화된다.

## 진입점
- 하단 도구 모음: 자르기 버튼 클릭

## 사용자 흐름
1. 사용자가 자르기 버튼을 클릭한다.
2. 이미지 위에 자르기 영역 선택 도구가 나타난다.
3. 원하는 영역을 드래그하여 선택한다.
4. 확인을 클릭하면 선택한 영역만 남는다.

## 성공 조건
- [ ] 클릭 시 자르기 모드가 활성화된다.
- [ ] 영역 선택 후 잘린 이미지가 표시된다.

## 실패 / 취소 흐름
- 취소 → 원본 이미지가 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 회전 동작 (별도 spec).
- 크기 조정 동작.
