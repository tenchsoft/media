# Spec: window-fullscreen

## 한 줄 정의
사용자가 전체 화면 버튼을 누르면 뷰어가 전체 화면으로 전환된다.

## 진입점
- 하단 도구 모음: 전체 화면 버튼 / F11 / 더블 클릭

## 사용자 흐름
1. 사용자가 전체 화면 버튼을 클릭한다.
2. 뷰어가 전체 화면으로 전환된다.
3. 메뉴 바와 타이틀 바가 숨겨진다.
4. Esc를 누르면 창 모드로 돌아간다.

## 성공 조건
- [ ] 클릭 시 전체 화면으로 전환된다.
- [ ] Esc로 창 모드로 복귀한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 모바일에서는 항상 전체 화면이다.

## 범위 외
- 창 모드 UI 동작.
