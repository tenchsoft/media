# Spec: keyboard-shortcut-ctrl-s

## 한 줄 정의
Ctrl+S를 누르면 변경 사항이 파일에 저장된다.

## 진입점
- 키보드: Ctrl+S

## 사용자 흐름
1. 사용자가 Ctrl+S를 누른다.
2. 변경 사항이 파일에 저장된다.
3. 완료 토스트가 표시된다.

## 성공 조건
- [ ] Ctrl+S로 파일이 저장된다.

## 실패 / 취소 흐름
- 변경 사항이 없으면 → 아무 일도 일어나지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 다른 이름으로 저장 동작 (별도 spec).
