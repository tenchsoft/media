# Spec: slideshow-speed

## 한 줄 정의
사용자가 슬라이드쇼 전환 속도를 설정할 수 있다.

## 진입점
- 슬라이드쇼 설정: 속도 드롭다운

## 사용자 흐름
1. 사용자가 슬라이드쇼 속도 설정을 연다.
2. 전환 간격(예: 1초, 3초, 5초, 10초)을 선택한다.
3. 슬라이드쇼가 선택한 간격으로 전환된다.

## 성공 조건
- [ ] 속도 선택 시 전환 간격이 변경된다.
- [ ] 선택한 간격으로 이미지가 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 슬라이드쇼 시작/종료 동작 (별도 spec).
