# Spec: redo

## 한 줄 정의
사용자가 다시 실행 버튼을 누르면 취소했던 편집 동작이 다시 적용된다.

## 진입점
- 하단 도구 모음: 다시 실행 버튼 / Ctrl+Shift+Z

## 사용자 흐름
1. 사용자가 다시 실행 버튼을 클릭한다.
2. 취소했던 편집 동작이 다시 적용된다.
3. 다음 상태로 복원된다.

## 성공 조건
- [ ] 클릭 시 취소한 편집이 다시 적용된다.
- [ ] 다음 상태로 복원된다.

## 실패 / 취소 흐름
- 다시 실행할 동작이 없으면 → 버튼이 비활성화된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 실행 취소 동작 (별도 spec).
