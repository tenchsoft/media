# Spec: format-convert

## 한 줄 정의
사용자가 다른 이름으로 저장에서 포맷을 변경하면 이미지 포맷이 변환된다.

## 진입점
- 다른 이름으로 저장 대화상자: 포맷 드롭다운

## 사용자 흐름
1. 사용자가 다른 이름으로 저장을 연다.
2. 포맷 드롭다운에서 변환할 포맷(JPEG, PNG, BMP, WebP 등)을 선택한다.
3. 저장 버튼을 누른다.
4. 선택한 포맷으로 이미지가 저장된다.

## 성공 조건
- [ ] 포맷 선택 후 저장 시 해당 포맷으로 변환된다.
- [ ] JPEG 선택 시 품질 설정이 나타난다.

## 실패 / 취소 흐름
- 취소 → 변환되지 않는다.

## 경계 / 예외
- 무손실 → 손실 포맷 변환 시 품질 손실 경고.

## 범위 외
- 일괄 변환 동작.
