# Spec: context-menu-rotate-right

## 한 줄 정의
우클릭 메뉴에서 오른쪽으로 회전을 선택하면 이미지가 시계 방향으로 90도 회전한다.

## 진입점
- 이미지 영역: 우클릭 → "오른쪽으로 회전" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "오른쪽으로 회전"을 선택한다.
4. 이미지가 시계 방향으로 90도 회전한다.

## 성공 조건
- [ ] "오른쪽으로 회전" 선택 시 이미지가 시계 방향으로 90도 회전한다.

## 실패 / 취소 흐름
- 메뉴 외부 클릭 → 메뉴가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 왼쪽 회전 동작 (별도 spec).
