# Spec: bottom-toolbar-delete

## 한 줄 정의
사용자가 삭제 버튼을 누르면 현재 이미지 파일이 휴지통으로 이동한다.

## 진입점
- 하단 도구 모음: 삭제 버튼 클릭

## 사용자 흐름
1. 사용자가 삭제 버튼을 클릭한다.
2. 확인 대화상자가 표시된다.
3. 확인을 클릭하면 파일이 휴지통으로 이동한다.
4. 다음 이미지가 표시된다.

## 성공 조건
- [ ] 확인 시 파일이 휴지통으로 이동한다.
- [ ] 다음 이미지가 표시된다.

## 실패 / 취소 흐름
- 취소 → 파일이 유지된다.

## 경계 / 예외
- 읽기 전용 파일은 삭제할 수 없다.

## 범위 외
- 파일 복구 동작.
