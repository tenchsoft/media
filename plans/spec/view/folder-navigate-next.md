# Spec: folder-navigate-next

## 한 줄 정의
사용자가 다음 이미지 버튼을 누르면 같은 폴더의 다음 이미지가 표시된다.

## 진입점
- 탐색 버튼: 다음 버튼 / 오른쪽 화살표 키

## 사용자 흐름
1. 사용자가 다음 버튼을 클릭한다.
2. 같은 폴더의 다음 이미지가 표시된다.

## 성공 조건
- [ ] 클릭 시 다음 이미지가 표시된다.
- [ ] 오른쪽 화살표 키로도 이동할 수 있다.

## 실패 / 취소 흐름
- 마지막 이미지 → 첫 번째 이미지로 돌아간다.

## 경계 / 예외
- 폴더에 이미지가 1장뿐이면 → 버튼이 비활성화된다.

## 범위 외
- 이전 이미지 동작 (별도 spec).
