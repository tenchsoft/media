# Spec: file-save

## 한 줄 정의
사용자가 저장 버튼을 누르면 변경 사항이 파일에 저장된다.

## 진입점
- 하단 도구 모음: 저장 버튼 / 단축키: Ctrl+S

## 사용자 흐름
1. 사용자가 저장 버튼을 클릭한다.
2. 회전, 자르기 등 변경 사항이 파일에 저장된다.
3. 저장 완료 토스트가 표시된다.

## 성공 조건
- [ ] 클릭 시 변경 사항이 파일에 저장된다.
- [ ] 완료 토스트가 표시된다.

## 실패 / 취소 흐름
- 파일이 잠겨 있으면 → 오류 메시지가 표시된다.

## 경계 / 예외
- 변경 사항이 없으면 저장되지 않는다.

## 범위 외
- 다른 이름으로 저장 동작 (별도 spec).
