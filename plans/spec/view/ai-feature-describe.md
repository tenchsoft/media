# Spec: ai-feature-describe

## 한 줄 정의
사용자가 AI 패널에서 이미지 설명을 선택하면 이미지 내용이 텍스트로 설명된다.

## 진입점
- AI 패널: 이미지 설명 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 패널을 연다.
2. 이미지 설명 버튼을 클릭한다.
3. 실행 버튼을 누른다.
4. AI가 이미지를 분석하여 내용을 텍스트로 설명한다.
5. 설명 결과가 패널에 표시된다.

## 성공 조건
- [ ] 이미지 내용이 텍스트로 설명된다.
- [ ] 설명이 패널에 표시된다.

## 실패 / 취소 흐름
- AI 처리 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- AI 실행 버튼 동작 (별도 spec).
- 설명 복사 동작.
