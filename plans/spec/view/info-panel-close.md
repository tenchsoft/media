# Spec: info-panel-close

## 한 줄 정의
사용자가 정보 패널 닫기 버튼을 누르면 패널이 닫힌다.

## 진입점
- 정보 패널: 닫기 버튼 / 정보 버튼 다시 클릭

## 사용자 흐름
1. 사용자가 정보 패널의 닫기 버튼을 누른다.
2. 정보 패널이 닫힌다.
3. 이미지 영역이 넓어진다.

## 성공 조건
- [ ] 닫기 버튼으로 패널이 닫힌다.
- [ ] 정보 버튼 다시 클릭으로도 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 정보 패널 내용 (별도 spec).
