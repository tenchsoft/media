# Spec: context-menu-copy-path

## 한 줄 정의
우클릭 메뉴에서 경로 복사를 선택하면 파일 경로가 클립보드에 복사된다.

## 진입점
- 이미지 영역: 우클릭 → "경로 복사" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "경로 복사"를 선택한다.
4. 파일의 전체 경로가 클립보드에 복사된다.

## 성공 조건
- [ ] "경로 복사" 선택 시 파일 경로가 클립보드에 복사된다.

## 실패 / 취소 흐름
- 메뉴 외부 클릭 → 메뉴가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 이미지 복사 동작 (별도 spec).
