# Spec: canvas-double-click-zoom-toggle

## 한 줄 정의
이미지를 더블 클릭하면 원본 크기와 화면 맞춤이 전환된다.

## 진입점
- 이미지 영역: 더블 클릭

## 사용자 흐름
1. 사용자가 이미지를 더블 클릭한다.
2. 화면 맞춤 상태면 원본 크기(100%)로 전환된다.
3. 원본 크기 상태면 화면 맞춤으로 전환된다.

## 성공 조건
- [ ] 더블 클릭 시 확대/축소가 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 단일 클릭 동작.
