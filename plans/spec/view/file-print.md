# Spec: file-print

## 한 줄 정의
사용자가 인쇄 버튼을 누르면 시스템 인쇄 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 인쇄 버튼

## 사용자 흐름
1. 사용자가 인쇄 버튼을 클릭한다.
2. 시스템 인쇄 대화상자가 나타난다.
3. 프린터와 설정을 선택한다.
4. 인쇄 버튼을 누른다.

## 성공 조건
- [ ] 클릭 시 인쇄 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 인쇄되지 않는다.

## 경계 / 예외
- 프린터가 없으면 안내 메시지가 표시된다.

## 범위 외
- 인쇄 설정 상세 동작.
