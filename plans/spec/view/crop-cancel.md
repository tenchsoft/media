# Spec: crop-cancel

## 한 줄 정의
사용자가 자르기를 취소하면 원본 이미지가 유지된다.

## 진입점
- 자르기 모드: 취소 버튼 / Esc 키

## 사용자 흐름
1. 사용자가 자르기 모드에서 취소 버튼을 누른다.
2. 자르기 영역이 사라진다.
3. 원본 이미지가 그대로 유지된다.

## 성공 조건
- [ ] 취소 시 자르기 모드가 종료된다.
- [ ] 원본 이미지가 변경 없이 유지된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자르기 적용 동작 (별도 spec).
