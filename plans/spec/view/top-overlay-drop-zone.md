# Spec: top-overlay-drop-zone

## 한 줄 정의
빈 상태에서 파일을 드롭하면 이미지가 열린다.

## 진입점
- 빈 상태 오버레이: 파일 드래그 앤 드롭

## 사용자 흐름
1. 이미지가 열려 있지 않은 빈 상태에서 드롭 영역이 표시된다.
2. 사용자가 파일을 드래그하여 드롭한다.
3. 이미지가 열린다.

## 성공 조건
- [ ] 빈 상태에서 드롭 영역이 표시된다.
- [ ] 파일 드롭 시 이미지가 열린다.

## 실패 / 취소 흐름
- 지원하지 않는 포맷 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 열기 대화상자 동작 (별도 spec).
