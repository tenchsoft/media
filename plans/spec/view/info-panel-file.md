# Spec: info-panel-file

## 한 줄 정의
정보 패널에서 파일 기본 정보를 확인할 수 있다.

## 진입점
- 정보 패널: 파일 탭

## 사용자 흐름
1. 사용자가 정보 패널을 연다.
2. 파일 탭이 기본으로 표시된다.
3. 파일 이름, 크기, 해상도, 포맷, 생성/수정 날짜가 표시된다.

## 성공 조건
- [ ] 파일 탭에서 기본 정보가 표시된다.
- [ ] 이름, 크기, 해상도, 포맷, 날짜가 포함된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- EXIF 데이터 (별도 spec).
