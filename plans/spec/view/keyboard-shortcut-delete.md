# Spec: keyboard-shortcut-delete

## 한 줄 정의
Delete 키를 누르면 현재 파일 삭제 확인 대화상자가 나타난다.

## 진입점
- 키보드: Delete 키

## 사용자 흐름
1. 사용자가 Delete 키를 누른다.
2. 삭제 확인 대화상자가 나타난다.
3. 확인 시 파일이 휴지통으로 이동한다.

## 성공 조건
- [ ] Delete 키로 삭제 확인 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 파일이 유지된다.

## 경계 / 예외
- 텍스트 입력 필드에 포커스가 있으면 동작하지 않는다.

## 범위 외
- 파일 삭제 동작 (별도 spec).
