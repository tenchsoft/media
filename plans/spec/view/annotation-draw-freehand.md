# Spec: annotation-draw-freehand

## 한 줄 정의
사용자가 자유 그리기 도구를 선택하면 이미지 위에 자유롭게 선을 그릴 수 있다.

## 진입점
- 주석 도구 모음: 자유 그리기 도구 클릭

## 사용자 흐름
1. 사용자가 자유 그리기 도구를 선택한다.
2. 이미지 위에서 마우스를 드래그한다.
3. 드래그 경로를 따라 선이 그려진다.
4. 마우스를 놓으면 주석이 확정된다.

## 성공 조건
- [ ] 드래그 시 선이 그려진다.
- [ ] 선택한 색상과 두께가 적용된다.

## 실패 / 취소 흐름
- Ctrl+Z → 마지막 주석이 취소된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 색상 동작 (별도 spec).
- 주석 지우기 동작.
