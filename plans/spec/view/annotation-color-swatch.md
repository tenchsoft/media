# Spec: annotation-color-swatch

## 한 줄 정의
사용자가 주석 색상 견본을 클릭하면 새 주석에 해당 색상이 적용된다.

## 진입점
- 주석 도구 모음: 색상 견본 클릭

## 사용자 흐름
1. 사용자가 주석 도구 모음을 연다.
2. 원하는 색상 견본(검정, 파랑, 빨강, 초록, 노랑 등)을 클릭한다.
3. 선택한 색상이 활성화된다.
4. 이후 그리는 주석에 해당 색상이 적용된다.

## 성공 조건
- [ ] 클릭 시 해당 색상이 선택된다.
- [ ] 이후 주석에 선택한 색상이 적용된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 그리기 동작 (별도 spec).
