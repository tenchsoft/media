# Spec: bottom-toolbar-print

## 한 줄 정의
사용자가 인쇄 버튼을 누르면 인쇄 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 인쇄 버튼 클릭

## 사용자 흐름
1. 사용자가 인쇄 버튼을 클릭한다.
2. 시스템 인쇄 대화상자가 나타난다.
3. 인쇄 설정을 확인하고 인쇄한다.

## 성공 조건
- [ ] 클릭 시 인쇄 대화상자가 표시된다.

## 실패 / 취소 흐름
- 인쇄 취소 → 인쇄되지 않는다.

## 경계 / 예외
- 프린터가 연결되지 않으면 오류가 표시된다.

## 범위 외
- 인쇄 설정 동작.
