# Spec: file-save-as

## 한 줄 정의
사용자가 다른 이름으로 저장 버튼을 누르면 새 파일로 저장할 수 있다.

## 진입점
- 하단 도구 모음: 다른 이름으로 저장 버튼 / 메뉴: 다른 이름으로 저장

## 사용자 흐름
1. 사용자가 다른 이름으로 저장 버튼을 클릭한다.
2. 파일 저장 대화상자가 나타난다.
3. 파일 이름, 위치, 포맷을 선택한다.
4. 저장 버튼을 누른다.
5. 새 파일로 저장된다.

## 성공 조건
- [ ] 클릭 시 파일 저장 대화상자가 나타난다.
- [ ] 포맷을 선택할 수 있다.
- [ ] 새 파일로 저장된다.

## 실패 / 취소 흐름
- 취소 → 저장되지 않는다.
- 동일 이름 → 덮어쓰기 확인이 나타난다.

## 경계 / 예외
- 원본 포맷과 다른 포맷으로 저장 시 품질 손실 경고.

## 범위 외
- 저장 동작 (별도 spec).
