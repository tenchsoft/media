# Spec: context-menu-show-in-files

## 한 줄 정의
우클릭 메뉴에서 파일 위치 열기를 선택하면 시스템 파일 관리자에서 파일이 선택된다.

## 진입점
- 이미지 영역: 우클릭 → "파일 위치 열기" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "파일 위치 열기"를 선택한다.
4. 시스템 파일 관리자가 열린다.
5. 해당 파일이 선택된 상태로 표시된다.

## 성공 조건
- [ ] "파일 위치 열기" 선택 시 파일 관리자가 열린다.
- [ ] 파일이 선택된 상태로 표시된다.

## 실패 / 취소 흐름
- 파일이 이동/삭제된 경우 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 관리자 자체 동작.
