# Spec: drag-and-drop-open

## 한 줄 정의
사용자가 이미지 파일을 뷰어 창에 드롭하면 파일이 열린다.

## 진입점
- 뷰어 창: 파일 드래그 앤 드롭

## 사용자 흐름
1. 사용자가 파일 관리자에서 이미지 파일을 드래그한다.
2. 뷰어 창 위에 드롭한다.
3. 이미지가 열린다.

## 성공 조건
- [ ] 이미지 파일 드롭 시 파일이 열린다.
- [ ] 지원하지 않는 포맷은 오류 메시지가 표시된다.

## 실패 / 취소 흐름
- 지원하지 않는 포맷 → 오류 메시지가 표시된다.

## 경계 / 예외
- 여러 파일 드롭 시 첫 번째 파일이 열린다.

## 범위 외
- 파일 열기 대화상자 동작 (별도 spec).
