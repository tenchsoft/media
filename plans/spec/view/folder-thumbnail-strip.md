# Spec: folder-thumbnail-strip

## 한 줄 정의
폴더 내 이미지의 썸네일이 하단 스트립에 표시된다.

## 진입점
- 자동: 이미지 열기 시

## 사용자 흐름
1. 이미지를 열면 하단에 썸네일 스트립이 표시된다.
2. 썸네일을 클릭하면 해당 이미지로 전환된다.
3. 현재 이미지의 썸네일이 하이라이트된다.

## 성공 조건
- [ ] 폴더 내 이미지의 썸네일이 표시된다.
- [ ] 클릭 시 해당 이미지로 전환된다.
- [ ] 현재 이미지가 하이라이트된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 폴더에 이미지가 많으면 스크롤이 필요하다.

## 범위 외
- 썸네일 크기 설정 동작.
