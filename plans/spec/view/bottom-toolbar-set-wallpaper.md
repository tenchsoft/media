# Spec: bottom-toolbar-set-wallpaper

## 한 줄 정의
사용자가 배경화면 설정 버튼을 누르면 현재 이미지가 바탕화면으로 설정된다.

## 진입점
- 하단 도구 모음: 배경화면 설정 버튼 클릭

## 사용자 흐름
1. 사용자가 배경화면 설정 버튼을 클릭한다.
2. 현재 이미지가 시스템 바탕화면으로 설정된다.
3. 완료 알림이 표시된다.

## 성공 조건
- [ ] 클릭 시 이미지가 바탕화면으로 설정된다.
- [ ] 완료 알림이 표시된다.

## 실패 / 취소 흐름
- 설정 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 바탕화면 설정 옵션 동작.
