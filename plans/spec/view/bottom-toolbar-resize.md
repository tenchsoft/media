# Spec: bottom-toolbar-resize

## 한 줄 정의
사용자가 크기 조정 버튼을 누르면 이미지 크기 변경 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 크기 조정 버튼 클릭

## 사용자 흐름
1. 사용자가 크기 조정 버튼을 클릭한다.
2. 크기 조정 대화상자가 나타난다.
3. 새 너비/높이를 입력한다.
4. 비율 유지 옵션을 선택할 수 있다.
5. 확인을 클릭하면 이미지 크기가 변경된다.

## 성공 조건
- [ ] 클릭 시 크기 조정 대화상자가 표시된다.
- [ ] 확인 시 이미지 크기가 변경된다.

## 실패 / 취소 흐름
- 취소 → 원본 크기가 유지된다.

## 경계 / 예외
- 비율 유지 시 한쪽을 변경하면 다른 쪽이 자동 조정된다.

## 범위 외
- 자르기 동작 (별도 spec).
