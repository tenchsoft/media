# Spec: settings-image

## 한 줄 정의
설정 대화상자에서 이미지 표시 관련 설정을 변경할 수 있다.

## 진입점
- 설정 대화상자: 이미지 탭

## 사용자 흐름
1. 사용자가 설정을 연다.
2. 이미지 탭을 선택한다.
3. 배경색, 보간 방식, 자동 회전(EXIF 기반) 등을 변경한다.
4. 확인 버튼을 누른다.
5. 설정이 저장된다.

## 성공 조건
- [ ] 이미지 설정 항목이 표시된다.
- [ ] 변경한 설정이 즉시 반영된다.

## 실패 / 취소 흐름
- 취소 → 설정이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 일반 설정 동작 (별도 spec).
