# Spec: context-menu-delete-file

## 한 줄 정의
우클릭 메뉴에서 삭제를 선택하면 파일이 휴지동으로 이동한다.

## 진입점
- 이미지 영역: 우클릭 → "삭제" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "삭제"를 선택한다.
4. 확인 대화상자가 나타난다.
5. 확인 시 파일이 휴지통으로 이동한다.

## 성공 조건
- [ ] "삭제" 선택 시 확인 대화상자가 나타난다.
- [ ] 확인 시 파일이 휴지통으로 이동한다.

## 실패 / 취소 흐름
- 취소 → 파일이 유지된다.

## 경계 / 예외
- 파일이 잠겨 있으면 삭제할 수 없다.

## 범위 외
- 영구 삭제 동작.
