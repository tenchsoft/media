# Spec: keyboard-shortcut-ctrl-shift-s

## 한 줄 정의
Ctrl+Shift+S를 누르면 다른 이름으로 저장 대화상자가 나타난다.

## 진입점
- 키보드: Ctrl+Shift+S

## 사용자 흐름
1. 사용자가 Ctrl+Shift+S를 누른다.
2. 다른 이름으로 저장 대화상자가 나타난다.

## 성공 조건
- [ ] Ctrl+Shift+S로 저장 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 저장되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 저장 동작 (별도 spec).
