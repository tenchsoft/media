# Spec: settings-file-association

## 한 줄 정의
설정 대화상자에서 파일 확장자 연결을 관리할 수 있다.

## 진입점
- 설정 대화상자: 파일 연결 탭

## 사용자 흐름
1. 사용자가 설정을 연다.
2. 파일 연결 탭을 선택한다.
3. 지원 포맷(JPG, PNG, GIF, BMP, WebP, SVG 등)의 연결을 켜거나 끈다.
4. 확인 버튼을 누른다.
5. 연결이 시스템에 등록된다.

## 성공 조건
- [ ] 파일 연결 설정이 표시된다.
- [ ] 토글로 연결을 켜거나 끌 수 있다.

## 실패 / 취소 흐름
- 취소 → 설정이 유지된다.

## 경계 / 예외
- 일부 플랫폼에서는 시스템 설정이 필요하다.

## 범위 외
- 시스템 파일 연결 UI.
