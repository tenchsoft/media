# Spec: annotation-draw-arrow

## 한 줄 정의
사용자가 화살표 도구를 선택하면 이미지 위에 화살표를 그릴 수 있다.

## 진입점
- 주석 도구 모음: 화살표 도구 클릭

## 사용자 흐름
1. 사용자가 화살표 도구를 선택한다.
2. 이미지 위에서 시작점을 클릭하고 끝점으로 드래그한다.
3. 끝점에 화살촉이 있는 선이 그려진다.

## 성공 조건
- [ ] 드래그 시 화살표가 그려진다.
- [ ] 끝점에 화살촉이 표시된다.

## 실패 / 취소 흐름
- Ctrl+Z → 마지막 주석이 취소된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자유 그리기 동작 (별도 spec).
- 사각형 도구 동작.
