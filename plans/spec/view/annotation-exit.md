# Spec: annotation-exit

## 한 줄 정의
주석 모드에서 나가면 저장 여부를 확인하고 주석 모드가 종료된다.

## 진입점
- 주석 모드: 나가기 버튼 / Esc

## 사용자 흐름
1. 사용자가 주석 모드에서 나가기 버튼을 누른다.
2. 저장하지 않은 주석이 있으면 저장 확인 대화상자가 나타난다.
3. 저장/버리기를 선택한다.
4. 주석 모드가 종료된다.

## 성공 조건
- [ ] 나가기 시 저장 확인이 나타난다.
- [ ] 주석 모드가 종료된다.

## 실패 / 취소 흐름
- 취소 → 주석 모드가 유지된다.

## 경계 / 예외
- 주석이 없으면 확인 없이 종료된다.

## 범위 외
- 주석 저장 동작 (별도 spec).
