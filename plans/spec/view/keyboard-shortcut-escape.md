# Spec: keyboard-shortcut-escape

## 한 줄 정의
Esc 키를 누르면 전체 화면이 해제되거나 현재 모달이 닫힌다.

## 진입점
- 키보드: Esc

## 사용자 흐름
1. 전체 화면 상태에서 Esc를 누르면 창 모드로 전환된다.
2. 모달이 열려 있으면 모달이 닫힌다.
3. 자르기/편집 모드이면 편집이 취소된다.

## 성공 조건
- [ ] 전체 화면에서 Esc 시 창 모드로 전환된다.
- [ ] 모달이 닫힌다.
- [ ] 편집 모드가 취소된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
