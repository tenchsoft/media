# Spec: ai-run

## 한 줄 정의
사용자가 AI 실행 버튼을 누르면 선택한 AI 기능이 실행된다.

## 진입점
- AI 패널: 실행 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 패널에서 원하는 기능을 선택한다.
2. 실행 버튼을 클릭한다.
3. AI 작업이 시작된다.
4. 진행 상태가 표시된다.
5. 완료되면 결과가 표시된다.

## 성공 조건
- [ ] 클릭 시 선택한 AI 기능이 실행된다.
- [ ] 진행 상태가 표시된다.
- [ ] 완료 시 결과가 표시된다.

## 실패 / 취소 흐름
- AI 처리 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 기능이 선택되지 않으면 실행 버튼이 비활성화된다.

## 범위 외
- 개별 AI 기능 동작 (별도 spec).
