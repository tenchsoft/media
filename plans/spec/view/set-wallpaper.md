# Spec: set-wallpaper

## 한 줄 정의
사용자가 배경화면 설정 버튼을 누르면 현재 이미지가 바탕화면으로 설정된다.

## 진입점
- 하단 도구 모음: 배경화면 설정 버튼

## 사용자 흐름
1. 사용자가 배경화면 설정 버튼을 클릭한다.
2. 이미지가 바탕화면 배경으로 설정된다.
3. 완료 토스트가 표시된다.

## 성공 조건
- [ ] 클릭 시 바탕화면이 현재 이미지로 변경된다.
- [ ] 완료 토스트가 표시된다.

## 실패 / 취소 흐름
- 지원하지 않는 플랫폼 → 안내 메시지가 표시된다.

## 경계 / 예외
- 모바일에서는 지원되지 않을 수 있다.

## 범위 외
- 배경화면 설정 UI (시스템 설정).
