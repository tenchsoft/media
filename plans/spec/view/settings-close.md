# Spec: settings-close

## 한 줄 정의
설정 대화상자 닫기 버튼을 누르면 설정이 닫힌다.

## 진입점
- 설정 대화상자: 닫기 버튼 / Esc

## 사용자 흐름
1. 사용자가 설정 대화상자의 닫기 버튼을 누른다.
2. 설정 대화상자가 닫힌다.

## 성공 조건
- [ ] 닫기 버튼으로 설정이 닫힌다.
- [ ] Esc로도 닫을 수 있다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 저장하지 않은 변경 사항이 있으면 확인 대화상자가 나타난다.

## 범위 외
- 설정 저장 동작 (별도 spec).
