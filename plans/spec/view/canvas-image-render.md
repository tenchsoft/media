# Spec: canvas-image-render

## 한 줄 정의
이미지 파일을 열면 캔버스에 자동으로 렌더링된다.

## 진입점
- 자동: 이미지 파일 열기 시

## 사용자 흐름
1. 사용자가 이미지 파일을 연다.
2. 이미지가 캔버스에 렌더링된다.
3. 화면에 맞춰 자동 조정된다.

## 성공 조건
- [ ] 파일 열기 시 이미지가 즉시 표시된다.
- [ ] 화면에 맞게 자동 조정된다.

## 실패 / 취소 흐름
- 손상된 파일 → 오류 메시지가 표시된다.

## 경계 / 예외
- 대용량 이미지는 로딩 시간이 걸릴 수 있다.

## 범위 외
- 확대/축소 동작 (별도 spec).
