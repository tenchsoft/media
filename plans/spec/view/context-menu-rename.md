# Spec: context-menu-rename

## 한 줄 정의
우클릭 메뉴에서 이름 바꾸기를 선택하면 파일 이름을 편집할 수 있다.

## 진입점
- 이미지 영역: 우클릭 → "이름 바꾸기" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "이름 바꾸기"를 선택한다.
4. 이름 편집 대화상자가 나타난다.
5. 새 이름을 입력하고 확인한다.
6. 파일 이름이 변경된다.

## 성공 조건
- [ ] "이름 바꾸기" 선택 시 이름 편집 대화상자가 나타난다.
- [ ] 확인 시 파일 이름이 변경된다.

## 실패 / 취소 흐름
- 취소 → 파일 이름이 유지된다.
- 중복 이름 → 경고 메시지가 표시된다.

## 경계 / 예외
- 확장자를 변경하면 경고가 표시된다.

## 범위 외
- 일괄 이름 변경 동작.
