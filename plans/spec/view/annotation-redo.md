# Spec: annotation-redo

## 한 줄 정의
주석 모드에서 다시 실행하면 취소했던 주석 동작이 다시 적용된다.

## 진입점
- 주석 모드: 다시 실행 버튼 / Ctrl+Shift+Z

## 사용자 흐름
1. 사용자가 주석 모드에서 다시 실행 버튼을 누른다.
2. 취소했던 주석 동작이 다시 적용된다.

## 성공 조건
- [ ] 취소한 주석 동작이 다시 적용된다.

## 실패 / 취소 흐름
- 다시 실행할 동작이 없으면 → 버튼이 비활성화된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 실행 취소 동작 (별도 spec).
