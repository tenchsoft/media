# Spec: ai-feature-tag

## 한 줄 정의
사용자가 AI 패널에서 자동 태깅을 선택하면 이미지에 관련 태그가 자동으로 생성된다.

## 진입점
- AI 패널: 자동 태깅 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 패널을 연다.
2. 자동 태깅 버튼을 클릭한다.
3. 실행 버튼을 누른다.
4. AI가 이미지를 분석하여 관련 태그를 생성한다.
5. 태그 목록이 패널에 표시된다.

## 성공 조건
- [ ] 이미지 관련 태그가 자동 생성된다.
- [ ] 태그 목록이 패널에 표시된다.

## 실패 / 취소 흐름
- AI 처리 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 태그 편집 동작 (별도 spec).
- AI 실행 버튼 동작.
