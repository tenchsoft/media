# Spec: keyboard-shortcut-left-arrow

## 한 줄 정의
왼쪽 화살표 키를 누르면 이전 이미지가 표시된다.

## 진입점
- 키보드: 왼쪽 화살표 키

## 사용자 흐름
1. 사용자가 왼쪽 화살표 키를 누른다.
2. 이전 이미지가 표시된다.

## 성공 조건
- [ ] 왼쪽 화살표 키로 이전 이미지로 이동한다.

## 실패 / 취소 흐름
- 첫 번째 이미지 → 마지막 이미지로 이동한다.

## 경계 / 예외
- 텍스트 입력 필드에 포커스가 있으면 동작하지 않는다.

## 범위 외
- 다음 이미지 동작 (별도 spec).
