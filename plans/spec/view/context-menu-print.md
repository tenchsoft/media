# Spec: context-menu-print

## 한 줄 정의
우클릭 메뉴에서 인쇄를 선택하면 시스템 인쇄 대화상자가 나타난다.

## 진입점
- 이미지 영역: 우클릭 → "인쇄" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "인쇄"를 선택한다.
4. 시스템 인쇄 대화상자가 나타난다.

## 성공 조건
- [ ] "인쇄" 선택 시 인쇄 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 인쇄되지 않는다.

## 경계 / 예외
- 프린터가 없으면 안내 메시지가 표시된다.

## 범위 외
- 인쇄 설정 동작.
