# Spec: canvas-pan

## 한 줄 정의
사용자가 확대된 이미지를 드래그하면 이미지가 이동한다.

## 진입점
- 이미지 영역: 마우스 드래그 (확대 상태)

## 사용자 흐름
1. 이미지가 확대된 상태에서 마우스를 드래그한다.
2. 이미지가 드래그 방향으로 이동한다.
3. 숨겨진 영역을 볼 수 있다.

## 성공 조건
- [ ] 드래그 시 이미지가 이동한다.
- [ ] 숨겨진 영역이 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 화면에 맞춤 상태에서는 이동하지 않는다.

## 범위 외
- 확대/축소 동작 (별도 spec).
