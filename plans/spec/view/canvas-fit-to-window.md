# Spec: canvas-fit-to-window

## 한 줄 정의
사용자가 화면에 맞추기 버튼을 누르면 이미지가 창 크기에 맞춰 조정된다.

## 진입점
- 하단 도구 모음: 화면에 맞추기 버튼 클릭

## 사용자 흐름
1. 사용자가 화면에 맞추기 버튼을 클릭한다.
2. 이미지가 창 크기에 맞게 축소/확대된다.
3. 이미지 전체가 화면에 표시된다.

## 성공 조건
- [ ] 클릭 시 이미지가 창 크기에 맞춰진다.
- [ ] 이미지 전체가 화면에 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 원본 크기 동작 (별도 spec).
- 확대/축소 동작.
