# Spec: window-title-bar

## 한 줄 정의
타이틀 바에 현재 파일 이름과 뷰어 이름이 표시된다.

## 진입점
- 자동: 파일 열기 시

## 사용자 흐름
1. 이미지를 열면 타이틀 바에 파일 이름이 표시된다.
2. "파일이름 - View" 형식으로 표시된다.
3. 변경 사항이 있으면 이름 앞에 표시가 나타난다.

## 성공 조건
- [ ] 파일 이름이 타이틀 바에 표시된다.
- [ ] 변경 사항 표시가 나타난다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 파일이 없으면 "View"만 표시된다.

## 범위 외
- 타이틀 바 버튼 동작.
