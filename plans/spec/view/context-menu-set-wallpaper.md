# Spec: context-menu-set-wallpaper

## 한 줄 정의
우클릭 메뉴에서 배경화면으로 설정을 선택하면 현재 이미지가 바탕화면 배경으로 설정된다.

## 진입점
- 이미지 영역: 우클릭 → "배경화면으로 설정" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "배경화면으로 설정"을 선택한다.
4. 이미지가 바탕화면 배경으로 설정된다.
5. 완료 토스트가 표시된다.

## 성공 조건
- [ ] "배경화면으로 설정" 선택 시 바탕화면이 변경된다.
- [ ] 완료 토스트가 표시된다.

## 실패 / 취소 흐름
- 지원하지 않는 플랫폼 → 안내 메시지가 표시된다.

## 경계 / 예외
- 모바일에서는 지원되지 않을 수 있다.

## 범위 외
- 배경화면 설정 UI (시스템 설정).
