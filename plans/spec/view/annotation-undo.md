# Spec: annotation-undo

## 한 줄 정의
주석 모드에서 실행 취소하면 마지막 주석 동작이 취소된다.

## 진입점
- 주석 모드: 실행 취소 버튼 / Ctrl+Z

## 사용자 흐름
1. 사용자가 주석 모드에서 실행 취소 버튼을 누른다.
2. 마지막 주석 동작이 취소된다.
3. 이전 상태로 복원된다.

## 성공 조건
- [ ] 마지막 주석 동작이 취소된다.
- [ ] 이전 상태로 복원된다.

## 실패 / 취소 흐름
- 취소할 동작이 없으면 → 버튼이 비활성화된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 저장 동작 (별도 spec).
