# Spec: file-copy

## 한 줄 정의
사용자가 복사 버튼을 누르면 이미지가 클립보드에 복사된다.

## 진입점
- 하단 도구 모음: 복사 버튼 / 단축키: Ctrl+C

## 사용자 흐름
1. 사용자가 복사 버튼을 클릭한다.
2. 현재 이미지가 클립보드에 복사된다.
3. 다른 프로그램에 붙여넣을 수 있다.

## 성공 조건
- [ ] 클릭 시 이미지가 클립보드에 복사된다.
- [ ] 다른 프로그램에 붙여넣기가 가능하다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 경로 복사 동작 (별도 spec).
