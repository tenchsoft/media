# Spec: slideshow-transition

## 한 줄 정의
슬라이드쇼에서 이미지 전환 시 전환 효과가 적용된다.

## 진입점
- 슬라이드쇼 설정: 전환 효과 드롭다운

## 사용자 흐름
1. 사용자가 슬라이드쇼 전환 효과 설정을 연다.
2. 전환 효과(예: 페이드, 슬라이드, 없음)를 선택한다.
3. 슬라이드쇼에서 선택한 효과로 이미지가 전환된다.

## 성공 조건
- [ ] 전환 효과 선택 시 슬라이드쇼에 반영된다.
- [ ] "없음" 선택 시 즉시 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 슬라이드쇼 시작/종료 동작 (별도 spec).
