# Spec: folder-recent-files

## 한 줄 정의
최근 연 파일 목록이 표시되어 빠르게 다시 열 수 있다.

## 진입점
- 메뉴: 최근 파일

## 사용자 흐름
1. 사용자가 최근 파일 메뉴를 연다.
2. 최근 열어본 파일 목록이 표시된다.
3. 파일을 클릭하면 해당 이미지가 열린다.

## 성공 조건
- [ ] 최근 파일 목록이 표시된다.
- [ ] 클릭 시 파일이 열린다.

## 실패 / 취소 흐름
- 파일이 이동/삭제된 경우 → 오류 메시지가 표시된다.

## 경계 / 예외
- 목록이 비어 있으면 안내 메시지가 표시된다.

## 범위 외
- 최근 파일 기록 관리 동작.
