# Spec: slideshow-loop

## 한 줄 정의
슬라이드쇼에서 마지막 이미지 후 처음으로 돌아간다.

## 진입점
- 슬라이드쇼 설정: 반복 토글

## 사용자 흐름
1. 사용자가 슬라이드쇼 반복 설정을 켠다.
2. 슬라이드쇼가 마지막 이미지에서 처음으로 돌아간다.
3. 반복을 끄면 마지막 이미지에서 슬라이드쇼가 종료된다.

## 성공 조건
- [ ] 반복 켜짐 시 마지막 이미지 후 처음으로 돌아간다.
- [ ] 반전 꺼짐 시 마지막 이미지에서 종료된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 슬라이드쇼 시작/종료 동작 (별도 spec).
