# Spec: annotation-clear-all

## 한 줄 정의
사용자가 전체 지우기 버튼을 누르면 모든 주석이 제거된다.

## 진입점
- 주석 도구 모음: 전체 지우기 버튼 클릭

## 사용자 흐름
1. 사용자가 전체 지우기 버튼을 클릭한다.
2. 확인 대화상자가 표시된다.
3. 확인을 클릭하면 모든 주석이 제거된다.

## 성공 조건
- [ ] 확인 시 모든 주석이 제거된다.

## 실패 / 취소 흐름
- 취소 → 주석이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 주석 지우기 동작 (별도 spec).
