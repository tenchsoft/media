# Spec: top-bar-menu

## 한 줄 정의
상단 메뉴 바에서 파일, 편집, 보기 등의 메뉴에 접근할 수 있다.

## 진입점
- 상단 바: 메뉴 항목 클릭

## 사용자 흐름
1. 사용자가 메뉴 항목(파일, 편집, 보기 등)을 클릭한다.
2. 드롭다운 메뉴가 나타난다.
3. 원하는 항목을 선택한다.

## 성공 조건
- [ ] 메뉴 클릭 시 드롭다운이 나타난다.
- [ ] 항목 선택 시 해당 동작이 실행된다.

## 실패 / 취소 흐름
- 메뉴 외부 클릭 → 메뉴가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 메뉴 항목 동작 (별도 spec).
