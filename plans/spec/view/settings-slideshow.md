# Spec: settings-slideshow

## 한 줄 정의
설정 대화상자에서 슬라이드쇼 관련 설정을 변경할 수 있다.

## 진입점
- 설정 대화상자: 슬라이드쇼 탭

## 사용자 흐름
1. 사용자가 설정을 연다.
2. 슬라이드쇼 탭을 선택한다.
3. 전환 시간, 전환 효과, 반복 여부 등을 변경한다.
4. 확인 버튼을 누른다.
5. 설정이 저장된다.

## 성공 조건
- [ ] 슬라이드쇼 설정 항목이 표시된다.
- [ ] 변경한 설정이 슬라이드쇼에 반영된다.

## 실패 / 취소 흐름
- 취소 → 설정이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 슬라이드쇼 시작/종료 동작 (별도 spec).
