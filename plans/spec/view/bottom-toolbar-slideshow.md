# Spec: bottom-toolbar-slideshow

## 한 줄 정의
사용자가 슬라이드쇼 버튼을 누르면 이미지 슬라이드쇼가 시작된다.

## 진입점
- 하단 도구 모음: 슬라이드쇼 버튼 클릭

## 사용자 흐름
1. 사용자가 슬라이드쇼 버튼을 클릭한다.
2. 전체 화면 슬라이드쇼가 시작된다.
3. 설정된 시간 간격으로 이미지가 자동 전환된다.
4. Esc 키로 슬라이드쇼를 종료한다.

## 성공 조건
- [ ] 클릭 시 전체 화면 슬라이드쇼가 시작된다.
- [ ] 이미지가 자동 전환된다.
- [ ] Esc 키로 종료된다.

## 실패 / 취소 흐름
- 이미지가 1장뿐이면 전환이 발생하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 슬라이드쇼 속도 설정 동작 (별도 spec).
