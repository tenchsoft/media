# Spec: canvas-checkerboard-bg

## 한 줄 정의
투명한 이미지에서 체크무늬 배경이 자동으로 표시된다.

## 진입점
- 자동: 투명 이미지 열기 시

## 사용자 흐름
1. 사용자가 투명 영역이 있는 이미지(PNG 등)를 연다.
2. 투명 영역에 체크무늬 배경이 표시된다.
3. 투명과 불투명 영역을 구분할 수 있다.

## 성공 조건
- [ ] 투명 이미지에서 체크무늬 배경이 표시된다.
- [ ] 투명/불투명 영역이 구분된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 불투명 이미지에서는 체크무늬가 표시되지 않는다.

## 범위 외
- 배경색 설정 동작.
