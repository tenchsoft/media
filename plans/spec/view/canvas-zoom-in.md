# Spec: canvas-zoom-in

## 한 줄 정의
사용자가 확대 버튼을 누르면 이미지가 확대된다.

## 진입점
- 하단 도구 모음: 확대 버튼 클릭 / 마우스 휠 위로

## 사용자 흐름
1. 사용자가 확대 버튼을 클릭한다.
2. 이미지가 확대된다.
3. 마우스 휠로도 확대할 수 있다.

## 성공 조건
- [ ] 클릭 시 이미지가 확대된다.
- [ ] 마우스 휠로 확대할 수 있다.

## 실패 / 취소 흐름
- 최대 확대 도달 → 더 이상 확대되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 축소 동작 (별도 spec).
- 원본 크기 동작.
