# Spec: folder-sort-by-name

## 한 줄 정의
폴더 내 이미지를 파일 이름순으로 정렬한다.

## 진입점
- 정렬 메뉴: 이름순 정렬

## 사용자 흐름
1. 사용자가 정렬 메뉴를 연다.
2. "이름순"을 선택한다.
3. 이미지가 파일 이름 기준으로 정렬된다.
4. 이전/다음 탐색이 정렬 순서를 따른다.

## 성공 조건
- [ ] 선택 시 이미지가 이름순으로 정렬된다.
- [ ] 탐색이 정렬 순서를 따른다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 날짜순, 크기순 정렬 (별도 spec).
