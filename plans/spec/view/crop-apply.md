# Spec: crop-apply

## 한 줄 정의
사용자가 자르기 영역을 지정하고 적용하면 이미지가 해당 영역으로 잘린다.

## 진입점
- 하단 도구 모음: 자르기 버튼 → 영역 드래그 → 적용

## 사용자 흐름
1. 사용자가 자르기 버튼을 클릭한다.
2. 이미지 위에 자르기 영역 핸들이 나타난다.
3. 핸들을 드래그하여 영역을 조정한다.
4. 적용 버튼을 누른다.
5. 이미지가 선택한 영역으로 잘린다.

## 성공 조건
- [ ] 자르기 모드에서 영역 핸들이 표시된다.
- [ ] 적용 시 이미지가 잘린다.

## 실패 / 취소 흐름
- 취소 → 원본 이미지가 유지된다.

## 경계 / 예외
- 영역이 너무 작으면 경고가 표시된다.

## 범위 외
- 비율 고정 자르기 동작.
