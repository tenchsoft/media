# Spec: resize-dialog

## 한 줄 정의
사용자가 크기 조정 버튼을 누르면 이미지 크기를 변경할 수 있는 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 크기 조정 버튼

## 사용자 흐름
1. 사용자가 크기 조정 버튼을 클릭한다.
2. 크기 조정 대화상자가 나타난다.
3. 너비, 높이를 입력한다.
4. 비율 유지 여부를 선택한다.
5. 확인 버튼을 누른다.
6. 이미지 크기가 변경된다.

## 성공 조건
- [ ] 클릭 시 크기 조정 대화상자가 나타난다.
- [ ] 너비/높이 입력 시 이미지 크기가 변경된다.
- [ ] 비율 유지 옵션이 동작한다.

## 실패 / 취소 흐름
- 취소 → 원본 크기가 유지된다.

## 경계 / 예외
- 0 또는 음수 값은 입력할 수 없다.

## 범위 외
- 자르기 동작 (별도 spec).
