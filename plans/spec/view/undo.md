# Spec: undo

## 한 줄 정의
사용자가 실행 취소 버튼을 누르면 마지막 편집 동작이 취소된다.

## 진입점
- 하단 도구 모음: 실행 취소 버튼 / Ctrl+Z

## 사용자 흐름
1. 사용자가 실행 취소 버튼을 클릭한다.
2. 마지막 편집 동작(회전, 자르기, 반전 등)이 취소된다.
3. 이전 상태로 복원된다.

## 성공 조건
- [ ] 클릭 시 마지막 편집이 취소된다.
- [ ] 이전 상태로 복원된다.

## 실패 / 취소 흐름
- 실행 취소할 동작이 없으면 → 버튼이 비활성화된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 다시 실행 동작 (별도 spec).
