# Spec: bottom-toolbar-share

## 한 줄 정의
사용자가 공유 버튼을 누르면 시스템 공유 시트가 나타난다.

## 진입점
- 하단 도구 모음: 공유 버튼 클릭

## 사용자 흐름
1. 사용자가 공유 버튼을 클릭한다.
2. 시스템 공유 시트가 나타난다.
3. 공유 대상을 선택하여 이미지를 공유한다.

## 성공 조건
- [ ] 클릭 시 시스템 공유 시트가 표시된다.

## 실패 / 취소 흐름
- 공유 취소 → 공유되지 않는다.

## 경계 / 예외
- 일부 플랫폼에서는 공유 기능을 사용할 수 없다.

## 범위 외
- 복사 동작 (별도 spec).
