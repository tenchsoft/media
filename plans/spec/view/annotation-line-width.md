# Spec: annotation-line-width

## 한 줄 정의
사용자가 주석 선 두께를 변경하면 이후 주석에 해당 두께가 적용된다.

## 진입점
- 주석 도구 모음: 선 두께 버튼 클릭

## 사용자 흐름
1. 사용자가 주석 도구 모음을 연다.
2. 원하는 선 두께(얇음, 보통, 두꺼움)를 선택한다.
3. 이후 그리는 주석에 해당 두께가 적용된다.

## 성공 조건
- [ ] 선택 시 선 두께가 변경된다.
- [ ] 이후 주석에 적용된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 그리기 동작 (별도 spec).
- 주석 색상 동작.
