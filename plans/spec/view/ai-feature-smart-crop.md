# Spec: ai-feature-smart-crop

## 한 줄 정의
사용자가 AI 패널에서 스마트 자르기를 선택하면 주요 피사체 중심으로 이미지가 자동으로 잘린다.

## 진입점
- AI 패널: 스마트 자르기 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 패널을 연다.
2. 스마트 자르기 버튼을 클릭한다.
3. 실행 버튼을 누른다.
4. AI가 주요 피사체를 감지하여 자동으로 자른다.
5. 잘린 이미지가 표시된다.

## 성공 조건
- [ ] 주요 피사체 중심으로 이미지가 잘린다.
- [ ] 결과 이미지가 표시된다.

## 실패 / 취소 흐름
- AI 처리 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 수동 자르기 동작 (별도 spec).
- AI 실행 버튼 동작.
