# Spec: canvas-zoom-out

## 한 줄 정의
사용자가 축소 버튼을 누르면 이미지가 축소된다.

## 진입점
- 하단 도구 모음: 축소 버튼 클릭 / 마우스 휠 아래로

## 사용자 흐름
1. 사용자가 축소 버튼을 클릭한다.
2. 이미지가 축소된다.
3. 마우스 휠로도 축소할 수 있다.

## 성공 조건
- [ ] 클릭 시 이미지가 축소된다.
- [ ] 마우스 휠로 축소할 수 있다.

## 실패 / 취소 흐름
- 최소 축소 도달 → 더 이상 축소되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 확대 동작 (별도 spec).
- 화면에 맞추기 동작.
