# Spec: flip-vertical

## 한 줄 정의
사용자가 상하 반전 버튼을 누르면 이미지가 수직으로 뒤집힌다.

## 진입점
- 하단 도구 모음: 상하 반전 버튼

## 사용자 흐름
1. 사용자가 상하 반전 버튼을 클릭한다.
2. 이미지가 상하로 반전된다.
3. 화면에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 이미지가 수직 반전된다.
- [ ] 즉시 화면에 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 좌우 반전 동작 (별도 spec).
- 회전 동작.
