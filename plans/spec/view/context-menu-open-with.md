# Spec: context-menu-open-with

## 한 줄 정의
우클릭 메뉴에서 다른 프로그램으로 열기를 선택하면 시스템 파일 열기 대화상자가 나타난다.

## 진입점
- 이미지 영역: 우클릭 → "다른 프로그램으로 열기" 메뉴 항목

## 사용자 흐름
1. 사용자가 이미지 위에서 우클릭한다.
2. 컨텍스트 메뉴가 나타난다.
3. "다른 프로그램으로 열기"를 선택한다.
4. 시스템의 프로그램 선택 대화상자가 나타난다.
5. 프로그램을 선택하여 파일을 연다.

## 성공 조건
- [ ] "다른 프로그램으로 열기" 선택 시 시스템 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 아무 일도 일어나지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 연결 설정 동작.
