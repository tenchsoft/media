# Spec: keyboard-shortcut-f2

## 한 줄 정의
F2 키를 누르면 파일 이름 바꾸기 대화상자가 나타난다.

## 진입점
- 키보드: F2 키

## 사용자 흐름
1. 사용자가 F2 키를 누른다.
2. 이름 바꾸기 대화상자가 나타난다.
3. 새 이름을 입력하고 확인한다.
4. 파일 이름이 변경된다.

## 성공 조건
- [ ] F2 키로 이름 바꾸기 대화상자가 나타난다.

## 실패 / 취소 흐름
- 취소 → 파일 이름이 유지된다.

## 경계 / 예외
- 텍스트 입력 필드에 포커스가 있으면 동작하지 않는다.

## 범위 외
- 이름 바꾸기 동작 (별도 spec).
