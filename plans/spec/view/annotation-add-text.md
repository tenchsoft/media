# Spec: annotation-add-text

## 한 줄 정의
사용자가 텍스트 도구를 선택하면 이미지 위에 텍스트를 추가할 수 있다.

## 진입점
- 주석 도구 모음: 텍스트 도구 클릭

## 사용자 흐름
1. 사용자가 텍스트 도구를 선택한다.
2. 이미지 위 원하는 위치를 클릭한다.
3. 텍스트 입력 필드가 나타난다.
4. 텍스트를 입력하고 확인한다.
5. 텍스트가 이미지 위에 표시된다.

## 성공 조건
- [ ] 클릭 위치에 텍스트가 표시된다.
- [ ] 선택한 색상이 적용된다.

## 실패 / 취소 흐름
- 빈 텍스트 → 텍스트가 추가되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 주석 그리기 동작 (별도 spec).
