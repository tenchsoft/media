# Spec: bottom-toolbar-save-as

## 한 줄 정의
사용자가 다른 이름으로 저장 버튼을 누르면 저장 대화상자가 나타난다.

## 진입점
- 하단 도구 모음: 다른 이름으로 저장 버튼 클릭

## 사용자 흐름
1. 사용자가 다른 이름으로 저장 버튼을 클릭한다.
2. 저장 대화상자가 나타난다.
3. 파일 이름, 위치, 형식을 지정한다.
4. 저장을 클릭한다.
5. 이미지가 지정한 형식으로 저장된다.

## 성공 조건
- [ ] 클릭 시 저장 대화상자가 표시된다.
- [ ] 선택한 형식으로 이미지가 저장된다.

## 실패 / 취소 흐름
- 저장 취소 → 파일이 저장되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 저장 동작 (별도 spec).
