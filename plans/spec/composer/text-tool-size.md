# Spec: text-tool-size

## 한 줄 정의
사용자가 텍스트 크기를 변경하면 텍스트 크기가 조정된다.

## 진입점
- 속성 패널: 텍스트 크기 입력

## 사용자 흐름
1. 사용자가 텍스트 클립을 선택한다.
2. 속성 패널에서 텍스트 크기를 입력한다.
3. 텍스트 크기가 조정된다.
4. 프리뷰에 반영된다.

## 성공 조건
- [ ] 크기 입력 시 텍스트 크기가 조정된다.
- [ ] 프리뷰에 반영된다.

## 실패 / 취소 흐름
- 잘못된 값 → 이전 값으로 복원된다.

## 경계 / 예외
- 최소/최대 크기 제한이 있다.

## 범위 외
- 글꼴 선택 동작 (별도 spec).
- 텍스트 색상 동작.
