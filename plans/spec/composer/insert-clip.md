# Spec: insert-clip

## 한 줄 정의
사용자가 삽입 버튼을 클릭하면 인/아웃 구간이 타임라인에 삽입된다.

## 진입점
- 소스 프리뷰: 삽입 버튼 또는 ,

## 사용자 흐름
1. 사용자가 소스 프리뷰에서 인/아웃 포인트를 설정한다.
2. 삽입 버튼을 클릭한다.
3. 인/아웃 구간이 재생 헤드 위치에 삽입된다.
4. 기존 클립이 뒤로 밀려난다.

## 성공 조건
- [ ] 인/아웃 구간이 재생 헤드 위치에 삽입된다.
- [ ] 기존 클립이 뒤로 밀려난다.

## 실패 / 취소 흐름
- 인/아웃 미설정 → 전체 클립이 삽입된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 덮어쓰기 동작 (별도 spec).
- 클립 드래그 동작.
