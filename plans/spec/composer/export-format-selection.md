# Spec: export-format-selection

## 한 줄 정의
사용자가 내보내기 형식을 선택하면 해당 형식으로 내보내기가 설정된다.

## 진입점
- 내보내기 대화상자: 형식 드롭다운

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 형식 드롭다운을 클릭한다.
2. 지원 형식 목록이 표시된다 (MP4, MOV, AVI, WebM, GIF).
3. 원하는 형식을 선택한다.

## 성공 조건
- [ ] 드롭다운에서 형식을 선택할 수 있다.
- [ ] 선택한 형식에 따라 코덱 옵션이 변경된다.

## 실패 / 취소 흐름
- 바깥 클릭 → 드롭다운이 닫히고 이전 형식이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 프로젝트 저장 동작.
