# Spec: export-execute

## 한 줄 정의
사용자가 내보내기 실행 버튼을 클릭하면 렌더링이 시작된다.

## 진입점
- 내보내기 대화상자: 내보내기 버튼 클릭

## 사용자 흐름
1. 사용자가 내보내기 설정을 확인한다.
2. 내보내기 버튼을 클릭한다.
3. 렌더링이 시작된다.
4. 진행 바가 표시된다.
5. 완료 시 파일이 저장되고 완료 토스트가 표시된다.

## 성공 조건
- [ ] 클릭 시 렌더링이 시작된다.
- [ ] 진행 바가 표시된다.
- [ ] 완료 시 파일이 저장된다.

## 실패 / 취소 흐름
- 렌더링 중 취소 → 렌더링이 중단되고 임시 파일이 삭제된다.
- 디스크 공간 부족 → 오류 토스트가 표시된다.

## 경계 / 예외
- 대용량 프로젝트 → 렌더링에 시간이 오래 걸릴 수 있다.

## 범위 외
- 내보내기 설정 동작 (별도 spec).
- 프로젝트 저장 동작.
