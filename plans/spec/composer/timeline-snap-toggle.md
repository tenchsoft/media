# Spec: timeline-snap-toggle

## 한 줄 정의
사용자가 스냅 토글을 켜면 클립이 다른 클립이나 마커에 스냅된다.

## 진입점
- 타임라인: 스냅 버튼 클릭 또는 N

## 사용자 흐름
1. 사용자가 스냅 버튼을 클릭하여 활성화한다.
2. 클립을 드래그할 때 다른 클립의 시작/끝에 스냅된다.
3. 마커 위치에도 스냅된다.

## 성공 조건
- [ ] 활성화 시 클립이 다른 클립의 시작/끝에 스냅된다.
- [ ] 비활성화 시 자유롭게 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 타임라인 확대/축소 동작 (별도 spec).
- 클립 이동 동작.
