# Spec: keyframe-delete

## 한 줄 정의
사용자가 키프레임을 삭제하면 해당 키프레임이 제거된다.

## 진입점
- 타임라인: 키프레임 다이아몬드 우클릭 → 삭제

## 사용자 흐름
1. 사용자가 타임라인에서 키프레임 다이아몬드를 우클릭한다.
2. 삭제를 선택한다.
3. 해당 키프레임이 제거된다.

## 성공 조건
- [ ] 삭제 시 해당 키프레임이 제거된다.
- [ ] 애니메이션이 인접 키프레임으로 재계산된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 마지막 키프레임 삭제 → 애니메이션이 고정값이 된다.

## 범위 외
- 키프레임 추가 동작 (별도 spec).
- 키프레임 보간 동작.
