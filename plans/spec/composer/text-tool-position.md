# Spec: text-tool-position

## 한 줄 정의
사용자가 텍스트를 드래그하면 텍스트 위치가 이동한다.

## 진입점
- 프리뷰: 텍스트 드래그

## 사용자 흐름
1. 사용자가 텍스트 클립을 선택한다.
2. 프리뷰에서 텍스트를 잡는다.
3. 원하는 위치로 드래그한다.
4. 놓으면 텍스트 위치가 이동한다.

## 성공 조건
- [ ] 드래그 시 텍스트가 실시간으로 이동한다.
- [ ] 놓으면 텍스트 위치가 확정된다.

## 실패 / 취소 흐름
- Esc → 이동이 취소되고 원래 위치로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 텍스트 내용 편집 동작 (별도 spec).
- 텍스트 크기 동작.
