# Spec: clip-scale-transform

## 한 줄 정의
사용자가 클립 크기를 조정하면 비디오가 확대되거나 축소된다.

## 진입점
- 속성 패널: 크기 입력 또는 프리뷰 핸들 드래그

## 사용자 흐름
1. 사용자가 비디오 클립을 선택한다.
2. 속성 패널에서 크기를 입력하거나 프리뷰에서 모서리 핸들을 드래그한다.
3. 비디오 크기가 실시간으로 변경된다.

## 성공 조건
- [ ] 크기 입력 또는 핸들 드래그 시 비디오 크기가 실시간으로 변경된다.
- [ ] 프리뷰에 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- Shift 드래그 → 비율이 유지된다.

## 범위 외
- 클립 위치 동작 (별도 spec).
- 클립 회전 동작.
