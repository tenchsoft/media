# Spec: in-point-set

## 한 줄 정의
사용자가 인 포인트를 설정하면 재생 헤드 위치가 클립의 시작점으로 설정된다.

## 진입점
- 소스 프리뷰: 인 포인트 버튼 또는 I

## 사용자 흐름
1. 사용자가 소스 프리뷰에서 재생 헤드를 원하는 위치로 이동한다.
2. 인 포인트 버튼을 클릭하거나 I 키를 누른다.
3. 해당 위치가 인 포인트로 설정된다.
4. 인 포인트 이전 구간이 회색으로 표시된다.

## 성공 조건
- [ ] 클릭 시 해당 위치가 인 포인트로 설정된다.
- [ ] 인 포인트 이전 구간이 회색으로 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 인 포인트가 아웃 포인트 뒤 → 아웃 포인트가 자동 조정된다.

## 범위 외
- 아웃 포인트 동작 (별도 spec).
- 클립 삽입 동작.
