# Spec: next-frame-button

## 한 줄 정의
사용자가 다음 프레임 버튼을 클릭하면 재생 헤드가 한 프레임 앞으로 이동한다.

## 진입점
- 타임라인: 다음 프레임 버튼 클릭 또는 오른쪽 화살표(Shift)

## 사용자 흐름
1. 사용자가 다음 프레임 버튼을 클릭한다.
2. 재생 헤드가 한 프레임 앞으로 이동한다.
3. 프리뷰가 해당 프레임을 표시한다.

## 성공 조건
- [ ] 클릭 시 재생 헤드가 한 프레임 앞으로 이동한다.
- [ ] 프리뷰가 해당 프레임을 표시한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 타임라인 끝 → 이동하지 않는다.

## 범위 외
- 이전 프레임 동작 (별도 spec).
- 재생 동작.
