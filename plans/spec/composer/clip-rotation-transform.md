# Spec: clip-rotation-transform

## 한 줄 정의
사용자가 클립 회전을 조정하면 비디오가 회전한다.

## 진입점
- 속성 패널: 회전 입력

## 사용자 흐름
1. 사용자가 비디오 클립을 선택한다.
2. 속성 패널에서 회전 값을 입력한다.
3. 비디오가 해당 각도로 회전한다.
4. 프리뷰에 반영된다.

## 성공 조건
- [ ] 회전 값 입력 시 비디오가 해당 각도로 회전한다.
- [ ] 프리뷰에 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 0~360도 범위.

## 범위 외
- 클립 크기 동작 (별도 spec).
- 클립 위치 동작.
