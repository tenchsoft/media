# Spec: effect-remove

## 한 줄 정의
사용자가 적용된 효과를 제거하면 클립에서 효과가 사라진다.

## 진입점
- 속성 패널: 효과 제거 버튼

## 사용자 흐름
1. 사용자가 클립을 선택한다.
2. 속성 패널에서 적용된 효과를 확인한다.
3. 제거 버튼을 클릭한다.
4. 효과가 클립에서 제거된다.

## 성공 조건
- [ ] 클릭 시 효과가 클립에서 제거된다.
- [ ] 프리뷰에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 효과 적용 동작 (별도 spec).
- 효과 설정 동작.
