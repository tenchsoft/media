# Spec: clip-nest

## 한 줄 정의
사용자가 여러 클립을 중첩하면 하나의 클립으로 결합된다.

## 진입점
- 타임라인: 여러 클립 선택 후 우클릭 → 중첩

## 사용자 흐름
1. 사용자가 타임라인에서 여러 클립을 선택한다.
2. 우클릭하여 중첩을 선택한다.
3. 선택된 클립이 하나의 중첩 시퀀스로 결합된다.
4. 타임라인에 단일 클립으로 표시된다.
5. 더블 클릭하면 내부 시퀀스를 편집할 수 있다.

## 성공 조건
- [ ] 여러 클립이 하나의 중첩 시퀀스로 결합된다.
- [ ] 더블 클릭으로 내부를 편집할 수 있다.

## 실패 / 취소 흐름
