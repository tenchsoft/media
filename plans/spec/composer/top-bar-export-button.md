# Spec: top-bar-export-button

## 한 줄 정의
사용자가 내보내기 버튼을 클릭하면 프로젝트를 비디오 파일로 내보낼 수 있다.

## 진입점
- 상단 바: 내보내기 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 내보내기 버튼을 클릭한다.
2. 내보내기 설정 대화상자가 열린다.
3. 형식, 해상도, 비트레이트 등을 설정한다.
4. 내보내기 버튼을 클릭하여 렌더링을 시작한다.
5. 진행 바가 표시되고 완료 시 파일이 저장된다.

## 성공 조건
- [ ] 클릭 시 내보내기 설정 대화상자가 열린다.
- [ ] 렌더링 완료 시 파일이 저장된다.

## 실패 / 취소 흐름
- 취소 → 대화상자가 닫힌다.
- 렌더링 중 취소 → 렌더링이 중단되고 임시 파일이 삭제된다.

## 경계 / 예외
- 디스크 공간 부족 → 오류 토스트가 표시된다.

## 범위 외
- 가져오기 동작 (별도 spec).
- 프로젝트 저장 동작.
