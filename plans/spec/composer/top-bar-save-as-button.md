# Spec: top-bar-save-as-button

## 한 줄 정의
사용자가 다른 이름으로 저장 버튼을 클릭하면 새 이름으로 프로젝트가 저장된다.

## 진입점
- 상단 바 메뉴: 다른 이름으로 저장 클릭 또는 Ctrl+Shift+S

## 사용자 흐름
1. 사용자가 다른 이름으로 저장을 클릭한다.
2. 파일 이름 입력 대화상자가 열린다.
3. 새 이름을 입력하고 저장을 클릭한다.
4. 새 파일로 프로젝트가 저장된다.

## 성공 조건
- [ ] 클릭 시 파일 이름 입력 대화상자가 열린다.
- [ ] 새 이름으로 프로젝트가 저장된다.

## 실패 / 취소 흐름
- 취소 → 대화상자가 닫히고 아무 일도 발생하지 않는다.

## 경계 / 예외
- 같은 이름 → 덮어쓰기 확인 대화상자가 표시된다.

## 범위 외
- 저장 동작 (별도 spec).
- 내보내기 동작.
