# Spec: audio-volume-keyframe

## 한 줄 정의
사용자가 오디오 볼륨 키프레임을 추가하면 볼륨이 시간에 따라 변한다.

## 진입점
- 타임라인: 오디오 클립 볼륨 라인 클릭

## 사용자 흐름
1. 사용자가 오디오 클립을 선택한다.
2. 타임라인에서 볼륨 라인 위의 원하는 위치를 클릭한다.
3. 키프레임이 추가된다.
4. 키프레임을 위아래로 드래그하여 볼륨을 조정한다.
5. 시간에 따라 볼륨이 변한다.

## 성공 조건
- [ ] 클릭 시 볼륨 키프레임이 추가된다.
- [ ] 드래그로 볼륨을 조정할 수 있다.
- [ ] 키프레임 간 볼륨이 부드럽게 변한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외

- 오디오 페이드 동작.
