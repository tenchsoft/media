# Spec: clip-copy-paste

## 한 줄 정의
사용자가 클립을 복사하고 붙여넣으면 동일한 클립이 타임라인에 추가된다.

## 진입점
- 타임라인: 클립 선택 후 Ctrl+C → Ctrl+V

## 사용자 흐름
1. 사용자가 타임라인에서 클립을 선택한다.
2. Ctrl+C를 눌러 복사한다.
3. 재생 헤드를 원하는 위치로 이동한다.
4. Ctrl+V를 눌러 붙여넣는다.
5. 동일한 클립이 재생 헤드 위치에 추가된다.

## 성공 조건
- [ ] Ctrl+C로 클립이 복사된다.
- [ ] Ctrl+V로 재생 헤드 위치에 붙여넣기된다.

## 실패 / 취소 흐름
- 클립 미선택 상태에서 복사 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 여러 클립 동시 복사/붙여넣기 가능하다.

## 범위 외
- 클립 이동 동작 (별도 spec).
- 클립 삭제 동작.
