# Spec: clip-speed-change

## 한 줄 정의
사용자가 클립 속도를 변경하면 재생 속도가 빨라지거나 느려진다.

## 진입점
- 타임라인: 클립 우클릭 → 속도/지속 시간

## 사용자 흐름
1. 사용자가 타임라인에서 클립을 우클릭한다.
2. 속도/지속 시간을 선택한다.
3. 속도 값을 입력하거나 슬라이더를 조정한다.
4. 적용 버튼을 클릭한다.
5. 클립의 재생 속도가 변경된다.

## 성공 조건
- [ ] 속도 변경 시 클립 재생 속도가 변경된다.
- [ ] 타임라인에서 클립 길이가 조정된다.

## 실패 / 취소 흐름
- 취소 → 이전 속도가 유지된다.

## 경계 / 예외
- 역방향 재생 설정이 가능하다.

## 범위 외
- 클립 자르기 동작 (별도 spec).
- 클립 분할 동작.
