# Spec: clip-split-at-playhead

## 한 줄 정의
사용자가 분할 버튼을 클릭하면 재생 헤드 위치에서 클립이 두 부분으로 나뉜다.

## 진입점
- 타임라인: 분할 버튼 클릭 또는 Ctrl+B

## 사용자 흐름
1. 사용자가 재생 헤드를 원하는 위치로 이동한다.
2. 분할할 클립을 선택한다.
3. 분할 버튼을 클릭한다.
4. 재생 헤드 위치에서 클립이 두 부분으로 나뉜다.

## 성공 조건
- [ ] 클릭 시 재생 헤드 위치에서 클립이 두 부분으로 나뉜다.
- [ ] 두 클립 모두 독립적으로 편집할 수 있다.

## 실패 / 취소 흐름
- 클립 미선택 → 아무 일도 발생하지 않는다.
- 재생 헤드가 클립 범위 밖 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 클립 자르기 동작 (별도 spec).
- 클립 삭제 동작.
