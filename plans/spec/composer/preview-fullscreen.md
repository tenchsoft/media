# Spec: preview-fullscreen

## 한 줄 정의
사용자가 전체 화면 버튼을 클릭하면 프리뷰가 전체 화면으로 표시된다.

## 진입점
- 프리뷰: 전체 화면 버튼 클릭

## 사용자 흐름
1. 사용자가 프리뷰에서 전체 화면 버튼을 클릭한다.
2. 프리뷰가 전체 화면으로 표시된다.
3. Esc 키를 누르면 원래 크기로 돌아간다.

## 성공 조건
- [ ] 클릭 시 프리뷰가 전체 화면으로 표시된다.
- [ ] Esc 키로 원래 크기로 돌아간다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 프리뷰 재생 동작 (별도 spec).
- 프리뷰 확대/축소 동작.
