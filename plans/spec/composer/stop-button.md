# Spec: stop-button

## 한 줄 정의
사용자가 정지 버튼을 클릭하면 재생이 정지되고 재생 헤드가 시작 위치로 돌아간다.

## 진입점
- 타임라인: 정지 버튼 클릭

## 사용자 흐름
1. 사용자가 정지 버튼을 클릭한다.
2. 재생이 정지된다.
3. 재생 헤드가 마지막 재생 시작 위치로 돌아간다.

## 성공 조건
- [ ] 클릭 시 재생이 정지된다.
- [ ] 재생 헤드가 시작 위치로 돌아간다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 일시정지 동작 (별도 spec).
- 재생 동작.
