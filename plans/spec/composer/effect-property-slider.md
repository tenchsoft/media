# Spec: effect-property-slider

## 한 줄 정의
사용자가 효과 속성 슬라이더를 조정하면 효과 강도가 변경된다.

## 진입점
- 속성 패널: 효과 속성 슬라이더

## 사용자 흐름
1. 사용자가 클립을 선택한다.
2. 속성 패널에서 적용된 효과의 속성 슬라이더를 드래그한다.
3. 효과 강도가 실시간으로 변경된다.
4. 프리뷰에 반영된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 효과 강도가 실시간으로 변경된다.
- [ ] 프리뷰에 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 효과 적용 동작 (별도 spec).
- 효과 제거 동작.
