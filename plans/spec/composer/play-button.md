# Spec: play-button

## 한 줄 정의
사용자가 재생 버튼을 클릭하면 타임라인이 재생 헤드 위치에서 재생된다.

## 진입점
- 타임라인: 재생 버튼 클릭 또는 Space

## 사용자 흐름
1. 사용자가 재생 버튼을 클릭하거나 Space 키를 누른다.
2. 재생 헤드 위치에서 재생이 시작된다.
3. 프리뷰에 비디오가 재생된다.
4. 오디오가 함께 재생된다.

## 성공 조건
- [ ] 클릭 시 재생 헤드 위치에서 재생이 시작된다.
- [ ] 프리뷰에 비디오가 재생된다.
- [ ] 오디오가 함께 재생된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 타임라인 끝 → 재생이 정지된다.

## 범위 외
- 일시정지 동작 (별도 spec).
- 정지 동작.
