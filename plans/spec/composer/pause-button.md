# Spec: pause-button

## 한 줄 정의
사용자가 일시정지 버튼을 클릭하면 재생이 일시정지된다.

## 진입점
- 타임라인: 일시정지 버튼 클릭 또는 Space

## 사용자 흐름
1. 사용자가 재생 중에 일시정지 버튼을 클릭하거나 Space 키를 누른다.
2. 재생이 일시정지된다.
3. 재생 헤드가 현재 위치에 멈춘다.

## 성공 조건
- [ ] 클릭 시 재생이 일시정지된다.
- [ ] 재생 헤드가 현재 위치에 멈춘다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 재생 동작 (별도 spec).
- 정지 동작.
