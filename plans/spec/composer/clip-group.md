# Spec: clip-group

## 한 줄 정의
사용자가 여러 클립을 그룹화하면 하나의 단위로 함께 이동할 수 있다.

## 진입점
- 타임라인: 여러 클립 선택 후 우클릭 → 그룹화

## 사용자 흐름
1. 사용자가 타임라인에서 여러 클립을 선택한다.
2. 우클릭하여 그룹화를 선택한다.
3. 클립이 그룹화된다.
4. 하나의 클립을 드래그하면 그룹 전체가 함께 이동한다.

## 성공 조건
- [ ] 여러 클립을 그룹화할 수 있다.
- [ ] 그룹화된 클립이 함께 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 그룹 해제 → 우클릭 → 그룹 해제.

## 범위 외
- 클립 복사/붙여넣기 동작 (별도 spec).
- 클립 잠금 동작.
