# Spec: export-bitrate

## 한 줄 정의
사용자가 내보내기 비트레이트를 조정하면 영상 품질이 설정된다.

## 진입점
- 내보내기 대화상자: 비트레이트 입력

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 비트레이트 값을 입력한다.
2. 예상 파일 크기가 표시된다.
3. 높은 비트레이트 → 높은 품질, 큰 파일.
4. 낮은 비트레이트 → 낮은 품질, 작은 파일.

## 성공 조건
- [ ] 비트레이트 입력 시 예상 파일 크기가 표시된다.
- [ ] 품질과 파일 크기의 관계가 명확하다.

## 실패 / 취소 흐름
- 잘못된 값 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 해상도 동작.
