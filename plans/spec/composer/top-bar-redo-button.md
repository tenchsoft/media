# Spec: top-bar-redo-button

## 한 줄 정의
사용자가 다시 실행 버튼을 클릭하면 취소된 편집 동작이 다시 적용된다.

## 진입점
- 상단 바: 다시 실행 버튼 클릭 또는 Ctrl+Shift+Z

## 사용자 흐름
1. 사용자가 다시 실행 버튼을 클릭하거나 Ctrl+Shift+Z를 누른다.
2. 취소된 편집 동작이 다시 적용된다.
3. 타임라인과 프리뷰가 다시 실행된 상태로 복원된다.

## 성공 조건
- [ ] 취소된 편집 동작이 다시 적용된다.
- [ ] 타임라인과 프리뷰가 다시 실행된 상태로 복원된다.

## 실패 / 취소 흐름
- 다시 실행할 동작이 없음 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 실행 취소 동작 (별도 spec).
- 프로젝트 저장 동작.
