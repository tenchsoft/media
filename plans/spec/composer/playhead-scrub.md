# Spec: playhead-scrub

## 한 줄 정의
사용자가 재생 헤드를 드래그하면 프리뷰가 해당 위치로 이동한다.

## 진입점
- 타임라인: 재생 헤드 드래그

## 사용자 흐름
1. 사용자가 타임라인 상단의 재생 헤드를 잡는다.
2. 드래그하여 원하는 위치로 이동한다.
3. 프리뷰가 해당 위치의 프레임을 표시한다.

## 성공 조건
- [ ] 드래그 시 프리뷰가 실시간으로 해당 프레임을 표시한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 빠른 드래그 → 낮은 해상도로 스크러빙된다.

## 범위 외
- 재생/일시정지 동작 (별도 spec).
- 타임라인 확대/축소 동작.
