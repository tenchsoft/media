# Spec: color-correction-panel

## 한 줄 정의
사용자가 색 보정 패널을 열면 클립의 색상을 보정할 수 있다.

## 진입점
- 속성 패널: 색 보정 탭

## 사용자 흐름
1. 사용자가 클립을 선택한다.
2. 속성 패널에서 색 보정 탭을 선택한다.
3. 밝기, 대비, 채도, 색온도 등을 조정한다.
4. 프리뷰에 실시간으로 반영된다.

## 성공 조건
- [ ] 색 보정 슬라이더를 조정할 수 있다.
- [ ] 프리뷰에 실시간으로 반영된다.

## 실패 / 취소 흐름
- 재설정 → 기본값으로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 효과 적용 동작 (별도 spec).
- 클립 불투명도 동작.
