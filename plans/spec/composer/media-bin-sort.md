# Spec: media-bin-sort

## 한 줄 정의
사용자가 미디어 bin 정렬 기준을 변경하면 파일 목록이 해당 기준으로 정렬된다.

## 진입점
- 미디어 bin: 정렬 드롭다운

## 사용자 흐름
1. 사용자가 미디어 bin에서 정렬 드롭다운을 클릭한다.
2. 정렬 기준을 선택한다 (이름, 날짜, 유형, 지속 시간).
3. 파일 목록이 해당 기준으로 정렬된다.

## 성공 조건
- [ ] 정렬 기준 선택 시 파일 목록이 해당 기준으로 정렬된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 검색 동작 (별도 spec).
- 파일 가져오기 동작.
