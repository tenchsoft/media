# Spec: clip-move-on-timeline

## 한 줄 정의
사용자가 타임라인에서 클립을 드래그하면 클립 위치가 이동된다.

## 진입점
- 타임라인: 클립 드래그

## 사용자 흐름
1. 사용자가 타임라인에서 클립을 잡는다.
2. 원하는 위치로 드래그한다.
3. 놓으면 클립이 새 위치로 이동된다.

## 성공 조건
- [ ] 드래그 시 클립이 실시간으로 이동한다.
- [ ] 놓으면 새 위치에 배치된다.

## 실패 / 취소 흐름
- Esc → 이동이 취소되고 원래 위치로 복원된다.

## 경계 / 예외
- 다른 클립과 겹침 → 스냅 또는 밀어내기 동작이 적용된다.

## 범위 외
- 클립 추가 동작 (별도 spec).
- 클립 자르기 동작.
