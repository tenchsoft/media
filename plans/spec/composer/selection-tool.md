# Spec: selection-tool

## 한 줄 정의
사용자가 선택 도구를 활성화하면 클립을 선택하고 이동할 수 있다.

## 진입점
- 도구 모음: 선택 도구 또는 V

## 사용자 흐름
1. 사용자가 도구 모음에서 선택 도구를 선택한다.
2. 타임라인에서 클립을 클릭하여 선택한다.
3. 클립을 드래그하여 이동한다.
4- 트림 핸들을 드래그하여 자른다.

## 성공 조건
- [ ] 클릭으로 클립을 선택할 수 있다.
- [ ] 드래그로 클립을 이동할 수 있다.
- [ ] 트림 핸들로 클립을 자를 수 있다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 여러 클립 선택 → Shift+클릭 또는 드래그 선택.

## 범위 외
- 면도날 도구 동작 (별도 spec).
- 텍스트 도구 동작.
