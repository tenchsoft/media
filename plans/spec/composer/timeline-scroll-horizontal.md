# Spec: timeline-scroll-horizontal

## 한 줄 정의
사용자가 타임라인을 수평 스크롤하면 타임라인이 좌우로 이동한다.

## 진입점
- 타임라인: Shift+마우스 휠 또는 수평 스크롤바

## 사용자 흐름
1. 사용자가 Shift+마우스 휠을 돌리거나 수평 스크롤바를 드래그한다.
2. 타임라인이 좌우로 이동한다.

## 성공 조건
- [ ] 수평 스크롤 시 타임라인이 좌우로 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 타임라인 확대/축소 동작 (별도 spec).
- 수직 스크롤 동작.
