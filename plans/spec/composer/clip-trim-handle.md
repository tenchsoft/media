# Spec: clip-trim-handle

## 한 줄 정의
사용자가 클립의 시작 또는 끝 핸들을 드래그하면 클립 길이가 조정된다.

## 진입점
- 타임라인: 클립 시작/끝 핸들 드래그

## 사용자 흐름
1. 사용자가 클립의 시작 또는 끝 가장자리를 잡는다.
2. 드래그하여 클립 길이를 조정한다.
3. 놓으면 클립 길이가 확정된다.

## 성공 조건
- [ ] 드래그 시 클립 길이가 실시간으로 조정된다.
- [ ] 놓으면 클립 길이가 확정된다.

## 실패 / 취소 흐름
- Esc → 조정이 취소되고 원래 길이로 복원된다.

## 경계 / 예외
- 원본 미디어 길이를 초과할 수 없다.
- 최소 클립 길이 제한이 있다.

## 범위 외
- 클립 분할 동작 (별도 spec).
- 클립 이동 동작.
