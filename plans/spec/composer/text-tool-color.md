# Spec: text-tool-color

## 한 줄 정의
사용자가 텍스트 색상을 변경하면 텍스트에 해당 색상이 적용된다.

## 진입점
- 속성 패널: 텍스트 색상 버튼

## 사용자 흐름
1. 사용자가 텍스트 클립을 선택한다.
2. 속성 패널에서 텍스트 색상 버튼을 클릭한다.
3. 색상 선택기에서 색상을 선택한다.
4. 텍스트에 해당 색상이 적용된다.

## 성공 조건
- [ ] 색상 선택 시 텍스트에 해당 색상이 적용된다.
- [ ] 프리뷰에 반영된다.

## 실패 / 취소 흐름
- 취소 → 이전 색상이 유지된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 글꼴 선택 동작 (별도 spec).
- 텍스트 크기 동작.
