# Spec: audio-fade-handle

## 한 줄 정의
사용자가 오디오 클립의 페이드 핸들을 드래그하면 오디오가 페이드 인/아웃된다.

## 진입점
- 타임라인: 오디오 클립 좌우 상단 핸들 드래그

## 사용자 흐름
1. 사용자가 오디오 클립의 왼쪽 상단 핸들을 드래그한다.
2. 오디오 페이드 인이 적용된다.
3. 오른쪽 상단 핸들을 드래그하면 오디오 페이드 아웃이 적용된다.

## 성공 조건
- [ ] 왼쪽 핸들 드래그 시 페이드 인이 적용된다.
- [ ] 오른쪽 핸들 드래그 시 페이드 아웃이 적용된다.
- [ ] 핸들 길이에 따라 페이드 시간이 결정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.
