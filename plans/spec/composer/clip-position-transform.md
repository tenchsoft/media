# Spec: clip-position-transform

## 한 줄 정의
사용자가 클립 위치를 드래그하면 비디오가 프레임 내에서 이동한다.

## 진입점
- 프리뷰: 비디오 클립 드래그

## 사용자 흐름
1. 사용자가 비디오 클립을 선택한다.
2. 프리뷰에서 비디오를 잡는다.
3. 원하는 위치로 드래그한다.
4. 놓으면 비디오 위치가 이동한다.

## 성공 조건
- [ ] 드래그 시 비디오가 실시간으로 이동한다.
- [ ] 놓으면 위치가 확정된다.

## 실패 / 취소 흐름
- Esc → 이동이 취소되고 원래 위치로 복원된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 클립 자르기 동작 (별도 spec).
- 클립 불투명도 동작.
