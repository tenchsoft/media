# Spec: transition-drag-between-clips

## 한 줄 정의
사용자가 전환 효과를 두 클립 사이에 드래그하면 전환이 적용된다.

## 진입점
- 효과 패널 → 타임라인: 두 클립 사이에 드래그 앤 드롭

## 사용자 흐름
1. 사용자가 효과 패널에서 전환 효과를 잡는다.
2. 타임라인에서 두 클립 사이에 드래그한다.
3. 놓으면 전환 효과가 적용된다.

## 성공 조건
- [ ] 드래그 앤 드롭으로 전환 효과가 적용된다.
- [ ] 프리뷰에 전환 효과가 반영된다.

## 실패 / 취소 흐름
- 클립 사이가 아닌 위치 → 전환이 적용되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전환 지속 시간 동작 (별도 spec).
- 전환 제거 동작.
