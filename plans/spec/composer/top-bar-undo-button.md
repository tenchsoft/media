# Spec: top-bar-undo-button

## 한 줄 정의
사용자가 실행 취소 버튼을 클릭하면 마지막 편집 동작이 취소된다.

## 진입점
- 상단 바: 실행 취소 버튼 클릭 또는 Ctrl+Z

## 사용자 흐름
1. 사용자가 실행 취소 버튼을 클릭하거나 Ctrl+Z를 누른다.
2. 마지막 편집 동작이 취소된다.
3. 타임라인과 프리뷰가 이전 상태로 복원된다.

## 성공 조건
- [ ] 마지막 편집 동작이 취소된다.
- [ ] 타임라인과 프리뷰가 이전 상태로 복원된다.

## 실패 / 취소 흐름
- 되돌릴 동작이 없음 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 다시 실행 동작 (별도 spec).
- 프로젝트 저장 동작.
