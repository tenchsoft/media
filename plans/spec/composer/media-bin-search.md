# Spec: media-bin-search

## 한 줄 정의
사용자가 미디어 bin에서 검색어를 입력하면 해당하는 파일만 표시된다.

## 진입점
- 미디어 bin: 검색 입력 필드

## 사용자 흐름
1. 사용자가 미디어 bin에서 검색 입력 필드를 클릭한다.
2. 검색어를 입력한다.
3. 파일 이름에 검색어가 포함된 파일만 표시된다.

## 성공 조건
- [ ] 검색어 입력 시 해당하는 파일만 표시된다.
- [ ] 실시간으로 필터링된다.

## 실패 / 취소 흐름
- 검색어 지움 → 전체 파일이 다시 표시된다.

## 경계 / 예외
- 결과 없음 → "검색 결과가 없습니다" 안내가 표시된다.

## 범위 외
- 파일 정렬 동작 (별도 spec).
- 파일 가져오기 동작.
