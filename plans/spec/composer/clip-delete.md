# Spec: clip-delete

## 한 줄 정의
사용자가 클립을 삭제하면 타임라인에서 클립이 제거된다.

## 진입점
- 타임라인: 클립 선택 후 Delete 키

## 사용자 흐름
1. 사용자가 타임라인에서 클립을 선택한다.
2. Delete 키를 누른다.
3. 클립이 타임라인에서 제거된다.
4. 뒤의 클립이 앞으로 당겨지거나 빈 공간이 남는다.

## 성공 조건
- [ ] Delete 키 시 클립이 타임라인에서 제거된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 잠긴 트랙의 클립 → 삭제되지 않는다.

## 범위 외
- 클립 분할 동작 (별도 spec).
- 클립 이동 동작.
