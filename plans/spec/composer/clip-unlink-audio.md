# Spec: clip-unlink-audio

## 한 줄 정의
사용자가 비디오와 오디오 연결을 해제하면 각각 독립적으로 편집할 수 있다.

## 진입점
- 타임라인: 클립 우클릭 → 연결 해제

## 사용자 흐름
1. 사용자가 연결된 비디오/오디오 클립을 우클릭한다.
2. 연결 해제를 선택한다.
3. 비디오와 오디오가 독립적으로 편집 가능해진다.
4. 각각 개별적으로 이동, 자르기, 삭제할 수 있다.

## 성공 조건
- [ ] 연결 해제 시 비디오와 오디오가 독립적으로 편집 가능해진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 클립 그룹 동작 (별도 spec).
- 클립 중첩 동작.
