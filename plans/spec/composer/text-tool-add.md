# Spec: text-tool-add

## 한 줄 정의
사용자가 텍스트 도구를 선택하고 클릭하면 텍스트 클립이 추가된다.

## 진입점
- 도구 모음: 텍스트 도구 → 프리뷰 클릭

## 사용자 흐름
1. 사용자가 도구 모음에서 텍스트 도구를 선택한다.
2. 프리뷰에서 원하는 위치를 클릭한다.
3. 텍스트 입력 필드가 나타난다.
4. 텍스트를 입력한다.
5. 텍스트 클립이 타임라인에 추가된다.

## 성공 조건
- [ ] 클릭 시 텍스트 입력 필드가 나타난다.
- [ ] 텍스트 입력 후 타임라인에 텍스트 클립이 추가된다.

## 실패 / 취소 흐름
- Esc → 텍스트 입력이 취소된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 텍스트 속성 편집 동작 (별도 spec).
- 자막 동작.
