# Spec: top-bar-import-button

## 한 줄 정의
사용자가 가져오기 버튼을 클릭하면 미디어 파일을 프로젝트에 추가할 수 있다.

## 진입점
- 상단 바: 가져오기 버튼 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 가져오기 버튼을 클릭한다.
2. 파일 선택 대화상자가 열린다.
3. 비디오, 오디오, 이미지 파일을 선택한다.
4. 선택한 파일이 프로젝트 미디어 bin에 추가된다.

## 성공 조건
- [ ] 클릭 시 파일 선택 대화상자가 열린다.
- [ ] 선택한 파일이 미디어 bin에 추가된다.

## 실패 / 취소 흐름
- 취소 → 대화상자가 닫히고 아무 일도 발생하지 않는다.
- 지원하지 않는 형식 → "지원하지 않는 형식입니다" 오류 토스트가 표시된다.

## 경계 / 예외
- 대용량 파일 → 가져오기 진행 바가 표시된다.

## 범위 외
- 내보내기 동작 (별도 spec).
- 프로젝트 저장 동작.
