# Spec: media-bin-preview

## 한 줄 정의
사용자가 미디어 bin에서 파일을 더블 클릭하면 소스 프리뷰가 표시된다.

## 진입점
- 미디어 bin: 파일 더블 클릭

## 사용자 흐름
1. 사용자가 미디어 bin에서 파일을 더블 클릭한다.
2. 소스 프리뷰 창에 해당 파일이 표시된다.
3. 재생/일시정지로 미리 볼 수 있다.
4- 인/아웃 포인트를 설정할 수 있다.

## 성공 조건
- [ ] 더블 클릭 시 소스 프리뷰 창에 파일이 표시된다.
- [ ] 재생/일시정지로 미리 볼 수 있다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 가져오기 동작 (별도 spec).
- 타임라인 배치 동작.
