# Spec: clip-link-audio

## 한 줄 정의
사용자가 비디오와 오디오를 연결하면 함께 이동하고 편집된다.

## 진입점
- 타임라인: 비디오+오디오 클립 선택 후 우클릭 → 연결

## 사용자 흐름
1. 사용자가 비디오 클립과 오디오 클립을 선택한다.
2. 우클릭하여 연결을 선택한다.
3. 두 클립이 연결된다.
4. 하나를 이동하면 다른 것도 함께 이동한다.

## 성공 조건
- [ ] 연결 시 두 클립이 함께 이동한다.
- [ ] 함께 편집된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 연결 해제 동작 (별도 spec).
- 클립 그룹 동작.
