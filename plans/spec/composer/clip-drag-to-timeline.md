# Spec: clip-drag-to-timeline

## 한 줄 정의
사용자가 미디어 bin에서 클립을 드래그하여 타임라인에 배치한다.

## 진입점
- 미디어 bin → 타임라인: 클립 드래그 앤 드롭

## 사용자 흐름
1. 사용자가 미디어 bin에서 클립을 잡는다.
2. 타임라인의 원하는 위치로 드래그한다.
3. 놓으면 클립이 해당 위치에 배치된다.

## 성공 조건
- [ ] 드래그 앤 드롭으로 클립이 타임라인에 배치된다.
- [ ] 비디오 클립은 비디오 트랙에, 오디오 클립은 오디오 트랙에 배치된다.
- [ ] 놓을 때 삽입 위치가 시각적으로 표시된다.

## 실패 / 취소 흐름
- 잘못된 트랙 유형에 놓기 → 클립이 배치되지 않는다.

## 경계 / 예외
- 기존 클립 위에 놓기 → 기존 클립이 분할되거나 밀려난다.

## 범위 외
- 클립 이동 동작 (별도 spec).
- 클립 자르기 동작.
