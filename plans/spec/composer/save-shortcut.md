# Spec: save-shortcut

## 한 줄 정의
사용자가 Ctrl+S를 누르면 프로젝트가 저장된다.

## 진입점
- 키보드: Ctrl+S

## 사용자 흐름
1. 사용자가 Ctrl+S를 누른다.
2. 프로젝트가 저장된다.
3. 저장 완료 토스트가 표시된다.

## 성공 조건
- [ ] Ctrl+S 시 프로젝트가 저장된다.

## 실패 / 취소 흐름
- 디스크 공간 부족 → 오류 토스트가 표시된다.

## 경계 / 예외
- 처음 저장 → 다른 이름으로 저장 대화상자가 열린다.

## 범위 외
- 다른 이름으로 저장 동작 (별도 spec).
- 내보내기 동작.
