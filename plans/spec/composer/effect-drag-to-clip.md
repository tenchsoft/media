# Spec: effect-drag-to-clip

## 한 줄 정의
사용자가 효과를 클립에 드래그하면 해당 효과가 클립에 적용된다.

## 진입점
- 효과 패널 → 타임라인 클립: 드래그 앤 드롭

## 사용자 흐름
1. 사용자가 효과 패널에서 효과를 잡는다.
2. 타임라인의 클립 위로 드래그한다.
3. 놓으면 효과가 클립에 적용된다.

## 성공 조건
- [ ] 드래그 앤 드롭으로 효과가 클립에 적용된다.
- [ ] 프리뷰에 즉시 반영된다.

## 실패 / 취소 흐름
- 잘못된 대상 → 효과가 적용되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 효과 설정 동작 (별도 spec).
- 효과 제거 동작.
