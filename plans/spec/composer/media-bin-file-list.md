# Spec: media-bin-file-list

## 한 줄 정의
사용자가 미디어 bin을 보면 프로젝트에 가져온 모든 미디어 파일이 목록으로 표시된다.

## 진입점
- 미디어 bin 패널

## 사용자 흐름
1. 사용자가 미디어 bin 패널을 연다.
2. 프로젝트에 가져온 모든 미디어 파일이 목록으로 표시된다.
3. 썸네일, 파일 이름, 지속 시간이 표시된다.

## 성공 조건
- [ ] 프로젝트에 가져온 모든 미디어 파일이 표시된다.
- [ ] 썸네일, 파일 이름, 지속 시간이 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 파일이 없음 → "미디어를 가져오세요" 안내가 표시된다.

## 범위 외
- 파일 가져오기 동작 (별도 spec).
- 파일 삭제 동작.
