# Spec: timeline-zoom-slider

## 한 줄 정의
사용자가 타임라인 확대/축소 슬라이더를 조정하면 타임라인이 확대되거나 축소된다.

## 진입점
- 타임라인: 확대/축소 슬라이더 또는 Ctrl+마우스 휠

## 사용자 흐름
1. 사용자가 확대/축소 슬라이더를 드래그한다.
2. 타임라인이 확대되거나 축소된다.
3. 클립과 시간 눈금이 조정된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 타임라인이 확대/축소된다.
- [ ] 클립과 시간 눈금이 조정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최대 확대 → 개별 프레임이 표시된다.
- 최소 축소 → 전체 타임라인이 보인다.

## 범위 외
- 재생 헤드 스크러빙 동작 (별도 spec).
- 클립 이동 동작.
