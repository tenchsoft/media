# Spec: track-visibility-toggle

## 한 줄 정의
사용자가 트랙 가시성 버튼을 클릭하면 트랙이 보이거나 숨겨진다.

## 진입점
- 타임라인: 트랙 헤더의 눈 아이콘 클릭

## 사용자 흐름
1. 사용자가 트랙 헤더에서 눈 아이콘을 클릭한다.
2. 트랙이 숨겨진다 (프리뷰에 표시되지 않음).
3. 다시 클릭하면 트랙이 다시 보인다.

## 성공 조건
- [ ] 클릭 시 트랙이 숨겨진다.
- [ ] 다시 클릭 시 트랙이 다시 보인다.
- [ ] 프리뷰에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 트랙 잠금 동작 (별도 spec).
- 트랙 삭제 동작.
