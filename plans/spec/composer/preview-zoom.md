# Spec: preview-zoom

## 한 줄 정의
사용자가 프리뷰 크기를 조정하면 프리뷰가 확대되거나 축소된다.

## 진입점
- 프리뷰: 크기 드롭다운 (맞춤, 50%, 100%, 200%)

## 사용자 흐름
1. 사용자가 프리뷰에서 크기 드롭다운을 클릭한다.
2. 원하는 크기를 선택한다.
3. 프리뷰가 해당 크기로 조정된다.

## 성공 조건
- [ ] 선택 시 프리뷰가 해당 크기로 조정된다.
- [ ] 맞춤 → 프리뷰 영역에 맞게 조정된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전체 화면 동작 (별도 spec).
- 프리뷰 재생 동작.
