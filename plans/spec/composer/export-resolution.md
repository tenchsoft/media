# Spec: export-resolution

## 한 줄 정의
사용자가 내보내기 해상도를 선택하면 해당 해상도로 내보내기가 설정된다.

## 진입점
- 내보내기 대화상자: 해상도 드롭다운

## 사용자 흐름
1. 사용자가 내보내기 대화상자에서 해상도 드롭다운을 클릭한다.
2. 프리셋 해상도 목록이 표시된다 (1080p, 720p, 4K 등).
3. 원하는 해상도를 선택한다.
4. 또는 사용자 정의 해상도를 입력한다.

## 성공 조건
- [ ] 프리셋 해상도를 선택할 수 있다.
- [ ] 사용자 정의 해상도를 입력할 수 있다.

## 실패 / 취소 흐름
- 잘못된 값 → 오류 메시지가 표시된다.

## 경계 / 예외
- 가로세로 비율이 유지된다.

## 범위 외
- 내보내기 실행 동작 (별도 spec).
- 내보내기 형식 동작.
