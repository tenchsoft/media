# Spec: razor-tool

## 한 줄 정의
사용자가 면도날 도구로 클릭하면 클릭한 위치에서 클립이 분할된다.

## 진입점
- 도구 모음: 면도날 도구 → 타임라인 클릭

## 사용자 흐름
1. 사용자가 도구 모음에서 면도날 도구를 선택한다.
2. 타임라인에서 원하는 위치를 클릭한다.
3. 해당 위치에서 클립이 분할된다.

## 성공 조건
- [ ] 클릭 시 해당 위치에서 클립이 분할된다.
- [ ] 두 클립 모두 독립적으로 편집할 수 있다.

## 실패 / 취소 흐름
- 빈 영역 클릭 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 여러 트랙 위를 클릭 → 모든 트랙의 클립이 분할된다.

## 범위 외
- 분할 동작 (별도 spec).
- 클립 삭제 동작.
