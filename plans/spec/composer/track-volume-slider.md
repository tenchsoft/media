# Spec: track-volume-slider

## 한 줄 정의
사용자가 트랙 볼륨 슬라이더를 조정하면 해당 트랙의 오디오 볼륨이 변경된다.

## 진입점
- 타임라인: 트랙 헤더의 볼륨 슬라이더

## 사용자 흐름
1. 사용자가 트랙 헤더에서 볼륨 슬라이더를 드래그한다.
2. 해당 트랙의 오디오 볼륨이 실시간으로 변경된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 오디오 볼륨이 실시간으로 변경된다.
- [ ] 0dB → 원래 볼륨.
- [ ] -inf → 음소거.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 최대 볼륨 초과 → 클리핑 경고가 표시된다.

## 범위 외
- 클립 볼륨 동작 (별도 spec).
- 팬 동작.
