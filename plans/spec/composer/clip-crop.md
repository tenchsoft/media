# Spec: clip-crop

## 한 줄 정의
사용자가 클립 자르기를 설정하면 비디오 프레임이 해당 영역으로 제한된다.

## 진입점
- 속성 패널: 자르기 설정

## 사용자 흐름
1. 사용자가 비디오 클립을 선택한다.
2. 속성 패널에서 자르기 설정을 연다.
3. 위, 아래, 좌, 우 여백을 입력한다.
4. 프리뷰에 자르기 영역이 표시된다.
5. 적용 시 해당 영역만 표시된다.

## 성공 조건
- [ ] 여백 입력 시 자르기 영역이 실시간으로 표시된다.
- [ ] 적용 시 해당 영역만 표시된다.

## 실패 / 취소 흐름
- 취소 → 이전 상태로 복원된다.

## 경계 / 예외
- 여백이 너무 큼 → 경고가 표시된다.

## 범위 외
- 클립 불투명도 동작 (별도 spec).
- 클립 위치 동작.
