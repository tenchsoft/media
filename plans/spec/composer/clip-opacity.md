# Spec: clip-opacity

## 한 줄 정의
사용자가 클립 불투명도를 조정하면 비디오 클립의 투명도가 변경된다.

## 진입점
- 속성 패널: 불투명도 슬라이더

## 사용자 흐름
1. 사용자가 비디오 클립을 선택한다.
2. 속성 패널에서 불투명도 슬라이더를 드래그한다.
3. 클립의 투명도가 실시간으로 변경된다.
4. 프리뷰에 반영된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 클립 투명도가 실시간으로 변경된다.
- [ ] 100% → 완전 불투명.
- [ ] 0% → 완전 투명.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 클립 볼륨 동작 (별도 spec).
- 클립 속도 동작.
