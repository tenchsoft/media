# Spec: keyframe-interpolation

## 한 줄 정의
사용자가 키프레임 보간 방식을 변경하면 키프레임 간 애니메이션 곡선이 변경된다.

## 진입점
- 타임라인: 키프레임 우클릭 → 보간 방식

## 사용자 흐름
1. 사용자가 키프레임을 우클릭한다.
2. 보간 방식을 선택한다 (선형, 베지어, 홀드).
3. 키프레임 간 애니메이션 곡선이 변경된다.

## 성공 조건
- [ ] 보간 방식 선택 시 애니메이션 곡선이 변경된다.
- [ ] 선형 → 일정한 속도.
- [ ] 베지어 → 자연스러운 가감속.
- [ ] 홀드 → 다음 키프레임까지 값 유지.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 키프레임 추가 동작 (별도 spec).
- 키프레임 삭제 동작.
