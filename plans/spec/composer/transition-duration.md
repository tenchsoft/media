# Spec: transition-duration

## 한 줄 정의
사용자가 전환 지속 시간을 변경하면 전환 효과의 길이가 변경된다.

## 진입점
- 속성 패널: 전환 지속 시간 입력

## 사용자 흐름
1. 사용자가 전환 효과가 적용된 위치를 클릭한다.
2. 속성 패널에서 지속 시간을 입력한다.
3. 전환 효과의 길이가 변경된다.

## 성공 조건
- [ ] 지속 시간 입력 시 전환 길이가 변경된다.
- [ ] 타임라인에 반영된다.

## 실패 / 취소 흐름
- 잘못된 값 → 이전 값으로 복원된다.

## 경계 / 예외
- 인접 클립 길이를 초과할 수 없다.

## 범위 외
- 전환 적용 동작 (별도 spec).
- 전환 제거 동작.
