# Spec: delete-track-button

## 한 줄 정의
사용자가 트랙 삭제 버튼을 클릭하면 선택된 트랙이 삭제된다.

## 진입점
- 타임라인: 트랙 삭제 버튼 클릭

## 사용자 흐름
1. 사용자가 타임라인에서 트랙을 선택한다.
2. 트랙 삭제 버튼을 클릭한다.
3. 확인 대화상자가 표시된다.
4. 확인을 클릭하면 트랙과 모든 클립이 삭제된다.

## 성공 조건
- [ ] 확인 시 트랙과 모든 클립이 삭제된다.

## 실패 / 취소 흐름
- 취소 → 아무 일도 발생하지 않는다.

## 경계 / 예외
- 마지막 트랙 → 삭제할 수 없다.

## 범위 외
- 트랙 추가 동작 (별도 spec).
- 클립 삭제 동작.
