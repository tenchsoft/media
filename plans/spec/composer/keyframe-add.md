# Spec: keyframe-add

## 한 줄 정의
사용자가 키프레임 추가 버튼을 클릭하면 현재 시간에 키프레임이 추가된다.

## 진입점
- 속성 패널: 키프레임 추가 버튼 또는 타임라인에서 다이아몬드 클릭

## 사용자 흐름
1. 사용자가 재생 헤드를 원하는 위치로 이동한다.
2. 클립을 선택한다.
3. 속성 패널에서 키프레임 추가 버튼을 클릭한다.
4. 현재 속성 값으로 키프레임이 추가된다.

## 성공 조건
- [ ] 클릭 시 현재 시간에 키프레임이 추가된다.
- [ ] 타임라인에 다이아몬드 마커가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 같은 위치에 이미 키프레임이 있음 → 값이 업데이트된다.

## 범위 외
- 키프레임 삭제 동작 (별도 spec).
- 키프레임 보간 동작.
