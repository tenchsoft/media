# Spec: add-track-button

## 한 줄 정의
사용자가 트랙 추가 버튼을 클릭하면 타임라인에 새 트랙이 추가된다.

## 진입점
- 타임라인: 트랙 추가 버튼 클릭

## 사용자 흐름
1. 사용자가 타임라인에서 트랙 추가 버튼을 클릭한다.
2. 비디오 또는 오디오 트랙 유형을 선택한다.
3. 새 빈 트랙이 타임라인에 추가된다.

## 성공 조건
- [ ] 클릭 시 새 빈 트랙이 추가된다.
- [ ] 트랙 유형(비디오/오디오)을 선택할 수 있다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 트랙 삭제 동작 (별도 spec).
- 클립 추가 동작.
