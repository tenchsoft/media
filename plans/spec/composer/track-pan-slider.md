# Spec: track-pan-slider

## 한 줄 정의
사용자가 트랙 팬 슬라이더를 조정하면 해당 트랙의 좌우 밸런스가 변경된다.

## 진입점
- 타임라인: 트랙 헤더의 팬 슬라이더

## 사용자 흐름
1. 사용자가 트랙 헤더에서 팬 슬라이더를 드래그한다.
2. 해당 트랙의 좌우 밸런스가 실시간으로 변경된다.

## 성공 조건
- [ ] 슬라이더 드래그 시 좌우 밸런스가 실시간으로 변경된다.
- [ ] 중앙 → 좌우 동일.
- [ ] 좌/우 끝 → 해당 채널만 출력.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 트랙 볼륨 동작 (별도 spec).
- 클립 팬 동작.
