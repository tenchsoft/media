# Spec: clip-ripple-delete

## 한 줄 정의
사용자가 리플 삭제를 수행하면 클립이 제거되고 뒤의 클립들이 앞으로 당겨진다.

## 진입점
- 타임라인: 클립 우클릭 → 리플 삭제

## 사용자 흐름
1. 사용자가 타임라인에서 클립을 우클릭한다.
2. 리플 삭제를 선택한다.
3. 클립이 제거된다.
4. 뒤의 모든 클립이 앞으로 당겨진다.

## 성공 조건
- [ ] 클릭 시 클립이 제거된다.
- [ ] 뒤의 모든 클립이 앞으로 당겨진다.
- [ ] 클립 사이에 빈 공간이 남지 않는다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 일반 삭제 동작 (별도 spec).
- 클립 분할 동작.
