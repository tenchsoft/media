# Spec: track-lock-toggle

## 한 줄 정의
사용자가 트랙 잠금 버튼을 클릭하면 트랙이 잠기거나 잠금 해제된다.

## 진입점
- 타임라인: 트랙 헤더의 잠금 아이콘 클릭

## 사용자 흐름
1. 사용자가 트랙 헤더에서 잠금 아이콘을 클릭한다.
2. 트랙이 잠긴다 (편집 불가).
3. 다시 클릭하면 잠금이 해제된다.

## 성공 조건
- [ ] 클릭 시 트랙이 잠긴다.
- [ ] 잠긴 트랙의 클립은 편집할 수 없다.
- [ ] 다시 클릭 시 잠금이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 잠긴 트랙에 편집 시도 → "잠긴 트랙입니다" 토스트가 표시된다.

## 범위 외
- 트랙 가시성 동작 (별도 spec).
- 트랙 삭제 동작.
