# Spec: effects-panel-list

## 한 줄 정의
사용자가 효과 패널을 열면 사용 가능한 비디오/오디오 효과 목록이 표시된다.

## 진입점
- 효과 패널

## 사용자 흐름
1. 사용자가 효과 패널을 연다.
2. 카테고리별로 정리된 효과 목록이 표시된다.
3. 카테고리를 펼치면 개별 효과가 표시된다.

## 성공 조건
- [ ] 카테고리별로 정리된 효과 목록이 표시된다.
- [ ] 카테고리를 펼치면 개별 효과가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 효과 적용 동작 (별도 spec).
- 효과 설정 동작.
