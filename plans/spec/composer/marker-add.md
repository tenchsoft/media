# Spec: marker-add

## 한 줄 정의
사용자가 마커 추가 버튼을 클릭하면 재생 헤드 위치에 마커가 추가된다.

## 진입점
- 타임라인: 마커 추가 버튼 또는 M

## 사용자 흐름
1. 사용자가 재생 헤드를 원하는 위치로 이동한다.
2. 마커 추가 버튼을 클릭하거나 M 키를 누른다.
3. 재생 헤드 위치에 마커가 추가된다.

## 성공 조건
- [ ] 클릭 시 재생 헤드 위치에 마커가 추가된다.
- [ ] 타임라인 상단에 마커가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 마커 삭제 동작 (별도 spec).
- 마커 이름 편집 동작.
