# Spec: bottom-play-pause-button

## 한 줄 정의
사용자가 재생/일시정지 버튼을 누르면 재생 상태가 전환된다.

## 진입점
- 하단 컨트롤 바: 재생/일시정지 버튼 클릭

## 사용자 흐름
1. 사용자가 재생/일시정지 버튼을 클릭한다.
2. 재생 중이면 일시정지된다.
3. 일시정지 중이면 재생된다.
4. 버튼 아이콘이 상태에 맞게 변경된다.

## 성공 조건
- [ ] 클릭 시 재생/일시정지가 전환된다.
- [ ] 아이콘이 상태에 맞게 변경된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 파일이 열려 있지 않으면 버튼이 비활성화된다.

## 범위 외
- 스페이스바 단축키 동작 (별도 spec).
