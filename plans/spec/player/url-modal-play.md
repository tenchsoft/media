# Spec: url-modal-play

## 한 줄 정의
사용자가 URL 재생 대화상자에서 재생 버튼을 누르면 URL의 미디어가 재생된다.

## 진입점
- URL 재생 대화상자: 재생 버튼 클릭

## 사용자 흐름
1. 사용자가 URL을 입력한다.
2. 재생 버튼을 클릭한다.
3. URL의 미디어 스트리밍이 시작된다.
4. 대화상자가 닫힌다.

## 성공 조건
- [ ] 클릭 시 URL 미디어 재생이 시작된다.
- [ ] 대화상자가 닫힌다.

## 실패 / 취소 흐름
- 잘못된 URL → 오류 메시지가 표시된다.
- 네트워크 오류 → 연결 실패 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 파일 열기 동작 (별도 spec).
