# Spec: context-menu-stop-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 정지를 선택하면 재생이 정지된다.

## 진입점
- 우클릭 메뉴: 정지 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 정지 항목을 클릭한다.
3. 재생이 정지되고 파일 시작점으로 돌아간다.

## 성공 조건
- [ ] 클릭 시 재생이 정지된다.
- [ ] 재생 위치가 파일 시작점으로 돌아간다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.
