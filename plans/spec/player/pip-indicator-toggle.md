# Spec: pip-indicator-toggle

## 한 줄 정의
사용자가 PiP 버튼을 누르면 화면 속 화면 모드가 전환된다.

## 진입점
- 하단 컨트롤 바: PiP 버튼 클릭

## 사용자 흐름
1. 사용자가 PiP 버튼을 클릭한다.
2. 비디오가 작은 창으로 분리되어 항상 위에 표시된다.
3. 다른 애플리케이션을 사용하면서도 비디오를 볼 수 있다.
4. 다시 클릭하면 PiP 모드가 해제된다.

## 성공 조건
- [ ] 클릭 시 PiP 모드가 활성화된다.
- [ ] 작은 창이 항상 위에 표시된다.
- [ ] 다시 클릭 시 PiP 모드가 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 오디오 전용 파일은 PiP를 사용할 수 없다.

## 범위 외
- 전체 화면 동작 (별도 spec).
