# Spec: top-bar-playlist-drawer-tab

## 한 줄 정의
사용자가 상단 바 재생목록 탭을 누르면 재생목록 패널이 열린다.

## 진입점
- 상단 바: 재생목록 탭 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 재생목록 탭을 클릭한다.
2. 재생목록 패널이 열린다.
3. 파일을 추가/제거하고 재생 순서를 관리할 수 있다.

## 성공 조건
- [ ] 클릭 시 재생목록 패널이 열린다.
- [ ] 다시 클릭 시 패널이 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 재생목록 관리 동작 (별도 spec).
