# Spec: automatic-gif-recording-frame-capture

## 한 줄 정의
GIF 녹화 중 매 프레임이 자동으로 캡처된다.

## 진입점
- 자동: GIF 녹화 시작 후

## 사용자 흐름
1. 사용자가 GIF 녹화를 시작한다.
2. 재생 중 매 프레임이 자동으로 캡처된다.
3. 녹화 중지 시 캡처된 프레임이 GIF로 변환된다.

## 성공 조건
- [ ] 녹화 중 프레임이 자동 캡처된다.
- [ ] 녹화 완료 시 GIF가 생성된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작/중지 버튼 동작 (별도 spec).
- GIF 옵션 설정 동작.
