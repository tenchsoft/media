# Spec: subtitle-style-text-color-plus

## 한 줄 정의
사용자가 자막 텍스트 색상 + 버튼을 누르면 다음 색상으로 전환된다.

## 진입점
- 자막 스타일 대화상자: 텍스트 색상 + 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 스타일 대화상자를 연다.
2. 텍스트 색상 + 버튼을 클릭한다.
3. 미리 정의된 색상 목록에서 다음 색상으로 전환된다.
4. 미리보기에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 다음 색상으로 전환된다.
- [ ] 미리보기에 즉시 반영된다.

## 실패 / 취소 흐름
- 마지막 색상 → 첫 번째 색상으로 순환된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 텍스트 색상 - 버튼 동작 (별도 spec).
