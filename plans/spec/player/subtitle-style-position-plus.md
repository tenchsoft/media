# Spec: subtitle-style-position-plus

## 한 줄 정의
사용자가 자막 위치 + 버튼을 누르면 자막 표시 위치가 아래로 이동한다.

## 진입점
- 자막 스타일 대화상자: 위치 + 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 스타일 대화상자를 연다.
2. 위치 + 버튼을 클릭한다.
3. 자막 표시 위치가 화면 아래쪽으로 이동한다.
4. 미리보기에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 자막 위치가 아래로 이동한다.
- [ ] 미리보기에 즉시 반영된다.

## 실패 / 취소 흐름
- 최하단 도달 → 더 이상 이동하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 위치 - 버튼 동작 (별도 spec).
