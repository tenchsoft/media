# Spec: bottom-ab-loop-button

## 한 줄 정의
사용자가 AB 루프 버튼을 누르면 A-B 구간 반복 재생이 설정된다.

## 진입점
- 하단 컨트롤 바: AB 루프 버튼 클릭

## 사용자 흐름
1. 사용자가 AB 루프 버튼을 처음 클릭하면 A 지점이 설정된다.
2. 버튼을 다시 클릭하면 B 지점이 설정되고 A-B 구간 반복이 시작된다.
3. 세 번째 클릭하면 AB 루프가 해제된다.

## 성공 조건
- [ ] 첫 클릭 시 A 지점이 설정된다.
- [ ] 두 번째 클릭 시 B 지점이 설정되고 반복이 시작된다.
- [ ] 세 번째 클릭 시 루프가 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- A 지점 설정 후 시크바를 이동하면 A 지점은 유지된다.

## 범위 외
- AB 루프 재생 강제 동작 (별도 spec).
