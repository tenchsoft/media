# Spec: ai-panel-input-field

## 한 줄 정의
사용자가 AI 패널에서 자연어로 질문을 입력할 수 있다.

## 진입점
- AI 패널: 입력 필드 클릭/포커스

## 사용자 흐름
1. 사용자가 AI 패널을 연다.
2. 입력 필드에 질문을 입력한다.
3. 엔터 키를 누르거나 전송 버튼을 클릭한다.
4. AI가 응답을 생성하여 표시한다.

## 성공 조건
- [ ] 텍스트 입력이 실시간으로 반영된다.
- [ ] 엔터/전송 시 AI 요청이 전송된다.

## 실패 / 취소 흐름
- 빈 입력 → 전송되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- AI 요청 취소 동작 (별도 spec).
- AI 기능 버튼 동작.
