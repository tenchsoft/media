# Spec: gif-options-modal-start-recording

## 한 줄 정의
사용자가 GIF 옵션 대화상자에서 녹화 시작 버튼을 누르면 설정된 옵션으로 GIF 녹화가 시작된다.

## 진입점
- GIF 옵션 대화상자: 녹화 시작 버튼 클릭

## 사용자 흐름
1. 사용자가 GIF 옵션 대화상자를 연다.
2. 해상도, FPS 등 옵션을 설정한다.
3. 녹화 시작 버튼을 클릭한다.
4. 설정된 옵션으로 GIF 녹화가 시작된다.

## 성공 조건
- [ ] 클릭 시 설정된 옵션으로 GIF 녹화가 시작된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 캡처 모달 동작 (별도 spec).
