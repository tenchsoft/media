# Spec: bottom-repeat-mode-button

## 한 줄 정의
사용자가 반복 모드 버튼을 누르면 반복 재생 모드가 전환된다.

## 진입점
- 하단 컨트롤 바: 반복 모드 버튼 클릭

## 사용자 흐름
1. 사용자가 반복 모드 버튼을 클릭한다.
2. 반복 모드가 전환된다 (반복 없음 → 전체 반복 → 1곡 반복).
3. 현재 모드가 버튼 아이콘에 반영된다.

## 성공 조건
- [ ] 클릭 시 반복 모드가 순환 전환된다.
- [ ] 현재 모드가 아이콘에 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 셔플 모드 동작 (별도 spec).
- AB 루프 동작.
