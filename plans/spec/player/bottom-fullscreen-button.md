# Spec: bottom-fullscreen-button

## 한 줄 정의
사용자가 전체 화면 버튼을 누르면 전체 화면 모드로 전환된다.

## 진입점
- 하단 컨트롤 바: 전체 화면 버튼 클릭

## 사용자 흐름
1. 사용자가 전체 화면 버튼을 클릭한다.
2. 플레이어가 전체 화면 모드로 전환된다.
3. 컨트롤 바는 마우스 이동 시에만 표시된다.
4. 다시 클릭하거나 Esc 키로 전체 화면을 해제한다.

## 성공 조건
- [ ] 클릭 시 전체 화면 모드로 전환된다.
- [ ] Esc 키로 전체 화면이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외

- PiP 모드 동작.
