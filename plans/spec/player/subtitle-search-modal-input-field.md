# Spec: subtitle-search-modal-input-field

## 한 줄 정의
사용자가 자막 검색 대화상자에서 검색어를 입력할 수 있다.

## 진입점
- 자막 검색 대화상자: 입력 필드 클릭/포커스

## 사용자 흐름
1. 사용자가 자막 검색 대화상자를 연다.
2. 입력 필드에 검색어를 입력한다.
3. 입력한 텍스트가 실시간으로 표시된다.
4. 엔터 키를 누르면 검색이 실행된다.

## 성공 조건
- [ ] 텍스트 입력이 실시간으로 반영된다.
- [ ] 엔터 시 검색이 실행된다.

## 실패 / 취소 흐름
- 빈 입력 → 검색이 실행되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자막 검색 결과 동작 (별도 spec).
