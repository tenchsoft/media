# Spec: builtin-subtitle-none-button

## 한 줄 정의
사용자가 자막 없음을 선택하면 내장 자막이 표시되지 않는다.

## 진입점
- 자막 선택 메뉴: 자막 없음 항목 클릭

## 사용자 흐름
1. 사용자가 자막 선택 메뉴를 연다.
2. 자막 없음을 선택한다.
3. 내장 자막이 화면에서 사라진다.

## 성공 조건
- [ ] 선택 시 내장 자막이 표시되지 않는다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 외부 자막은 영향을 받지 않는다.

## 범위 외
- 외부 자막 선택 동작 (별도 spec).
- 자막 스타일 동작.
