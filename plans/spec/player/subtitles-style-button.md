# Spec: subtitles-style-button

## 한 줄 정의
사용자가 자막 스타일 버튼을 누르면 자막 스타일 설정 대화상자가 열린다.

## 진입점
- 자막 패널: 스타일 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 패널을 연다.
2. 스타일 버튼을 클릭한다.
3. 자막 스타일 설정 대화상자가 열린다.
4. 글꼴, 크기, 색상, 배경 등을 변경할 수 있다.

## 성공 조건
- [ ] 클릭 시 자막 스타일 대화상자가 열린다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자막 스타일 개별 설정 동작 (별도 spec).
