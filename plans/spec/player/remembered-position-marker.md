# Spec: remembered-position-marker

## 한 줄 정의
사용자가 이전에 재생하던 위치를 표시하는 마커를 클릭하면 해당 위치로 이동한다.

## 진입점
- 시크바: 기억된 위치 마커 클릭

## 사용자 흐름
1. 파일을 열면 이전에 재생하던 위치 마커가 시크바에 표시된다.
2. 마커를 클릭한다.
3. 재생 위치가 기억된 위치로 이동한다.

## 성공 조건
- [ ] 파일 열기 시 이전 위치 마커가 표시된다.
- [ ] 마커 클릭 시 해당 위치로 이동한다.

## 실패 / 취소 흐름
- 처음 재생하는 파일은 마커가 표시되지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 시크바 수동 조작 동작 (별도 spec).
