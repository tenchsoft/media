# Spec: equalizer-modal-close

## 한 줄 정의
사용자가 이퀄라이저 닫기 버튼을 누르면 이퀄라이저 대화상자가 닫힌다.

## 진입점
- 이퀄라이저 대화상자: 닫기 버튼 클릭

## 사용자 흐름
1. 사용자가 이퀄라이저 대화상자를 연다.
2. 닫기 버튼을 클릭한다.
3. 대화상자가 닫힌다.
4. 설정된 이퀄라이저 값은 유지된다.

## 성공 조건
- [ ] 클릭 시 대화상자가 닫힌다.
- [ ] 이퀄라이저 설정이 유지된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 이퀄라이저 밴드 조작 동작 (별도 spec).
