# Spec: bottom-speed-menu-toggle-button

## 한 줄 정의
사용자가 속도 메뉴 버튼을 누르면 재생 속도 선택 메뉴가 표시된다.

## 진입점
- 하단 컨트롤 바: 속도 메뉴 버튼 클릭

## 사용자 흐름
1. 사용자가 속도 메뉴 버튼을 클릭한다.
2. 속도 선택 메뉴가 표시된다.
3. 원하는 속도를 선택한다.
4. 재생 속도가 변경된다.
5. 메뉴가 닫힌다.

## 성공 조건
- [ ] 클릭 시 속도 선택 메뉴가 표시된다.
- [ ] 속도 선택 시 재생 속도가 변경된다.

## 실패 / 취소 흐름
- 메뉴 외부 클릭 → 메뉴가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 속도 옵션 동작 (별도 spec).
