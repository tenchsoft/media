# Spec: subtitles-search-button

## 한 줄 정의
사용자가 자막 검색 버튼을 누르면 자막 검색 대화상자가 열린다.

## 진입점
- 자막 패널: 검색 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 패널을 연다.
2. 검색 버튼을 클릭한다.
3. 자막 검색 대화상자가 열린다.

## 성공 조건
- [ ] 클릭 시 자막 검색 대화상자가 열린다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 자막이 없으면 검색 버튼이 비활성화된다.

## 범위 외
- 자막 검색 동작 (별도 spec).
