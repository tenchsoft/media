# Spec: context-menu-show-in-files-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 파일 탐색기에서 보기를 선택하면 파일 위치가 열린다.

## 진입점
- 우클릭 메뉴: 파일 탐색기에서 보기 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 파일 탐색기에서 보기를 클릭한다.
3. 시스템 파일 탐색기가 열리고 현재 파일 위치가 표시된다.

## 성공 조건
- [ ] 클릭 시 파일 탐색기가 열린다.
- [ ] 현재 재생 중인 파일이 선택된 상태로 표시된다.

## 실패 / 취소 흐름
- 파일을 찾을 수 없음 → 오류 메시지가 표시된다.

## 경계 / 예외
- URL 재생 중에는 비활성화된다.

## 범위 외
- 파일 열기 동작 (별도 spec).
