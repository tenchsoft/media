# Spec: subtitle-encoding-option

## 한 줄 정의
사용자가 자막 인코딩을 선택하면 자막이 해당 인코딩으로 다시 해석된다.

## 진입점
- 자막 설정: 인코딩 옵션 클릭

## 사용자 흐름
1. 사용자가 자막 설정을 연다.
2. 인코딩 옵션(자동, UTF-8, EUC-KR, Shift-JIS, CP1252)을 선택한다.
3. 자막이 선택한 인코딩으로 다시 해석된다.
4. 깨진 문자가 올바르게 표시된다.

## 성공 조건
- [ ] 선택 시 자막이 해당 인코딩으로 다시 해석된다.
- [ ] 문자가 올바르게 표시된다.

## 실패 / 취소 흐름
- 잘못된 인코딩 → 문자가 깨질 수 있다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자막 스타일 동작 (별도 spec).
- 자막 싱크 조정 동작.
