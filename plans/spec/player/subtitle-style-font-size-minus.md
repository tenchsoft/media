# Spec: subtitle-style-font-size-minus

## 한 줄 정의
사용자가 자막 글꼴 크기 - 버튼을 누르면 자막 글꼴 크기가 감소한다.

## 진입점
- 자막 스타일 대화상자: 글꼴 크기 - 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 스타일 대화상자를 연다.
2. 글꼴 크기 - 버튼을 클릭한다.
3. 자막 글꼴 크기가 한 단계 감소한다.
4. 변경이 즉시 미리보기에 반영된다.

## 성공 조건
- [ ] 클릭 시 글꼴 크기가 감소한다.
- [ ] 미리보기에 즉시 반영된다.

## 실패 / 취소 흐름
- 최소 크기 도달 → 더 이상 감소하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 글꼴 크기 + 버튼 동작 (별도 spec).
