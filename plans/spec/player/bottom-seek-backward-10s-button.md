# Spec: bottom-seek-backward-10s-button

## 한 줄 정의
사용자가 10초 뒤로 버튼을 누르면 재생 위치가 10초 뒤로 이동한다.

## 진입점
- 하단 컨트롤 바: 10초 뒤로 버튼 클릭

## 사용자 흐름
1. 사용자가 10초 뒤로 버튼을 클릭한다.
2. 재생 위치가 10초 뒤로 이동한다.
3. 이동한 위치에서 재생이 계속된다.

## 성공 조건
- [ ] 클릭 시 재생 위치가 10초 뒤로 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 파일 시작점에 가까우면 파일 시작점으로 이동한다.

## 범위 외
- 10초 앞으로 동작 (별도 spec).
- 시크바 조작 동작.
