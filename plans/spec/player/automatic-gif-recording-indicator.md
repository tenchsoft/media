# Spec: automatic-gif-recording-indicator

## 한 줄 정의
GIF 녹화 중임을 나타내는 표시가 자동으로 나타난다.

## 진입점
- 자동: GIF 녹화 시작 시

## 사용자 흐름
1. 사용자가 GIF 녹화를 시작한다.
2. 화면에 녹화 중 표시(빨간 점 등)가 나타난다.
3. 녹화가 끝나면 표시가 사라진다.

## 성공 조건
- [ ] 녹화 중 표시가 나타난다.
- [ ] 녹화 종료 시 표시가 사라진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작/중지 동작 (별도 spec).
