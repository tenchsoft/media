# Spec: external-subtitle-track-row

## 한 줄 정의
사용자가 외부 자막 트랙을 선택하면 해당 자막이 표시된다.

## 진입점
- 자막 선택 메뉴: 외부 자막 트랙 항목 클릭

## 사용자 흐름
1. 사용자가 자막 선택 메뉴를 연다.
2. 불러온 외부 자막 목록이 표시된다.
3. 원하는 자막을 선택한다.
4. 해당 자막이 화면에 표시된다.

## 성공 조건
- [ ] 선택 시 해당 외부 자막이 표시된다.
- [ ] 이전 자막이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 외부 자막이 없으면 목록이 비어 있다.

## 범위 외
- 자막 불러오기 동작 (별도 spec).
- 내장 자막 동작.
