# Spec: gif-options-modal-cancel

## 한 줄 정의
사용자가 GIF 옵션 대화상자에서 취소 버튼을 누르면 GIF 생성이 취소된다.

## 진입점
- GIF 옵션 대화상자: 취소 버튼 클릭

## 사용자 흐름
1. GIF 녹화가 완료되면 옵션 대화상자가 나타난다.
2. 사용자가 취소 버튼을 클릭한다.
3. GIF 생성이 취소된다.
4. 대화상자가 닫힌다.

## 성공 조건
- [ ] 클릭 시 GIF 생성이 취소된다.
- [ ] 대화상자가 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작 동작 (별도 spec).
