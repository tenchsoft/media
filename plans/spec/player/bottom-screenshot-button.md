# Spec: bottom-screenshot-button

## 한 줄 정의
사용자가 스크린샷 버튼을 누르면 현재 프레임이 이미지로 저장된다.

## 진입점
- 하단 컨트롤 바: 스크린샷 버튼 클릭

## 사용자 흐름
1. 사용자가 스크린샷 버튼을 클릭한다.
2. 현재 프레임이 이미지 파일로 저장된다.
3. 저장 완료 알림이 표시된다.

## 성공 조건
- [ ] 클릭 시 현재 프레임이 이미지로 저장된다.
- [ ] 저장 완료 알림이 표시된다.

## 실패 / 취소 흐름
- 저장 실패 → 오류 알림이 표시된다.

## 경계 / 예외
- 오디오 전용 파일은 스크린샷을 찍을 수 없다.

## 범위 외
- 스크린샷 저장 경로 설정 동작 (별도 spec).
