# Spec: bottom-mute-toggle-button

## 한 줄 정의
사용자가 음소거 버튼을 누르면 오디오가 음소거/해제된다.

## 진입점
- 하단 컨트롤 바: 음소거 버튼 클릭

## 사용자 흐름
1. 사용자가 음소거 버튼을 클릭한다.
2. 오디오가 음소거된다.
3. 버튼 아이콘이 음소거 상태로 변경된다.
4. 다시 클릭하면 음소거가 해제된다.

## 성공 조건
- [ ] 클릭 시 음소거/해제가 전환된다.
- [ ] 아이콘이 상태에 맞게 변경된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 음소거 해제 시 이전 볼륨으로 복원된다.

## 범위 외
- 볼륨 슬라이더 동작 (별도 spec).
