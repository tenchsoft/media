# Spec: chapter-row-jump-control

## 한 줄 정의
사용자가 챕터 행을 클릭하면 해당 챕터 위치로 재생이 이동한다.

## 진입점
- 챕터 패널: 챕터 행 클릭

## 사용자 흐름
1. 사용자가 챕터 패널을 연다.
2. 원하는 챕터 행을 클릭한다.
3. 재생 위치가 해당 챕터 시작점으로 이동한다.

## 성공 조건
- [ ] 클릭 시 해당 챕터 시작점으로 이동한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 챕터 삭제 동작 (별도 spec).
- 챕터 이름 변경 동작.
