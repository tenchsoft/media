# Spec: ai-feature-generate-chapter-marks

## 한 줄 정의
사용자가 AI를 통해 자동으로 챕터 마크를 생성할 수 있다.

## 진입점
- AI 기능 패널: 챕터 자동 생성 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 기능 패널을 연다.
2. 챕터 자동 생성 버튼을 클릭한다.
3. AI가 영상을 분석하여 장면 전환점을 기준으로 챕터를 자동 생성한다.
4. 생성된 챕터 목록이 표시된다.

## 성공 조건
- [ ] 버튼 클릭 시 AI가 영상을 분석한다.
- [ ] 장면 전환점에 챕터가 자동 생성된다.
- [ ] 챕터 목록에 반영된다.

## 실패 / 취소 흐름
- 분석 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 짧은 영상은 챕터가 생성되지 않을 수 있다.

## 범위 외
- 수동 챕터 추가 동작 (별도 spec).
- 챕터 내보내기 동작.
