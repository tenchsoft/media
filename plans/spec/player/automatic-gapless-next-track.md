# Spec: automatic-gapless-next-track

## 한 줄 정의
현재 트랙 재생이 끝나면 다음 트랙이 끊김 없이 자동 재생된다.

## 진입점
- 자동: 현재 트랙 재생 완료 시

## 사용자 흐름
1. 사용자가 재생목록의 트랙을 재생한다.
2. 트랙이 끝나면 다음 트랙이 끊김 없이 재생된다.
3. 재생목록의 마지막 트랙이면 정지된다.

## 성공 조건
- [ ] 트랙 완료 시 다음 트랙이 자동 재생된다.
- [ ] 곡 간 간격 없이 끊김 없이 재생된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 반복 모드가 켜져 있으면 다음 트랙 대신 현재 트랙이 반복된다.

## 범위 외
- 반복 모드 동작 (별도 spec).
- 셔플 모드 동작.
