# Spec: automatic-empty-state-drop-prompt

## 한 줄 정의
파일이 열려 있지 않을 때 드롭 안내 메시지가 자동으로 표시된다.

## 진입점
- 자동: 파일이 열려 있지 않을 때

## 사용자 흐름
1. 플레이어를 실행한다.
2. 파일이 열려 있지 않으면 빈 화면에 드롭 안내 메시지가 표시된다.
3. 파일을 드래그 앤 드롭하면 재생이 시작된다.

## 성공 조건
- [ ] 파일이 없을 때 드롭 안내가 표시된다.
- [ ] 파일 드롭 시 재생이 시작된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 지원하지 않는 형식 드롭 → 오류 메시지가 표시된다.

## 범위 외
- 파일 열기 버튼 동작 (별도 spec).
- URL 재생 동작.
