# Spec: playlist-row-remove

## 한 줄 정의
사용자가 재생목록 행의 제거 버튼을 누르면 해당 파일이 재생목록에서 제거된다.

## 진입점
- 재생목록 패널: 파일 행의 제거 버튼 클릭

## 사용자 흐름
1. 사용자가 재생목록 패널을 연다.
2. 제거할 파일 행의 제거 버튼을 클릭한다.
3. 해당 파일이 재생목록에서 제거된다.

## 성공 조건
- [ ] 클릭 시 파일이 재생목록에서 제거된다.
- [ ] 현재 재생 중인 파일은 제거 후 다음 파일로 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 마지막 파일이 제거되면 빈 재생목록이 표시된다.

## 범위 외
- 재생목록 파일 추가 동작 (별도 spec).
