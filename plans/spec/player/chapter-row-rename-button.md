# Spec: chapter-row-rename-button

## 한 줄 정의
사용자가 챕터 이름 변경 버튼을 누르면 챕터 이름을 편집할 수 있다.

## 진입점
- 챕터 패널: 챕터 행의 이름 변경 버튼 클릭

## 사용자 흐름
1. 사용자가 챕터 행의 이름 변경 버튼을 클릭한다.
2. 챕터 이름이 편집 가능한 상태가 된다.
3. 새 이름을 입력한다.
4. 엔터 키를 누르거나 다른 곳을 클릭하면 이름이 저장된다.

## 성공 조건
- [ ] 클릭 시 이름이 편집 가능해진다.
- [ ] 새 이름이 저장된다.

## 실패 / 취소 흐름
- Esc 키 → 편집이 취소된다.

## 경계 / 예외
- 빈 이름은 허용되지 않는다.

## 범위 외
- 챕터 추가 동작 (별도 spec).
- 챕터 삭제 동작.
