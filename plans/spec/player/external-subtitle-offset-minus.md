# Spec: external-subtitle-offset-minus

## 한 줄 정의
사용자가 자막 싱크 - 버튼을 누르면 자막 타이밍이 늦춰진다.

## 진입점
- 자막 설정: 싱크 - 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 설정을 연다.
2. 싱크 - 버튼을 클릭한다.
3. 자막 타이밍이 늦춰진다 (예: 0.5초 늦게 표시).
4. 변경이 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 자막 타이밍이 늦춰진다.
- [ ] 변경이 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자막 싱크 + 버튼 동작 (별도 spec).
