# Spec: chapter-row-delete-button

## 한 줄 정의
사용자가 챕터 삭제 버튼을 누르면 해당 챕터가 제거된다.

## 진입점
- 챕터 패널: 챕터 행의 삭제 버튼 클릭

## 사용자 흐름
1. 사용자가 챕터 행의 삭제 버튼을 클릭한다.
2. 해당 챕터가 목록에서 제거된다.
3. 시크바의 챕터 마크도 함께 제거된다.

## 성공 조건
- [ ] 클릭 시 챕터가 목록에서 제거된다.
- [ ] 시크바 마크도 함께 제거된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 챕터 추가 동작 (별도 spec).
- 챕터 이름 변경 동작.
