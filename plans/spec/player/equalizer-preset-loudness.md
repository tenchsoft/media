# Spec: equalizer-preset-loudness

## 한 줄 정의
사용자가 라우드니스 프리셋을 선택하면 저음과 고음이 강조된다.

## 진입점
- 이퀄라이저 패널: 라우드니스 프리셋 클릭

## 사용자 흐름
1. 사용자가 이퀄라이저 패널을 연다.
2. 라우드니스 프리셋을 클릭한다.
3. 저음과 고음 대역이 강조된다.
4. 오디오에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 저음/고음 대역 게인이 프리셋 값으로 설정된다.
- [ ] 오디오에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.
