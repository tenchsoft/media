# Spec: subtitle-style-shadow-offset-plus

## 한 줄 정의
사용자가 자막 그림자 오프셋 + 버튼을 누르면 그림자 오프셋이 증가한다.

## 진입점
- 자막 스타일 대화상자: 그림자 오프셋 + 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 스타일 대화상자를 연다.
2. 그림자 오프셋 + 버튼을 클릭한다.
3. 그림자 오프셋이 증가한다.
4. 미리보기에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 그림자 오프셋이 증가한다.
- [ ] 미리보기에 즉시 반영된다.

## 실패 / 취소 흐름
- 최대 오프셋 도달 → 더 이상 증가하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 그림자 오프셋 - 버튼 동작 (별도 spec).
