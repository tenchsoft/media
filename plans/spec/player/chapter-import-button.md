# Spec: chapter-import-button

## 한 줄 정의
사용자가 챕터 가져오기 버튼을 누르면 외부 챕터 파일을 불러올 수 있다.

## 진입점
- 챕터 패널: 가져오기 버튼 클릭

## 사용자 흐름
1. 사용자가 챕터 패널을 연다.
2. 가져오기 버튼을 클릭한다.
3. 파일 선택 대화상자가 나타난다.
4. 챕터 파일을 선택한다.
5. 챕터가 목록에 추가된다.

## 성공 조건
- [ ] 클릭 시 파일 선택 대화상자가 표시된다.
- [ ] 선택한 파일의 챕터가 목록에 추가된다.

## 실패 / 취소 흐름
- 잘못된 형식 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 챕터 내보내기 동작 (별도 spec).
