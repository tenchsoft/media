# Spec: context-menu-screenshot-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 스크린샷을 선택하면 현재 프레임이 저장된다.

## 진입점
- 우클릭 메뉴: 스크린샷 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 스크린샷 항목을 클릭한다.
3. 현재 프레임이 이미지로 저장된다.

## 성공 조건
- [ ] 클릭 시 현재 프레임이 이미지로 저장된다.

## 실패 / 취소 흐름
- 저장 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 오디오 전용 파일에서는 비활성화된다.

## 범위 외
- 하단 바 스크린샷 버튼 동작 (별도 spec).
