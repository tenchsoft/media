# Spec: seekbar-position

## 한 줄 정의
사용자가 시크바를 클릭/드래그하면 재생 위치가 해당 위치로 이동한다.

## 진입점
- 하단 컨트롤 바: 시크바 클릭/드래그

## 사용자 흐름
1. 사용자가 시크바를 클릭한다.
2. 재생 위치가 클릭한 위치로 즉시 이동한다.
3. 드래그하면 실시간으로 위치가 갱신된다.
4. 드래그를 놓으면 해당 위치에서 재생이 계속된다.

## 성공 조건
- [ ] 클릭 시 해당 위치로 즉시 이동한다.
- [ ] 드래그 중 실시간으로 위치가 갱신된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 썸네일 미리보기 동작 (별도 spec).
- 버퍼링 진행률 동작.
