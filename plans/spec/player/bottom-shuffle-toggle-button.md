# Spec: bottom-shuffle-toggle-button

## 한 줄 정의
사용자가 셔플 버튼을 누르면 재생목록 순서가 무작위로 재생된다.

## 진입점
- 하단 컨트롤 바: 셔플 버튼 클릭

## 사용자 흐름
1. 사용자가 셔플 버튼을 클릭한다.
2. 셔플 모드가 활성화된다.
3. 다음 트랙이 무작위로 선택된다.
4. 다시 클릭하면 셔플이 해제된다.

## 성공 조건
- [ ] 클릭 시 셔플 모드가 전환된다.
- [ ] 셔플 활성화 시 다음 트랙이 무작위로 선택된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외

- 재생목록 관리 동작.
