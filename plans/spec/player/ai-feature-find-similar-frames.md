# Spec: ai-feature-find-similar-frames

## 한 줄 정의
사용자가 현재 프레임과 유사한 장면을 AI가 검색하도록 요청할 수 있다.

## 진입점
- AI 기능 패널: 유사 프레임 검색 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 기능 패널을 연다.
2. 유사 프레임 검색 버튼을 클릭한다.
3. AI가 현재 프레임과 유사한 장면을 검색한다.
4. 검색 결과가 썸네일 목록으로 표시된다.
5. 썸네일 클릭 시 해당 위치로 이동한다.

## 성공 조건
- [ ] 버튼 클릭 시 유사 프레임 검색이 시작된다.
- [ ] 결과가 썸네일 목록으로 표시된다.
- [ ] 썸네일 클릭 시 해당 위치로 이동한다.

## 실패 / 취소 흐름
- 검색 결과 없음 → 결과 없음 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 대화 설명 동작 (별도 spec).
- 챕터 자동 생성 동작.
