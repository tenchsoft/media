# Spec: equalizer-250hz-minus

## 한 줄 정의
사용자가 250Hz - 버튼을 누르면 250Hz 대역 게인이 감소한다.

## 진입점
- 이퀄라이저 패널: 250Hz - 버튼 클릭

## 사용자 흐름
1. 사용자가 이퀄라이저 패널을 연다.
2. 250Hz - 버튼을 클릭한다.
3. 250Hz 대역 게인이 한 단계 감소한다.
4. 오디오 출력에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 250Hz 게인이 감소한다.
- [ ] 오디오에 즉시 반영된다.

## 실패 / 취소 흐름
- 최소 게인 도달 → 더 이상 감소하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 250Hz + 버튼 동작 (별도 spec).
