# Spec: ai-feature-explain-dialogue

## 한 줄 정의
사용자가 현재 장면의 대화를 AI가 설명하도록 요청할 수 있다.

## 진입점
- AI 기능 패널: 대화 설명 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 기능 패널을 연다.
2. 대화 설명 버튼을 클릭한다.
3. AI가 현재 재생 중인 장면의 대화 내용을 분석하여 설명을 표시한다.

## 성공 조건
- [ ] 버튼 클릭 시 AI가 대화를 분석한다.
- [ ] 설명 결과가 패널에 표시된다.

## 실패 / 취소 흐름
- AI 응답 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 오디오 트랙이 없으면 → 분석 불가 메시지가 표시된다.

## 범위 외
- AI 패널 입력 동작 (별도 spec).
- 유사 프레임 찾기 동작.
