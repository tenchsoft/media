# Spec: gif-capture-modal-close

## 한 줄 정의
사용자가 GIF 캡처 대화상자에서 닫기 버튼을 누르면 대화상자가 닫힌다.

## 진입점
- GIF 캡처 대화상자: 닫기 버튼 클릭

## 사용자 흐름
1. 사용자가 GIF 캡처 대화상자를 연다.
2. 닫기 버튼을 클릭한다.
3. 대화상자가 닫힌다.
4. 녹화 중이면 녹화도 중단된다.

## 성공 조건
- [ ] 클릭 시 대화상자가 닫힌다.
- [ ] 진행 중인 녹화가 중단된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작/중지 동작 (별도 spec).
