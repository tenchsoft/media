# Spec: context-menu-fullscreen-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 전체 화면을 선택하면 전체 화면 모드로 전환된다.

## 진입점
- 우클릭 메뉴: 전체 화면 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 전체 화면 항목을 클릭한다.
3. 전체 화면 모드로 전환된다.

## 성공 조건
- [ ] 클릭 시 전체 화면 모드로 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 하단 바 전체 화면 버튼 동작 (별도 spec).
