# Spec: playlist-add-files-button

## 한 줄 정의
사용자가 재생목록 추가 버튼을 누르면 파일을 재생목록에 추가할 수 있다.

## 진입점
- 재생목록 패널: 파일 추가 버튼 클릭

## 사용자 흐름
1. 사용자가 재생목록 패널을 연다.
2. 파일 추가 버튼을 클릭한다.
3. 파일 선택 대화상자가 나타난다.
4. 파일을 선택한다.
5. 선택한 파일이 재생목록에 추가된다.

## 성공 조건
- [ ] 클릭 시 파일 선택 대화상자가 표시된다.
- [ ] 선택한 파일이 재생목록에 추가된다.

## 실패 / 취소 흐름
- 파일 선택 취소 → 아무 일도 일어나지 않는다.

## 경계 / 예외
- 지원하지 않는 형식 → 추가되지 않고 오류 메시지가 표시된다.

## 범위 외
- 재생목록에서 파일 제거 동작 (별도 spec).
