# Spec: automatic-side-panel-layout-reserve

## 한 줄 정의
사이드 패널 열림/닫힘에 따라 비디오 영역 레이아웃이 자동으로 조정된다.

## 진입점
- 자동: 사이드 패널 열림/닫힘 시

## 사용자 흐름
1. 사용자가 사이드 패널을 연다.
2. 비디오 영역이 패널 너비만큼 축소된다.
3. 패널을 닫으면 비디오 영역이 원래 크기로 복원된다.

## 성공 조건
- [ ] 패널 열림 시 비디오 영역이 축소된다.
- [ ] 패널 닫힘 시 비디오 영역이 복원된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 전체 화면 모드에서는 패널이 오버레이로 표시된다.

## 범위 외
- 개별 패널 내용 동작 (별도 spec).
