# Spec: automatic-toast-lifecycle

## 한 줄 정의
알림 메시지가 표시된 후 일정 시간이 지나면 자동으로 사라진다.

## 진입점
- 자동: 알림 메시지 표시 후

## 사용자 흐름
1. 사용자 작업으로 알림 메시지가 표시된다.
2. 일정 시간(예: 3초) 후 알림이 자동으로 사라진다.

## 성공 조건
- [ ] 알림이 일정 시간 후 자동으로 사라진다.
- [ ] 새 알릘이 이전 알림 위에 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 알림을 유발하는 동작 (별도 spec).
