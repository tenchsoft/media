# Spec: context-menu-play-pause-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 재생/일시정지를 선택하면 재생 상태가 전환된다.

## 진입점
- 우클릭 메뉴: 재생/일시정지 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 재생 또는 일시정지 항목을 클릭한다.
3. 재생 상태가 전환된다.

## 성공 조건
- [ ] 클릭 시 재생/일시정지가 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 파일이 열려 있지 않으면 항목이 비활성화된다.

## 범위 외
- 하단 바 재생/일시정지 버튼 동작 (별도 spec).
