# Spec: ai-panel-cancel-request

## 한 줄 정의
사용자가 AI 패널에서 진행 중인 요청을 취소할 수 있다.

## 진입점
- AI 패널: 취소 버튼 클릭

## 사용자 흐름
1. AI 요청이 진행 중일 때 취소 버튼이 표시된다.
2. 사용자가 취소 버튼을 클릭한다.
3. 요청이 즉시 중단된다.
4. 이전 결과가 사라지고 초기 상태로 돌아간다.

## 성공 조건
- [ ] 취소 버튼 클릭 시 요청이 즉시 중단된다.
- [ ] 패널이 초기 상태로 돌아간다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 이미 완료된 요청은 취소할 수 없다.

## 범위 외
- AI 패널 입력 동작 (별도 spec).
