# Spec: playlist-row-play

## 한 줄 정의
사용자가 재생목록 행을 더블클릭하면 해당 파일이 재생된다.

## 진입점
- 재생목록 패널: 파일 행 더블클릭

## 사용자 흐름
1. 사용자가 재생목록 패널을 연다.
2. 재생할 파일 행을 더블클릭한다.
3. 해당 파일이 재생된다.

## 성공 조건
- [ ] 더블클릭 시 해당 파일이 재생된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 재생목록 파일 추가 동작 (별도 spec).
- 재생목록 파일 제거 동작.
