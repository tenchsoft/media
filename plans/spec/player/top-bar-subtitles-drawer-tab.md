# Spec: top-bar-subtitles-drawer-tab

## 한 줄 정의
사용자가 상단 바 자막 탭을 누르면 자막 패널이 열린다.

## 진입점
- 상단 바: 자막 탭 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 자막 탭을 클릭한다.
2. 자막 패널이 열린다.
3. 자막 트랙 선택, 검색, 스타일 설정을 할 수 있다.

## 성공 조건
- [ ] 클릭 시 자막 패널이 열린다.
- [ ] 다시 클릭 시 패널이 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 자막 선택/검색/스타일 동작 (별도 spec).
