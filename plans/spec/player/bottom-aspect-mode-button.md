# Spec: bottom-aspect-mode-button

## 한 줄 정의
사용자가 화면 비율 버튼을 누르면 비디오 화면 비율 모드가 전환된다.

## 진입점
- 하단 컨트롤 바: 화면 비율 버튼 클릭

## 사용자 흐름
1. 사용자가 화면 비율 버튼을 클릭한다.
2. 화면 비율 모드가 전환된다 (원본 비율 ↔ 16:9 ↔ 4:3 ↔ 늘리기).
3. 비디오 화면이 선택된 비율에 맞게 조정된다.

## 성공 조건
- [ ] 클릭 시 화면 비율 모드가 전환된다.
- [ ] 비디오가 선택된 비율에 맞게 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 전체 화면 동작 (별도 spec).
