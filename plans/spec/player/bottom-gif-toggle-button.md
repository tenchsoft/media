# Spec: bottom-gif-toggle-button

## 한 줄 정의
사용자가 GIF 버튼을 누르면 GIF 녹화 모드가 활성화된다.

## 진입점
- 하단 컨트롤 바: GIF 버튼 클릭

## 사용자 흐름
1. 사용자가 GIF 버튼을 클릭한다.
2. GIF 녹화 모드가 활성화된다.
3. 시작/중지 버튼으로 녹화를 제어할 수 있다.

## 성공 조건
- [ ] 클릭 시 GIF 녹화 모드가 활성화된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작/중지 동작 (별도 spec).
- GIF 옵션 설정 동작.
