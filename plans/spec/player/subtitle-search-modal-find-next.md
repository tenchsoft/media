# Spec: subtitle-search-modal-find-next

## 한 줄 정의
사용자가 다음 찾기 버튼을 누르면 다음 일치 자막으로 이동한다.

## 진입점
- 자막 검색 대화상자: 다음 찾기 버튼 클릭

## 사용자 흐름
1. 사용자가 자막 검색 대화상자에서 검색을 실행한다.
2. 다음 찾기 버튼을 클릭한다.
3. 다음 일치하는 자막으로 재생 위치가 이동한다.
4. 해당 자막이 강조된다.

## 성공 조건
- [ ] 클릭 시 다음 일치 자막으로 이동한다.
- [ ] 일치하는 자막이 강조된다.

## 실패 / 취소 흐름
- 더 이상 일치하는 자막이 없으면 → 마지막이라는 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 이전 찾기 동작 (별도 spec).
