# Spec: equalizer-preset-flat

## 한 줄 정의
사용자가 플랫 프리셋을 선택하면 모든 대역 게인이 중립으로 초기화된다.

## 진입점
- 이퀄라이저 패널: 플랫 프리셋 클릭

## 사용자 흐름
1. 사용자가 이퀄라이저 패널을 연다.
2. 플랫 프리셋을 클릭한다.
3. 모든 대역 게인이 0dB로 초기화된다.
4. 오디오에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 모든 대역이 0dB로 초기화된다.
- [ ] 오디오에 즉시 반영된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 수동 이퀄라이저 조작 동작 (별도 spec).
