# Spec: context-menu-dismiss

## 한 줄 정의
사용자가 우클릭 메뉴 외부를 클릭하면 메뉴가 닫힌다.

## 진입점
- 우클릭 메뉴: 메뉴 외부 클릭 / Esc 키

## 사용자 흐름
1. 사용자가 우클릭 메뉴를 연다.
2. 메뉴 외부를 클릭하거나 Esc 키를 누른다.
3. 메뉴가 닫힌다.

## 성공 조건
- [ ] 외부 클릭 시 메뉴가 닫힌다.
- [ ] Esc 키로 메뉴가 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 우클릭 메뉴 항목 동작 (별도 spec).
