# Spec: automatic-seek-hover-thumbnail-preview

## 한 줄 정의
시크바 위에 마우스를 올리면 해당 위치의 썸네일 미리보기가 자동으로 표시된다.

## 진입점
- 자동: 시크바에 마우스 호버 시

## 사용자 흐름
1. 사용자가 시크바 위에 마우스를 올린다.
2. 마우스 위치에 해당하는 시간의 썸네일이 표시된다.
3. 마우스를 이동하면 썸네일이 실시간으로 갱신된다.

## 성공 조건
- [ ] 마우스 호버 시 썸네일이 표시된다.
- [ ] 마우스 이동에 따라 썸네일이 갱신된다.
- [ ] 해당 위치의 시간이 함께 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 오디오 전용 파일은 썸네일이 표시되지 않는다.

## 범위 외
- 시크바 클릭/드래그 동작 (별도 spec).
