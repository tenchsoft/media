# Spec: bottom-volume-slider

## 한 줄 정의
사용자가 볼륨 슬라이더를 조작하면 오디오 볼륨이 변경된다.

## 진입점
- 하단 컨트롤 바: 볼륨 슬라이더 드래그

## 사용자 흐름
1. 사용자가 볼륨 슬라이더를 드래그한다.
2. 오디오 볼륨이 실시간으로 변경된다.
3. 0%에 도달하면 음소거 상태가 된다.

## 성공 조건
- [ ] 드래그 시 볼륨이 실시간으로 변경된다.
- [ ] 0%에서 음소거 상태가 된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 음소거 버튼과 독립적으로 동작한다.

## 범위 외
- 음소거 버튼 동작 (별도 spec).
