# Spec: automatic-subtitle-timing-overlay

## 한 줄 정의
자막이 타이밍에 맞춰 비디오 위에 자동으로 표시된다.

## 진입점
- 자동: 자막이 있는 미디어 재생 시

## 사용자 흐름
1. 사용자가 자막이 포함된 미디어를 재생한다.
2. 자막이 오디오/비디오 타이밍에 맞춰 화면에 표시된다.
3. 자막이 지나가면 화면에서 사라진다.

## 성공 조건
- [ ] 자막이 타이밍에 맞춰 표시된다.
- [ ] 자막이 지나가면 사라진다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 자막 스타일 설정에 따라 표시 모양이 달라진다.

## 범위 외
- 자막 스타일 설정 동작 (별도 spec).
- 자막 인코딩 선택 동작.
