# Spec: url-modal-cancel

## 한 줄 정의
사용자가 URL 재생 대화상자에서 취소 버튼을 누르면 대화상자가 닫힌다.

## 진입점
- URL 재생 대화상자: 취소 버튼 클릭

## 사용자 흐름
1. 사용자가 URL 재생 대화상자를 연다.
2. 취소 버튼을 클릭한다.
3. 대화상자가 닫힌다.
4. 재생이 시작되지 않는다.

## 성공 조건
- [ ] 클릭 시 대화상자가 닫힌다.
- [ ] 재생이 시작되지 않는다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
