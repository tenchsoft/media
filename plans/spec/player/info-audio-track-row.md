# Spec: info-audio-track-row

## 한 줄 정의
사용자가 정보 패널에서 오디오 트랙 항목을 클릭하면 오디오 트랙을 선택할 수 있다.

## 진입점
- 정보 패널: 오디오 트랙 항목 클릭

## 사용자 흐름
1. 사용자가 정보 패널을 연다.
2. 오디오 트랙 항목을 클릭한다.
3. 사용 가능한 오디오 트랙 목록이 표시된다.
4. 원하는 트랙을 선택한다.
5. 선택한 오디오 트랙으로 재생이 전환된다.

## 성공 조건
- [ ] 클릭 시 오디오 트랙 목록이 표시된다.
- [ ] 선택 시 해당 트랙으로 재생이 전환된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 오디오 트랙이 하나뿐이면 선택할 수 없다.

## 범위 외
- 오디오 장치 선택 동작 (별도 spec).
