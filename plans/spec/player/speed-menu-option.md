# Spec: speed-menu-option

## 한 줄 정의
사용자가 속도 메뉴에서 원하는 재생 속도를 선택하면 속도가 변경된다.

## 진입점
- 속도 메뉴: 속도 옵션 클릭

## 사용자 흐름
1. 사용자가 속도 메뉴를 연다.
2. 원하는 속도(0.25x, 0.5x, 0.75x, 1x, 1.25x, 1.5x, 1.75x, 2x, 3x, 4x)를 선택한다.
3. 재생 속도가 변경된다.
4. 현재 속도가 메뉴에 표시된다.

## 성공 조건
- [ ] 선택 시 재생 속도가 변경된다.
- [ ] 현재 속도가 메뉴에 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 속도 메뉴 열기 동작 (별도 spec).
