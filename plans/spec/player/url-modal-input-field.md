# Spec: url-modal-input-field

## 한 줄 정의
사용자가 URL 재생 대화상자에서 재생할 URL을 입력할 수 있다.

## 진입점
- URL 재생 대화상자: 입력 필드 클릭/포커스

## 사용자 흐름
1. 사용자가 URL 재생 대화상자를 연다.
2. 입력 필드에 URL을 입력한다.
3. 입력한 텍스트가 실시간으로 표시된다.
4. 엔터 키를 누르면 재생이 시도된다.

## 성공 조건
- [ ] 텍스트 입력이 실시간으로 반영된다.
- [ ] 엔터 시 URL 재생이 시도된다.

## 실패 / 취소 흐름
- 빈 입력 → 재생 버튼이 비활성화된다.

## 경계 / 예외
- 잘못된 URL → 오류 메시지가 표시된다.

## 범위 외
- URL 재생 버튼 동작 (별도 spec).
