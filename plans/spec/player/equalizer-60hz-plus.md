# Spec: equalizer-60hz-plus

## 한 줄 정의
사용자가 60Hz + 버튼을 누르면 60Hz 대역 게인이 증가한다.

## 진입점
- 이퀄라이저 패널: 60Hz + 버튼 클릭

## 사용자 흐름
1. 사용자가 이퀄라이저 패널을 연다.
2. 60Hz + 버튼을 클릭한다.
3. 60Hz 대역 게인이 한 단계 증가한다.
4. 오디오 출력에 즉시 반영된다.

## 성공 조건
- [ ] 클릭 시 60Hz 게인이 증가한다.
- [ ] 오디오에 즉시 반영된다.

## 실패 / 취소 흐름
- 최대 게인 도달 → 더 이상 증가하지 않는다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 60Hz - 버튼 동작 (별도 spec).
- 이퀄라이저 프리셋 동작.
