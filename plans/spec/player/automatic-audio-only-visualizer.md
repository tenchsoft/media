# Spec: automatic-audio-only-visualizer

## 한 줄 정의
오디오 파일 재생 시 시각화 이펙트가 자동으로 표시된다.

## 진입점
- 자동: 오디오 전용 파일 재생 시

## 사용자 흐름
1. 사용자가 오디오 파일을 연다.
2. 비디오 영역에 오디오 시각화 이펙트가 표시된다.
3. 음악의 주파수/리듬에 맞춰 시각화가 변한다.

## 성공 조건
- [ ] 오디오 파일 재생 시 시각화가 자동 표시된다.
- [ ] 오디오 주파수에 맞춰 시각화가 변한다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 비디오 파일에는 시각화가 표시되지 않는다.

## 범위 외
- 이퀄라이저 동작 (별도 spec).
- 오디오 트랙 선택 동작.
