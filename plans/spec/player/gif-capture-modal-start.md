# Spec: gif-capture-modal-start

## 한 줄 정의
사용자가 GIF 캡처 대화상자에서 시작 버튼을 누르면 GIF 녹화가 시작된다.

## 진입점
- GIF 캡처 대화상자: 시작 버튼 클릭

## 사용자 흐름
1. 사용자가 GIF 캡처 대화상자를 연다.
2. 시작 버튼을 클릭한다.
3. 현재 위치에서 GIF 녹화가 시작된다.
4. 녹화 중 표시가 나타난다.

## 성공 조건
- [ ] 클릭 시 GIF 녹화가 시작된다.
- [ ] 녹화 중 표시가 나타난다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 중지 동작 (별도 spec).
- GIF 옵션 설정 동작.
