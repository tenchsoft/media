# Spec: ai-feature-summarize-current-scene

## 한 줄 정의
사용자가 현재 장면을 AI가 요약하도록 요청할 수 있다.

## 진입점
- AI 기능 패널: 장면 요약 버튼 클릭

## 사용자 흐름
1. 사용자가 AI 기능 패널을 연다.
2. 장면 요약 버튼을 클릭한다.
3. AI가 현재 재생 중인 장면을 분석하여 요약을 표시한다.

## 성공 조건
- [ ] 버튼 클릭 시 AI가 장면을 분석한다.
- [ ] 요약 결과가 패널에 표시된다.

## 실패 / 취소 흐름
- 분석 실패 → 오류 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 대화 설명 동작 (별도 spec).
- 유사 프레임 검색 동작.
