# Spec: automatic-click-region-refresh

## 한 줄 정의
재생 상태 변화에 따라 클릭 영역이 자동으로 갱신된다.

## 진입점
- 자동: 재생 상태 변화 시

## 사용자 흐름
1. 재생/일시정지/정지 등 상태가 변한다.
2. 컨트롤 버튼의 클릭 영역이 새 상태에 맞게 갱신된다.
3. 사용자가 올바른 버튼을 클릭할 수 있다.

## 성공 조건
- [ ] 상태 변화 시 클릭 영역이 즉시 갱신된다.
- [ ] 갱신 후 올바른 버튼이 클릭된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 개별 버튼 동작 (별도 spec).
