# Spec: automatic-playback-position-progress

## 한 줄 정의
재생 중 현재 위치가 시크바에 자동으로 반영된다.

## 진입점
- 자동: 재생 중

## 사용자 흐름
1. 미디어가 재생 중이다.
2. 시크바의 재생 위치 표시기가 재생에 맞춰 자동으로 이동한다.
3. 시간 표시도 실시간으로 갱신된다.

## 성공 조건
- [ ] 재생 위치가 시크바에 실시간으로 반영된다.
- [ ] 시간 표시가 실시간으로 갱신된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 일시정지 시 위치 표시가 멈춘다.

## 범위 외
- 시크바 수동 조작 동작 (별도 spec).
