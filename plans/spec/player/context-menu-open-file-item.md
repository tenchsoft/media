# Spec: context-menu-open-file-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 파일 열기를 선택하면 파일 선택 대화상자가 나타난다.

## 진입점
- 우클릭 메뉴: 파일 열기 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 파일 열기 항목을 클릭한다.
3. 파일 선택 대화상자가 나타난다.
4. 파일을 선택하면 재생이 시작된다.

## 성공 조건
- [ ] 클릭 시 파일 선택 대화상자가 표시된다.
- [ ] 파일 선택 시 재생이 시작된다.

## 실패 / 취소 흐름
- 파일 선택 취소 → 대화상자가 닫힌다.

## 경계 / 예외
- 지원하지 않는 형식 → 오류 메시지가 표시된다.

## 범위 외
- URL 재생 동작 (별도 spec).
