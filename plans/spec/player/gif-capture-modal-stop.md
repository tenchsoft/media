# Spec: gif-capture-modal-stop

## 한 줄 정의
사용자가 GIF 캡처 대화상자에서 중지 버튼을 누르면 GIF 녹화가 중단된다.

## 진입점
- GIF 캡처 대화상자: 중지 버튼 클릭

## 사용자 흐름
1. GIF 녹화가 진행 중이다.
2. 중지 버튼을 클릭한다.
3. 녹화가 중단된다.
4. 캡처된 프레임으로 GIF 생성이 시작된다.

## 성공 조건
- [ ] 클릭 시 녹화가 중단된다.
- [ ] GIF 생성이 시작된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- GIF 녹화 시작 동작 (별도 spec).
- GIF 옵션 설정 동작.
