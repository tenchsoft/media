# Spec: recent-file-row

## 한 줄 정의
사용자가 최근 파일 목록에서 파일을 클릭하면 해당 파일이 재생된다.

## 진입점
- 최근 파일 목록: 파일 행 클릭

## 사용자 흐름
1. 사용자가 최근 파일 목록을 연다.
2. 원하는 파일 행을 클릭한다.
3. 해당 파일이 재생된다.

## 성공 조건
- [ ] 클릭 시 해당 파일이 재생된다.

## 실패 / 취소 흐름
- 파일이 이동/삭제된 경우 → 파일을 찾을 수 없다는 메시지가 표시된다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 최근 파일 목록 관리 동작.
