# Spec: automatic-media-info-refresh

## 한 줄 정의
파일이 열리면 미디어 정보가 자동으로 갱신된다.

## 진입점
- 자동: 파일 열기 시

## 사용자 흐름
1. 사용자가 파일을 연다.
2. 미디어 정보(해상도, 코덱, 비트레이트, 재생 시간 등)가 자동으로 갱신된다.
3. 정보 패널에서 확인할 수 있다.

## 성공 조건
- [ ] 파일 열기 시 미디어 정보가 자동 갱신된다.
- [ ] 정보 패널에 올바른 정보가 표시된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 손상된 파일 → 읽을 수 있는 정보만 표시된다.

## 범위 외
- 정보 패널 UI 동작 (별도 spec).
