# Spec: top-bar-info-drawer-tab

## 한 줄 정의
사용자가 상단 바 정보 탭을 누르면 미디어 정보 패널이 열린다.

## 진입점
- 상단 바: 정보 탭 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 정보 탭을 클릭한다.
2. 미디어 정보 패널이 열린다.
3. 파일 정보, 오디오/비디오 트랙, 오디오 장치를 확인할 수 있다.

## 성공 조건
- [ ] 클릭 시 정보 패널이 열린다.
- [ ] 다시 클릭 시 패널이 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 정보 패널 내부 동작 (별도 spec).
