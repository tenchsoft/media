# Spec: add-chapter-modal-add

## 한 줄 정의
사용자가 챕터 추가 대화상자에서 추가 버튼을 누르면 새 챕터가 현재 재생 위치에 생성된다.

## 진입점
- 챕터 추가 대화상자: 추가 버튼 클릭

## 사용자 흐름
1. 사용자가 챕터 추가 대화상자를 연다.
2. 챕터 이름을 입력한다.
3. 추가 버튼을 클릭한다.
4. 현재 재생 위치에 새 챕터가 생성된다.
5. 대화상자가 닫힌다.

## 성공 조건
- [ ] 추가 버튼 클릭 시 현재 위치에 챕터가 생성된다.
- [ ] 입력한 이름이 챕터에 반영된다.
- [ ] 대화상자가 닫힌다.

## 실패 / 취소 흐름
- 이름이 비어 있으면 → 추가 버튼이 비활성화된다.

## 경계 / 예외
- 동일 위치에 여러 챕터가 생성될 수 있다.

## 범위 외
- 챕터 가져오기 동작 (별도 spec).
- 챕터 내보내기 동작.
