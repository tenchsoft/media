# Spec: context-menu-aspect-item

## 한 줄 정의
사용자가 우클릭 메뉴에서 화면 비율을 선택하면 비율 모드가 전환된다.

## 진입점
- 우클릭 메뉴: 화면 비율 항목 클릭

## 사용자 흐름
1. 사용자가 비디오 영역을 우클릭한다.
2. 메뉴에서 화면 비율 항목을 클릭한다.
3. 하위 메뉴에서 원하는 비율을 선택한다.
4. 비디오 화면이 선택된 비율로 조정된다.

## 성공 조건
- [ ] 클릭 시 하위 메뉴가 표시된다.
- [ ] 선택한 비율이 적용된다.

## 실패 / 취소 흐름
- 메뉴 외부 클릭 → 메뉴가 닫힌다.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 하단 바 화면 비율 버튼 동작 (별도 spec).
