# Spec: builtin-subtitle-track-row

## 한 줄 정의
사용자가 내장 자막 트랙을 선택하면 해당 자막이 표시된다.

## 진입점
- 자막 선택 메뉴: 자막 트랙 항목 클릭

## 사용자 흐름
1. 사용자가 자막 선택 메뉴를 연다.
2. 사용 가능한 내장 자막 트랙 목록이 표시된다.
3. 원하는 트랙을 선택한다.
4. 해당 자막이 화면에 표시된다.

## 성공 조건
- [ ] 트랙 선택 시 해당 자막이 표시된다.
- [ ] 이전 자막이 해제된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 내장 자막이 없으면 목록이 비어 있다.

## 범위 외
- 자막 없음 동작 (별도 spec).
- 외부 자막 동작.
