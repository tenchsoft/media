# Spec: add-chapter-modal-input-field

## 한 줄 정의
사용자가 챕터 추가 대화상자에서 챕터 이름을 입력할 수 있다.

## 진입점
- 챕터 추가 대화상자: 입력 필드 클릭/포커스

## 사용자 흐름
1. 사용자가 챕터 추가 대화상자를 연다.
2. 입력 필드에 챕터 이름을 입력한다.
3. 입력한 텍스트가 실시간으로 표시된다.

## 성공 조건
- [ ] 텍스트 입력이 실시간으로 반영된다.
- [ ] 빈 입력 시 추가 버튼이 비활성화된다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 챕터 추가 버튼 동작 (별도 spec).
