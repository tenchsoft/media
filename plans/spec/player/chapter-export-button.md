# Spec: chapter-export-button

## 한 줄 정의
사용자가 챕터 내보내기 버튼을 누르면 챕터 목록이 파일로 저장된다.

## 진입점
- 챕터 패널: 내보내기 버튼 클릭

## 사용자 흐름
1. 사용자가 챕터 패널을 연다.
2. 내보내기 버튼을 클릭한다.
3. 저장 대화상자가 나타난다.
4. 파일 이름과 위치를 지정하고 저장한다.
5. 챕터 목록이 파일로 저장된다.

## 성공 조건
- [ ] 클릭 시 저장 대화상자가 표시된다.
- [ ] 챕터 목록이 파일로 저장된다.

## 실패 / 취소 흐름
- 저장 취소 → 파일이 저장되지 않는다.

## 경계 / 예외
- 챕터가 없으면 내보내기 버튼이 비활성화된다.

## 범위 외
- 챕터 가져오기 동작 (별도 spec).
