# Spec: top-bar-chapters-drawer-tab

## 한 줄 정의
사용자가 상단 바 챕터 탭을 누르면 챕터 패널이 열린다.

## 진입점
- 상단 바: 챕터 탭 클릭

## 사용자 흐름
1. 사용자가 상단 바에서 챕터 탭을 클릭한다.
2. 챕터 패널이 열린다.
3. 챕터 목록을 확인하고 관리할 수 있다.

## 성공 조건
- [ ] 클릭 시 챕터 패널이 열린다.
- [ ] 다시 클릭 시 패널이 닫힌다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- 챕터 추가/삭제 동작 (별도 spec).
