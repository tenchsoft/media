# Spec: automatic-ab-loop-playback-enforcement

## 한 줄 정의
AB 루프가 설정되면 재생이 항상 구간 내에서만 반복된다.

## 진입점
- 자동: AB 루프 활성화 시

## 사용자 흐름
1. 사용자가 AB 루프를 설정한다.
2. 재생이 A 지점에서 B 지점까지 진행된다.
3. B 지점에 도달하면 A 지점으로 돌아간다.
4. 루프가 해제될 때까지 반복된다.

## 성공 조건
- [ ] AB 루프 구간 내에서만 재생이 반복된다.
- [ ] 루프 해제 시 정상 재생으로 돌아간다.

## 실패 / 취소 흐름
- 해당 없음.

## 경계 / 예외
- 해당 없음.

## 범위 외
- AB 루프 설정 버튼 동작 (별도 spec).
- 시크바 조작 동작.
